📘 1. Limites de fonctions

La limite d'une fonction en un point décrit le comportement de f(x) quand x s'approche de ce point.

🔹 Limite en un point (finie)

📐 Définition

limx→a f(x) = ℓ signifie :

Pour tout ε > 0, il existe δ > 0 tel que :

0 < |x - a| < δ ⟹ |f(x) - ℓ| < ε

On peut rendre f(x) aussi proche de ℓ qu'on veut, pourvu que x soit suffisamment proche de a.

🔹 Limite à droite et à gauche

Limites latérales
limx→a+ f(x) = ℓ⁺   |   x s'approche de a par valeurs supérieures
limx→a- f(x) = ℓ⁻   |   x s'approche de a par valeurs inférieures

📐 Théorème fondamental

limx→a f(x) = ℓ   ⟺   limx→a+ f(x) = limx→a- f(x) = ℓ

Si les limites à gauche et à droite sont différentes, la limite n'existe pas.

💡 Exemple : Fonction valeur absolue

f(x) = |x| en x = 0

limx→0+ |x| = limx→0+ x = 0

limx→0- |x| = limx→0- (-x) = 0

Donc limx→0 |x| = 0 ✓

💡 Exemple : Partie entière

E(x) = partie entière de x

En x = 2 :

limx→2- E(x) = 1

limx→2+ E(x) = 2

Donc limx→2 E(x) n'existe pas

🔹 Limites à l'infini

Limites infinies
limx→+∞ f(x) = +∞  ⟺  ∀A > 0, ∃M : x > M ⟹ f(x) > A

limx→-∞ f(x) = ℓ  ⟺  ∀ε > 0, ∃M : x < M ⟹ |f(x) - ℓ| < ε

🔹 Asymptotes

📉

Asymptote verticale en x = a : limx→a f(x) = ±∞

📈

Asymptote horizontale en +∞ : limx→+∞ f(x) = ℓ (droite y = ℓ)

📊

Asymptote oblique en +∞ : limx→+∞ [f(x) - (ax + b)] = 0 (droite y = ax + b)

💡 Exemple : Asymptotes

f(x) = (2x + 1)/(x - 3)

Asymptote verticale : x = 3 (dénominateur nul)

limx→3 f(x) = ±∞

Asymptote oblique : Division euclidienne

f(x) = 2 + 7/(x - 3)

Quand x → ∞, 7/(x - 3) → 0, donc asymptote y = 2

📗 2. Opérations sur les limites

Voici les règles pour calculer les limites quand x → a (ou ±∞).

🔹 Tableau récapitulatif des opérations

Limites de polynômes et fractions rationnelles
limx→±∞ P(x) = ±∞   (signe du coefficient dominant)

limx→±∞ P(x)/Q(x) = rapport des coefficients dominants

🔹 Somme et produit

➕

ℓ + ℓ' = ℓ + ℓ' (si deux limites finies)

➕

ℓ + (+∞) = +∞   |   ℓ + (-∞) = -∞

➕

(+∞) + (+∞) = +∞

⚡

(+∞) + (-∞) = FI ⚠️

🔹 Produit

✖️

ℓ × ℓ' = ℓℓ' (deux limites finies)

✖️

(+∞) × (+∞) = +∞   |   (-∞) × (-∞) = +∞

✖️

(+∞) × (-∞) = -∞

⚡

0 × ∞ = FI ⚠️

🔹 Quotient

➗

ℓ/ℓ' = ℓ/ℓ'    si ℓ' ≠ 0

➗

ℓ/±∞ = 0

➗

±∞/ℓ = ±∞   (ℓ ≠ 0)

⚡

0/0 = FI   |   ±∞/±∞ = FI ⚠️

🔹 Formes indéterminées (FI)

Les 7 formes indéterminées
① ∞ - ∞   |   ② 0 × ∞   |   ③ 0/0
④ ∞/∞   |   ⑤ 1∞   |   ⑥ ∞⁰   |   ⑦ 0⁰

💡 Lever l'indétermination : Factorisation

Calculer limx→2 (x² - 4)/(x - 2)

→ Forme : 0/0 (car x² - 4 = 0 quand x = 2)

→ Factoriser : (x² - 4) = (x - 2)(x + 2)

→ lim = limx→2 (x - 2)(x + 2)/(x - 2) = limx→2 (x + 2) = 4

💡 Lever l'indétermination : Conjugué

Calculer limx→0 (√(x+1) - 1)/x

→ Forme : 0/0

→ Multiplier par le conjugué :

= lim [(√(x+1) - 1)(√(x+1) + 1)]/[x(√(x+1) + 1)]

= lim [(x+1) - 1]/[x(√(x+1) + 1)]

= lim x/[x(√(x+1) + 1)] = lim 1/(√(x+1) + 1) = 1/2

💡 Lever l'indétermination : Factoriser le degré dominant

Calculer limx→+∞ (2x² + 3x - 1)/(x² - x + 5)

→ Forme : ∞/∞

→ Factoriser x² au numérateur et dénominateur :

= lim (2 + 3/x - 1/x²)/(1 - 1/x + 5/x²)

Quand x → ∞ : 3/x → 0, 1/x² → 0, 1/x → 0, 5/x² → 0

→ lim = (2 + 0 - 0)/(1 - 0 + 0) = 2

📙 3. Continuité d'une fonction

La continuité d'une fonction en un point signifie que sa courbe ne présente pas de "saut" en ce point.

🔹 Définition

📐 Continuité en un point

f est continue en a ⟺ limx→a f(x) = f(a)

Cela signifie :

• f(a) existe

• limx→a f(x) existe

• Les deux sont égales

🔹 Continuité à gauche et à droite

Continuité latérale
f est continue à droite en a  ⟺  limx→a+ f(x) = f(a)

f est continue à gauche en a  ⟺  limx→a- f(x) = f(a)

📐 Théorème

f est continue en a  ⟺  f est continue à droite ET à gauche en a

🔹 Continuité sur un intervalle

📝

f continue sur ]a, b[ : continue en tout point de ]a, b[

📝

f continue sur [a, b] : continue sur ]a, b[ et aux bornes

📝

f continue sur ℝ : continue en tout réel

🔹 Fonctions continues usuelles

Fonctions continues sur ℝ
• Fonctions polynomiales   |   • Fonction exponentielle eˣ
• Fonctions sinus et cosinus   |   • Valeur absolue |x|
• Racine carrée √x (sur [0, +∞[)   |   • ln(x) (sur ]0, +∞[)

⚠️ Points de discontinuité

• Continuité impossible : division par zéro → asymptote

• Saut : partie entière, signe(x)

• Trou : fonction définie par morceaux avec formule différente à la jonction

💡 Exemple : Continuité d'une fonction

f(x) = x² - 3x + 1

C'est un polynôme → continue sur ℝ

Vérification en x = 2 :

limx→2 f(x) = 4 - 6 + 1 = -1

f(2) = 4 - 6 + 1 = -1 ✓

💡 Exemple : Fonction avec valeur absolue

f(x) = |x - 2|/(x - 2)

Étudier la continuité en x = 2 :

limx→2+ |x-2|/(x-2) = lim (x-2)/(x-2) = 1

limx→2- |x-2|/(x-2) = lim -(x-2)/(x-2) = -1

Les limites sont différentes → discontinue en x = 2

🔹 Opérations sur les fonctions continues

📐 Propriétés

Si f et g sont continues en a, alors :

• f + g est continue en a

• f × g est continue en a

• f/g est continue en a si g(a) ≠ 0

• f ∘ g est continue en a

📕 4. Théorèmes des valeurs intermédiaires

Ces théorèmes sont fondamentaux pour démontrer l'existence de solutions d'équations.

🔹 Théorème des valeurs intermédiaires (TVI)

📐 TVI

Si f est continue sur [a, b] et si k est entre f(a) et f(b)

Alors il existe au moins un c ∈ [a, b] tel que f(c) = k

💡 Exemple : Montrer l'existence d'une solution

Montrer que l'équation x³ - 3x + 1 = 0 a une solution dans [0, 1]

f(x) = x³ - 3x + 1 est continue sur [0, 1]

f(0) = 1 > 0

f(1) = 1 - 3 + 1 = -1 < 0

0 ∈ [f(1), f(0)] donc ∃ c ∈ [0, 1] tel que f(c) = 0

Conclusion : l'équation a une solution dans [0, 1]

🔹 Corollaire : Images d'intervalles

📐 Image d'un intervalle

Si f est continue sur un intervalle I, alors f(I) est un intervalle.

En particulier : l'image d'un segment [a, b] par une fonction continue est le segment [m, M] où m = min f et M = max f sur [a, b].

🔹 Théorème de la bijection (ouTVI strict)

📐 Théorème de la bijection

Si f est continue et strictement monotone sur [a, b]

Alors f est une bijection de [a, b] sur [f(a), f(b)]

Conséquence : l'équation f(x) = k a une solution unique dans [a, b].

💡 Exemple : Solution unique

Montrer que x³ + x - 2 = 0 a une solution unique dans ℝ

f(x) = x³ + x - 2 est continue sur ℝ

f'(x) = 3x² + 1 > 0 pour tout x

→ f est strictement croissante sur ℝ

limx→-∞ f(x) = -∞ et limx→+∞ f(x) = +∞

Donc pour tout k ∈ ℝ, ∃! c ∈ ℝ : f(c) = k

En particulier, ∃! solution pour f(x) = 0

⚠️ Conditions importantes

• Le TVI donne l'existence d'une solution

• Pour l'unicité, il faut ajouter : f est strictement monotone

• Si f n'est pas monotone : il peut y avoir plusieurs solutions

🔹 Résolution approchée d'équations

🔍

Dichotomie : on divise l'intervalle en 2 à chaque étape

🔍

Méthode de Newton : xn+1 = xn - f(xn)/f'(xn)

🔍

Point fixe : si g(x) = x, la solution est un point fixe

📒 5. Limites remarquables

Ces limites sont à connaître par cœur et s'utilisent fréquemment dans les exercices.

🔹 Limites trigonométriques

Limites fondamentales
limx→0 sin(x)/x = 1   |   limx→0 tan(x)/x = 1

limx→0 (1 - cos(x))/x² = 1/2

💡 Exemple : Application

Calculer limx→0 sin(3x)/(2x)

= limx→0 [sin(3x)/(3x)] × [3x/(2x)]

= 1 × (3/2) = 3/2

💡 Exemple : sin(5x)/sin(2x)

limx→0 sin(5x)/sin(2x)

= lim [sin(5x)/(5x)] × [(2x)/sin(2x)] × (5/2)

= 1 × 1 × (5/2) = 5/2

🔹 Limites exponentielles

Nombre e
limx→0 (1 + x)1/x = e   (e ≈ 2,718)

limx→±∞ (1 + 1/x)x = e

💡 Exemple : Forme 1∞

Calculer limx→0 (1 + 3x)1/(2x)

= lim [(1 + 3x)1/(3x)]3/(2x) × (3x)

= e3/2

🔹 Croissance comparée

Croissance comparée
limx→+∞ xⁿ/eˣ = 0   |   limx→+∞ ln(x)/xⁿ = 0

limx→0⁺ xⁿ × ln(x) = 0   |   limx→+∞ xⁿ × aˣ = +∞ (si a > 1)

💡 Exemple : ln(x)/x

limx→+∞ ln(x)/x

= limx→+∞ (1/x)/(1)   (par DL ou Hospital)

= 0

Conclusion : le logarithme croît moins vite que x

⚠️ Règle de L'Hôpital

Si on a une forme indéterminée 0/0 ou ∞/∞, on peut calculer :

lim f(x)/g(x) = lim f'(x)/g'(x)

à condition que les dérivées existent et que la nouvelle limite existe

💡 Exemple : L'Hôpital

limx→0 sin(x)/x   (forme 0/0)

= limx→0 cos(x)/1 = 1

📘 1. Repères et coordonnées dans le plan

1.1 Repère cartésien

Un repère cartésien du plan est défini par un point O appelé origine et deux axes perpendiculaires (ou non) définissant une base. On distingue :

📐

Repère orthonormal (O, i⃗, j⃗) : les deux axes sont perpendiculaires et les vecteurs unitaires ont même longueur

📐

Repère orthogonal : les axes sont perpendiculaires mais les unités peuvent être différentes

📐

Repère oblique : les axes ne sont pas perpendiculaires

1.2 Coordonnées cartésiennes

Tout point M du plan peut être caractérisé par un couple unique (x, y) appelé coordonnées cartésiennes :

Coordonnées d'un point
OM⃗ = x·i⃗ + y·j⃗

où x est l'abscisse et y l'ordonnée du point M.

💡 Exemples de coordonnées

• M(3, 4) : abscisse 3, ordonnée 4

• N(-2, 5) : abscisse -2, ordonnée 5

• P(0, -3) : sur l'axe des ordonnées négatives

• O(0, 0) : l'origine du repère

1.3 Coordonnées polaires

Tout point M (différent de O) peut aussi être caractérisé par ses coordonnées polaires (ρ, θ) :

📏

ρ ≥ 0 : distance OM

🔄

θ ∈ ℝ : angle orienté (Ox⃗, OM⃗)

Passage coordonnées polaires → cartésiennes
x = ρ·cos θ
y = ρ·sin θ
Passage coordonnées cartésiennes → polaires
ρ = √(x² + y²)
cos θ = x/ρ et sin θ = y/ρ

💡 Exemple : Coordonnées polaires de M(3, 4)

ρ = √(9 + 16) = 5

cos θ = 3/5 et sin θ = 4/5

θ = arctan(4/3) ≈ 53,13°

Donc M a pour coordonnées polaires (5, 53,13°)

1.4 Milieu d'un segment

Le milieu I du segment [AB] où A(x₁, y₁) et B(x₂, y₂) a pour coordonnées :

Coordonnées du milieu
x_I = (x₁ + x₂)/2
y_I = (y₁ + y₂)/2

💡 Exemple : Milieu de [A(2, 3)] et [B(6, 7)]

x_I = (2 + 6)/2 = 4

y_I = (3 + 7)/2 = 5

Donc I(4, 5)

1.5 Distance entre deux points

Dans un repère orthonormal, la distance entre A(x₁, y₁) et B(x₂, y₂) est :

Distance AB
AB = √[(x₂ - x₁)² + (y₂ - y₁)²]

💡 Exemple : Distance entre A(1, 2) et B(4, 6)

AB = √[(4-1)² + (6-2)²]

= √[3² + 4²] = √[9 + 16] = √25 = 5

📗 2. Vecteurs et opérations vectorielles

2.1 Définition d'un vecteur

Un vecteur est défini par une direction, un sens et une longueur (norme). Si A et B sont deux points, le vecteur AB⃗ représente le déplacement de A vers B.

➡️

Direction : la droite qui porte le vecteur

➡️

Sens : orientation sur cette droite

📏

Norme : longueur du vecteur, notée ‖AB⃗‖ ou AB

2.2 Coordonnées d'un vecteur

Si A(x₁, y₁) et B(x₂, y₂), alors le vecteur AB⃗ a pour coordonnées :

Coordonnées du vecteur AB⃗
AB⃗ = (x₂ - x₁, y₂ - y₁)

💡 Exemple : Vecteur de A(2, 3) à B(7, 8)

AB⃗ = (7-2, 8-3) = (5, 5)

2.3 Opérations sur les vecteurs

Addition de vecteurs

Si u⃗ = (x₁, y₁) et v⃗ = (x₂, y₂), alors :

Somme de deux vecteurs
u⃗ + v⃗ = (x₁ + x₂, y₁ + y₂)

Multiplication par un scalaire

Si u⃗ = (x, y) et k ∈ ℝ, alors :

Produit par un scalaire
k·u⃗ = (k·x, k·y)

Vecteur opposé

Vecteur opposé
-u⃗ = (-x, -y)

Soustraction

Différence de deux vecteurs
u⃗ - v⃗ = (x₁ - x₂, y₁ - y₂)

💡 Exemple d'opérations

u⃗ = (3, 4), v⃗ = (1, -2)

u⃗ + v⃗ = (4, 2)

2·u⃗ = (6, 8)

u⃗ - v⃗ = (2, 6)

2.4 Propriétés des opérations

➕

Commutativité : u⃗ + v⃗ = v⃗ + u⃗

➕

Associativité : (u⃗ + v⃗) + w⃗ = u⃗ + (v⃗ + w⃗)

🔢

Élément neutre : u⃗ + 0⃗ = u⃗

🔄

Opposé : u⃗ + (-u⃗) = 0⃗

2.5 Colinéarité de vecteurs

Deux vecteurs u⃗ = (x₁, y₁) et v⃗ = (x₂, y₂) sont colinéaires si et seulement si :

Condition de colinéarité
x₁·y₂ - x₂·y₁ = 0

Autrement dit, il existe un réel k tel que u⃗ = k·v⃗.

💡 Exemple : Vecteurs colinéaires

u⃗ = (4, 6) et v⃗ = (2, 3)

4×3 - 2×6 = 12 - 12 = 0

Donc u⃗ et v⃗ sont colinéaires

on a u⃗ = 2·v⃗

2.6 Norme d'un vecteur

La norme (longueur) du vecteur u⃗ = (x, y) est :

Norme d'un vecteur
‖u⃗‖ = √(x² + y²)

💡 Exemple : Norme de u⃗ = (3, 4)

‖u⃗‖ = √(9 + 16) = √25 = 5

📙 3. Produit scalaire

3.1 Définition

Le produit scalaire de deux vecteurs u⃗ = (x₁, y₁) et v⃗ = (x₂, y₂) est le nombre réel :

Produit scalaire (forme analytique)
u⃗ · v⃗ = x₁·x₂ + y₁·y₂

Le produit scalaire peut aussi s'écrire en fonction des normes et de l'angle :

Produit scalaire (forme trigonométrique)
u⃗ · v⃗ = ‖u⃗‖ × ‖v⃗‖ × cos(u⃗, v⃗)

3.2 Propriétés du produit scalaire

🔄

Symétrie : u⃗ · v⃗ = v⃗ · u⃗

➕

Linéarité : (α·u⃗ + β·w⃗) · v⃗ = α(u⃗ · v⃗) + β(w⃗ · v⃗)

📌

Positivité : u⃗ · u⃗ = ‖u⃗‖² ≥ 0

📐

Orthogonalité : u⃗ · v⃗ = 0 ⟺ u⃗ ⟂ v⃗

💡 Exemple : Calcul du produit scalaire

u⃗ = (2, 3) et v⃗ = (-1, 4)

u⃗ · v⃗ = 2×(-1) + 3×4 = -2 + 12 = 10

3.3 Applications du produit scalaire

Norme d'un vecteur

Norme via le produit scalaire
‖u⃗‖ = √(u⃗ · u⃗)

Angle entre deux vecteurs

Cosinus de l'angle
cos θ = (u⃗ · v⃗) / (‖u⃗‖ × ‖v⃗‖)

Projection orthogonale

Projection de u⃗ sur v⃗
proj_{v⃗}(u⃗) = [(u⃗ · v⃗) / (v⃗ · v⃗)] × v⃗

💡 Exemple : Angle entre u⃗ = (1, 0) et v⃗ = (1, 1)

u⃗ · v⃗ = 1×1 + 0×1 = 1

‖u⃗‖ = 1 et ‖v⃗‖ = √2

cos θ = 1/(1×√2) = √2/2

Donc θ = 45° = π/4 rad

⛔ Critères d'orthogonalité

Pour vérifier que deux vecteurs sont perpendiculaires, on peut utiliser :

• u⃗ · v⃗ = 0 (produit scalaire nul)

• x₁·x₂ + y₁·y₂ = 0 (coordonnées)

• cos θ = 0 ⟺ θ = π/2

📕 4. Droites dans le plan

4.1 Détermination d'une droite

Une droite est entièrement déterminée par :

📍

Deux points distincts A et B

📍

Un point et une direction (vecteur directeur)

📍

Un point et un vecteur normal (perpendiculaire)

📍

Un point et le coefficient directeur (dans repère orthonormal)

4.2 Vecteur directeur

Un vecteur directeur d'une droite D est un vecteur v⃗ qui indique la direction de D. Si A et B sont deux points de D, alors AB⃗ est un vecteur directeur.

Propriété
Tous les vecteurs directeurs sont colinéaires

4.3 Vecteur normal

Un vecteur normal à une droite D est un vecteur n⃗ perpendiculaire à tous les vecteurs directeurs de D.

📐

Si v⃗ = (a, b) est un vecteur directeur, alors n⃗ = (-b, a) ou n⃗ = (b, -a) est un vecteur normal

📐

Réciproquement, si n⃗ = (a, b) est normal, alors v⃗ = (-b, a) est directeur

💡 Exemple : Vecteurs d'une droite

Droite D passant par A(1, 2) et B(4, 6)

Vecteur directeur : v⃗ = AB⃗ = (3, 4)

Vecteur normal : n⃗ = (-4, 3) ou (4, -3)

4.4 Droites remarquables

➡️

Axe Ox : y = 0, vecteur normal (0, 1)

⬆️

Axe Oy : x = 0, vecteur normal (1, 0)

📐

Parallèle à Ox : y = constante

📐

Parallèle à Oy : x = constante

📒 5. Équations de droites

5.1 Équation cartésienne

Dans un repère orthonormal, toute droite peut s'écrire sous la forme :

Équation cartésienne
ax + by + c = 0

où (a, b) est un vecteur normal à la droite (a et b ne sont pas tous deux nuls).

📐

Vecteur normal : n⃗ = (a, b)

➡️

Vecteur directeur : v⃗ = (-b, a)

5.2 Équation réduite

Si b ≠ 0, on peut écrire l'équation sous forme réduite :

Équation réduite
y = mx + p
📐

m est le coefficient directeur (ou pente)

📐

p est l'ordonnée à l'origine

Relation entre m et n⃗
m = tg α où α est l'angle que fait la droite avec l'axe Ox
m = -a/b (si ax + by + c = 0)

💡 Exemple : Passage entre formes

Droite : 2x + 3y - 6 = 0

Forme réduite : 3y = -2x + 6 ⟹ y = (-2/3)x + 2

Coefficient directeur : m = -2/3

Ordonnée à l'origine : p = 2

Vecteur normal : n⃗ = (2, 3)

Vecteur directeur : v⃗ = (-3, 2)

5.3 Équation paramétrique

Une droite passant par A(x₀, y₀) et de vecteur directeur v⃗ = (a, b) a pour équation paramétrique :

Équation paramétrique
x = x₀ + t·a
y = y₀ + t·b     (t ∈ ℝ)

💡 Exemple : Droite passant par A(1, 2) et B(4, 6)

v⃗ = AB⃗ = (3, 4)

x = 1 + 3t

y = 2 + 4t

Pour t = 1 : on obtient B(4, 6) ✓

5.4 Équation passant par deux points

La droite passant par A(x₁, y₁) et B(x₂, y₂) (avec x₁ ≠ x₂) a pour équation :

Équation par deux points
(y - y₁) / (y₂ - y₁) = (x - x₁) / (x₂ - x₁)

💡 Exemple : Droite passant par A(1, 2) et B(3, 6)

m = (6-2)/(3-1) = 4/2 = 2

Équation : y - 2 = 2(x - 1)

Soit : y = 2x

📘 6. Positions relatives de deux droites

6.1 Droites parallèles

Deux droites D₁ et D₂ sont parallèles si et seulement si leurs vecteurs normaux sont colinéaires :

Condition de parallélisme
D₁ // D₂ ⟺ a₁·b₂ - a₂·b₁ = 0
(pour D₁ : a₁x + b₁y + c₁ = 0 et D₂ : a₂x + b₂y + c₂ = 0)

En forme réduite, deux droites sont parallèles si et seulement si elles ont le même coefficient directeur :

Parallélisme en forme réduite
D₁ // D₂ ⟺ m₁ = m₂

6.2 Droites perpendiculaires

Deux droites D₁ et D₂ sont perpendiculaires si et seulement si :

Condition d'orthogonalité
D₁ ⟂ D₂ ⟺ a₁·a₂ + b₁·b₂ = 0

En forme réduite :

Orthogonalité en forme réduite
D₁ ⟂ D₂ ⟺ m₁ × m₂ = -1

💡 Exemple : Positions de droites

D₁ : y = 2x + 3 (m₁ = 2)

D₂ : y = 2x - 1 (m₂ = 2)

D₁ // D₂ car m₁ = m₂

D₃ : y = -x/2 + 1 (m₃ = -1/2)

D₁ ⟂ D₃ car 2 × (-1/2) = -1

6.3 Droites sécantes

Deux droites sont sécantes (non parallèles) si elles se coupent en un point. Pour trouver ce point, on résout le système formé par les deux équations.

💡 Exemple : Point d'intersection

D₁ : x + y = 6

D₂ : 2x - y = 3

En additionnant : 3x = 9 ⟹ x = 3

En substituant : 3 + y = 6 ⟹ y = 3

Point d'intersection : I(3, 3)

6.4 Tableau récapitulatif

➡️

Parallèles confondues : mêmes coefficients (a, b, c proportionnels)

➡️

Parallèles distinctes : mêmes directions mais c différent

❌

Sécantes : directions différentes

📐

Perpendiculaires : cas particulier de sécantes avec angle de 90°

📗 7. Distance d'un point à une droite

7.1 Formule de la distance

Dans un repère orthonormal, la distance du point A(x₀, y₀) à la droite D : ax + by + c = 0 est :

Distance point-droite
d(A, D) = |a·x₀ + b·y₀ + c| / √(a² + b²)

💡 Exemple : Distance de A(1, 2) à D : 3x + 4y - 5 = 0

d(A, D) = |3×1 + 4×2 - 5| / √(9 + 16)

= |3 + 8 - 5| / √25

= |6| / 5 = 6/5

7.2 Applications

Équation de la médiatrice

La médiatrice du segment [AB] est l'ensemble des points équidistants de A et B :

Médiatrice de [AB]
{M(x, y) / MA = MB}
⟺ (x - x_A)² + (y - y_A)² = (x - x_B)² + (y - y_B)²

Équation des bissectrices

Les bissectrices de deux droites D₁ et D₂ sont les ensembles de points équidistants :

Bissectrices
d(M, D₁) = d(M, D₂)

💡 Exemple : Bissectrices de D₁ : x = 0 et D₂ : y = 0

d(M, D₁) = |x| et d(M, D₂) = |y|

|x| = |y| ⟹ y = x ou y = -x

Les deux bissectrices sont les droites y = x et y = -x

7.3 Distance entre deux droites parallèles

Si D₁ et D₂ sont deux droites parallèles, la distance entre elles est la distance d'un point de D₁ à D₂ :

Distance entre deux droites parallèles
d(D₁, D₂) = |c₂ - c₁| / √(a² + b²)
(pour D₁ : ax + by + c₁ = 0 et D₂ : ax + by + c₂ = 0)

💡 Exemple : Distance entre D₁ : 2x + y + 1 = 0 et D₂ : 2x + y - 3 = 0

d(D₁, D₂) = |-3 - 1| / √(4 + 1)

= |-4| / √5 = 4/√5

📙 8. Le cercle

8.1 Définition

Un cercle de centre C et de rayon R est l'ensemble des points M tels que CM = R.

8.2 Équation cartésienne

Le cercle de centre C(α, β) et de rayon R a pour équation :

Équation du cercle (forme canonique)
(x - α)² + (y - β)² = R²

Forme développée :

Équation développée
x² + y² + 2px + 2qy + d = 0
avec centre C(-p, -q) et R = √(p² + q² - d)

💡 Exemple : Cercle de centre C(2, 3) et rayon 5

(x - 2)² + (y - 3)² = 25

En développé : x² - 4x + 4 + y² - 6y + 9 = 25

Soit : x² + y² - 4x - 6y - 12 = 0

8.3 Condition d'existence

L'équation x² + y² + 2px + 2qy + d = 0 représente un cercle si et seulement si :

Condition d'existence
p² + q² - d > 0
📌

Si = 0 : le cercle est réduit à un point (cercle dégénéré)

📌

Si < 0 : pas de cercle (ensemble vide)

8.4 Tangente à un cercle

La tangente au cercle de centre C en un point M₀ est la droite perpendiculaire à CM⃗₀ passant par M₀.

Équation de la tangente
(x_M₀ - α)(x - x_M₀) + (y_M₀ - β)(y - y_M₀) = 0

💡 Exemple : Tangente au cercle (x-1)² + (y-2)² = 25 au point M₀(4, 4)

Centre C(1, 2)

CM⃗₀ = (3, 2)

Équation : 3(x-4) + 2(y-4) = 0

Soit : 3x + 2y - 20 = 0

8.5 Intersection cercle-droite

Pour trouver l'intersection d'un cercle et d'une droite, on résout le système. Les cas possibles sont :

📌

2 points : la droite est sécante

📌

1 point : la droite est tangente

📌

0 point : la droite est extérieure

📕 9. Parabole, ellipse et hyperbole

9.1 La parabole

Une parabole est l'ensemble des points équidistants d'un point fixe F (foyer) et d'une droite D (directrice).

Équation de la parabole (foyer F(p/2, 0), directrice x = -p/2)
y² = 2px

💡 Exemple : Parabole y² = 4x

Cette parabole a pour foyer F(1, 0) et pour directrice x = -1

Le paramètre p = 2

9.2 L'ellipse

Une ellipse est l'ensemble des points dont la somme des distances à deux points fixes F₁ et F₂ (foyers) est constante.

Équation de l'ellipse (centre O, axes parallèles aux axes de coordonnées)
x²/a² + y²/b² = 1
📌

a : demi-grand axe

📌

b : demi-petit axe

📌

c² = a² - b² : distance focale

📌

Excentricité : e = c/a < 1

9.3 L'hyperbole

Une hyperbole est l'ensemble des points dont la différence des distances à deux points fixes F₁ et F₂ est constante.

Équation de l'hyperbole
x²/a² - y²/b² = 1
📌

c² = a² + b² : distance focale

📌

Excentricité : e = c/a > 1

📌

Asymptotes : y = ±(b/a)x

💡 Comparaison des excentricités

• Cercle : e = 0

• Ellipse : 0 < e < 1

• Parabole : e = 1

• Hyperbole : e > 1

📒 10. Transformations géométriques

10.1 Translation

Une translation de vecteur u⃗ = (a, b) transforme un point M(x, y) en M'(x', y') avec :

Translation
x' = x + a
y' = y + b

10.2 Homothétie

Une homothétie de centre O et de rapport k transforme M en M' avec :

Homothétie de centre O
OM⃗' = k·OM⃗
x' = k·x et y' = k·y

10.3 Rotation

Une rotation de centre O et d'angle θ transforme M(x, y) en M'(x', y') :

Rotation de centre O et d'angle θ
x' = x·cos θ - y·sin θ
y' = x·sin θ + y·cos θ

💡 Exemple : Rotation de 90° (π/2) de M(1, 0)

x' = 1×0 - 0×1 = 0

y' = 1×1 + 0×0 = 1

M'(0, 1) ✓

10.4 Symétrie centrale

La symétrie centrale de centre O transforme M en M' avec :

Symétrie centrale
OM⃗' = -OM⃗
x' = -x et y' = -y

10.5 Symétrie axiale

La symétrie axiale par rapport à une droite D transforme M en M' tel que D soit la médiatrice de [MM'].

📐

Par rapport à Ox : M'(x, -y)

📐

Par rapport à Oy : M'(-x, y)

📐

Par rapport à y = x : M'(y, x)

10.6 Propriétés conservées

✅

Translation : conserve les distances et les angles

✅

Homothétie : conserve les angles, multiplie les distances par |k|

✅

Rotation : conserve les distances et les angles

✅

Symétrie : conserve les distances et les angles

📘 11. Angles et trigonométrie

11.1 Mesure des angles

Un angle peut être mesuré en degrés ou en radians :

Conversion degré-radian
180° = π rad
1 rad = 180°/π ≈ 57,3°

11.2 Formules trigonométriques fondamentales

📐

cos² θ + sin² θ = 1

📐

tan θ = sin θ / cos θ

📐

1 + tan² θ = 1 / cos² θ

11.3 Valeurs remarquables

Tableau des valeurs remarquables
0° : cos=1, sin=0, tan=0
30° : cos=√3/2, sin=1/2, tan=√3/3
45° : cos=√2/2, sin=√2/2, tan=1
60° : cos=1/2, sin=√3/2, tan=√3
90° : cos=0, sin=1, tan=∞

11.4 Formules d'addition

📐

cos(a+b) = cos a·cos b - sin a·sin b

📐

cos(a-b) = cos a·cos b + sin a·sin b

📐

sin(a+b) = sin a·cos b + cos a·sin b

📐

sin(a-b) = sin a·cos b - cos a·sin b

11.5 Formules de duplication

📐

cos(2a) = cos² a - sin² a = 2cos² a - 1 = 1 - 2sin² a

📐

sin(2a) = 2·sin a·cos a

📐

tan(2a) = 2·tan a / (1 - tan² a)

💡 Exemple : Calcul de cos(75°)

cos(75°) = cos(45° + 30°)

= cos 45°·cos 30° - sin 45°·sin 30°

= (√2/2)(√3/2) - (√2/2)(1/2)

= (√6 - √2)/4

📗 12. Applications et problèmes types

12.1 Problème : Déterminer une équation de droite

💡 Exemple : Droite passant par A(1, 2) et perpendiculaire à D : 3x + 4y - 5 = 0

Vecteur normal de D : n⃗ = (3, 4)

La droite cherchée a pour vecteur normal v⃗ = (-4, 3) (perpendiculaire à n⃗)

Équation : -4(x-1) + 3(y-2) = 0

-4x + 4 + 3y - 6 = 0

Soit : -4x + 3y - 2 = 0 ou 4x - 3y + 2 = 0

12.2 Problème : Intersection et angle

💡 Exemple : Angle entre D₁ : y = 2x + 1 et D₂ : y = -x/2 + 3

m₁ = 2 et m₂ = -1/2

tan θ = |(m₁ - m₂)/(1 + m₁·m₂)|

= |(2 - (-1/2))/(1 + 2×(-1/2))|

= |(5/2)/(0)| = ∞

Donc θ = 90° (les droites sont perpendiculaires)

12.3 Problème : Cercle inscrit et circonscrit

💡 Exemple : Cercle circonscrit au triangle ABC avec A(0, 0), B(4, 0), C(0, 3)

Le cercle a pour équation x² + y² + 2px + 2qy + d = 0

A(0,0) ∈ cercle ⟹ d = 0

B(4,0) ∈ cercle ⟹ 16 + 8p = 0 ⟹ p = -2

C(0,3) ∈ cercle ⟹ 9 + 6q = 0 ⟹ q = -3/2

Équation : x² + y² - 4x - 3y = 0

Centre : (2, 3/2), rayon : √(4 + 9/4) = 5/2

⚠️ Conseils pour les examens

• Toujours identifier clairement les données et l'inconnue

• Choisir le type d'équation le plus adapté au problème

• Vérifier que vos résultats satisfont les conditions initiales

• Dessiner une figure pour visualiser le problème

📘 1. Introduction aux équations différentielles

1.1. Définition d'une équation différentielle

Une équation différentielle est une équation qui relie une fonction inconnue à ses dérivées successives. Elle permet de décrire l'évolution de phénomènes physiques, biologiques ou économiques dans le temps ou dans l'espace.

📖 Définition : Équation différentielle

Une équation différentielle est une équation dans laquelle apparaissent :

  • La fonction inconnue y = f(x)
  • Ses dérivées y', y'', ..., y(n)

On dit que c'est une équation différentielle d'ordre n si l'ordre le plus élevé de dérivation qui apparaît est n.

💡 Exemples d'équations différentielles

  • y' = 2x + 1 → équation différentielle d'ordre 1
  • y'' + 4y = 0 → équation différentielle d'ordre 2
  • y' = y² → équation différentielle d'ordre 1
  • y''' − 2y'' + y' = ex → équation différentielle d'ordre 3

1.2. Ordre d'une équation différentielle

L'ordre d'une équation différentielle est le plus grand ordre de dérivée qui apparaît dans l'équation.

1️⃣

Ordre 1 : L'équation contient y' au maximum. Exemple : y' + 2y = 0

2️⃣

Ordre 2 : L'équation contient y'' au maximum. Exemple : y'' + y = 0

n️⃣

Ordre n : L'équation contient y(n) au maximum. Exemple : y''' − y'' = 0

1.3. Solution d'une équation différentielle

Une solution d'une équation différentielle est une fonction y = f(x) qui, substituée dans l'équation, vérifie l'égalité pour tout x dans un intervalle donné.

📖 Définition : Solution générale et solutions particulières

Solution générale : C'est l'ensemble de toutes les solutions de l'équation différentielle. Elle contient une ou plusieurs constantes arbitraires (constantes d'intégration).

Solution particulière : C'est une solution spécifique obtenue en fixant les valeurs des constantes à l'aide de conditions initiales ou de conditions aux limites.

💡 Exemple : Solutions d'une équation différentielle

Considérons l'équation différentielle : y' = 2x

Solution générale : En intégrant, on obtient y = x² + C, où C est une constante arbitraire.

Solution particulière : Si y(0) = 3, alors 3 = 0 + C ⟹ C = 3 ⟹ y = x² + 3

1.4. Condition initiale et problème de Cauchy

Une condition initiale est une condition de la forme y(x₀) = y₀ qui précise la valeur de la solution en un point donné.

📖 Définition : Problème de Cauchy

Un problème de Cauchy est un problème constitué de :

  • Une équation différentielle
  • Une condition initiale (ou plusieurs pour les équations d'ordre supérieur)

Sa résolution consiste à trouver la solution particulière satisfaisant la (les) condition(s) initiale(s).

💡 Exemple : Problème de Cauchy

Résoudre le problème de Cauchy :

{ y' = 3x²

{ y(1) = 4

Solution :

y = ∫ 3x² dx = x³ + C

Condition : y(1) = 1³ + C = 4 ⟹ C = 3

Solution particulière : y = x³ + 3

⚠️ Nombre de conditions initiales nécessaires

Pour une équation différentielle d'ordre n, il faut n conditions initiales (ou conditions aux limites) pour déterminer complètement la solution.

  • Ordre 1 : 1 condition (y(x₀) = y₀)
  • Ordre 2 : 2 conditions (y(x₀) = y₀ et y'(x₀) = y₁)

📗 2. Équations différentielles du premier ordre

2.1. Forme générale

Une équation différentielle du premier ordre peut se mettre sous la forme générale :

Forme générale d'une équation du 1er ordre

F(x, y, y') = 0

ou sous forme résolue :

y' = f(x, y)

où y' = dydx est la dérivée première de y par rapport à x.

2.2. Classification des équations du premier ordre

Les équations différentielles du premier ordre se divisent en plusieurs catégories selon leur forme :

🔗

Équations à variables séparables : y' = g(x) · h(y)

📏

Équations linéaires : y' + a(x) · y = b(x)

🔄

Équations homogènes : y' = f(yx)

📐

Équations exactes : M(x,y)dx + N(x,y)dy = 0

2.3. Méthode de résolution générale

Pour résoudre une équation différentielle du premier ordre, on suit généralement les étapes suivantes :

📋 Méthode : Résoudre une équation différentielle du 1er ordre

  • Étape 1 : Identifier le type d'équation (à variables séparables, linéaire, etc.)
  • Étape 2 : Appliquer la méthode de résolution appropriée
  • Étape 3 : Calculer la primitive pour obtenir la solution générale
  • Étape 4 : Si une condition initiale est donnée, déterminer la constante d'intégration
  • Étape 5 : Vérifier la solution obtenue

📙 3. Équations à variables séparables

3.1. Définition

Une équation différentielle est dite à variables séparables si elle peut s'écrire sous la forme :

📖 Définition : Équation à variables séparables

Une équation à variables séparables est une équation de la forme :

y' = g(x) · h(y)

ou de manière équivalente :

dydx = g(x) · h(y)

où g(x) ne dépend que de x et h(y) ne dépend que de y.

3.2. Méthode de résolution

Pour résoudre une équation à variables séparables, on sépare les variables puis on intègre.

📋 Méthode : Résolution d'une équation à variables séparables

  • Étape 1 : Écrire l'équation sous la forme dydx = g(x) · h(y)
  • Étape 2 : Séparer les variables : dyh(y) = g(x) dx
  • Étape 3 : Intégrer les deux membres : ∫dyh(y) = ∫ g(x) dx
  • Étape 4 : Exprimer y en fonction de x (si possible)
  • Étape 5 : Déterminer la constante d'intégration avec la condition initiale

💡 Exemple 1 : Résolution d'une équation à variables séparables

Résoudre : y' = 2xy avec y(0) = 1

Étape 1 : dydx = 2xy

Étape 2 : dyy = 2x dx

Étape 3 : ∫dyy = ∫ 2x dx ⟹ ln|y| = x² + C

Étape 4 : y = ex² + C = eC · ex² = K · ex² où K = eC

Étape 5 : y(0) = K · e⁰ = K = 1 ⟹ K = 1

Solution particulière : y = ex²

💡 Exemple 2 : Équation plus complexe

Résoudre : dydx = yx + 1 avec x > −1 et y > 0

Séparation : dyy = dxx + 1

Intégration : ∫dyy = ∫dxx + 1

ln|y| = ln|x + 1| + C

Propriété : ln|y| = ln|x + 1| + C ⟹ ln|y| − ln|x + 1| = C

⟹ ln|y||x + 1| = C ⟹ yx + 1 = eC = K

Solution générale : y = K(x + 1)

💡 Exemple 3 : Sans condition initiale

Résoudre : dydx = 3x²y²

Séparation : dyy² = 3x² dx

Intégration : ∫ y−2 dy = ∫ 3x² dx

−y−1 = x³ + C

Solution générale : y = −1x³ + C

On peut aussi écrire : y = 1C − x³ en posant K = −C

⚠️ Cas où h(y) = 0

Si h(y₀) = 0 pour une valeur y₀, alors la fonction constante y(x) = y₀ est une solution de l'équation différentielle, appelée solution singulière.

Exemple : dydx = y ⟹ si y = 0, alors dydx = 0 = y ✓

La solution y = 0 est une solution singulière.

3.3. Vérification de la solution

Il est important de vérifier que la solution trouvée satisfait bien l'équation différentielle originale.

📝 Vérification de y = ex²

Équation : y' = 2xy

y = ex² ⟹ y' = 2x · ex² = 2x · y

⟹ y' = 2xy ✓

De plus, y(0) = e⁰ = 1 ✓

📕 4. Équations linéaires du premier ordre

4.1. Définition

Une équation différentielle linéaire du premier ordre est une équation de la forme :

📖 Définition : Équation linéaire du 1er ordre

Une équation différentielle est dite linéaire du premier ordre si elle peut s'écrire sous la forme :

y' + a(x) · y = b(x)

où a(x) et b(x) sont des fonctions continues sur un intervalle I.

Si b(x) = 0, l'équation est dite homogène (ou sans second membre).

Si b(x) ≠ 0, l'équation est dite avec second membre.

✅

Équation homogène : y' + a(x)·y = 0

📝

Équation complète : y' + a(x)·y = b(x)

🔢

Coefficient a(x) : fonction connue de x

⚡

Second membre b(x) : fonction connue de x

4.2. Résolution de l'équation homogène

Commençons par résoudre l'équation homogène associée :

Équation homogène

y' + a(x) · y = 0

Résolution :

dydx = −a(x) · y

dyy = −a(x) dx   (pour y ≠ 0)

∫dyy = −∫ a(x) dx

ln|y| = −∫ a(x) dx + C

y(x) = K · exp(−∫ a(x) dx)

où K est une constante arbitraire.

💡 Exemple : Équation homogène

Résoudre : y' + 2y = 0

Méthode :

dydx = −2y

dyy = −2 dx

∫dyy = −2 ∫ dx

ln|y| = −2x + C

y = K · e−2x

Vérification : y' = −2K·e−2x = −2y ✓

4.3. Méthode de variation de la constante

Pour résoudre l'équation complète y' + a(x)·y = b(x), on utilise la méthode de variation de la constante.

📐 Théorème : Solution générale d'une équation linéaire du 1er ordre

La solution générale de l'équation y' + a(x)·y = b(x) est :

y(x) = μ(x) · exp(−∫ a(x) dx)

où μ'(x) = b(x) · exp(∫ a(x) dx)

Autrement dit :

y(x) = e−∫ a(x) dx · [∫ b(x) · e∫ a(x) dx dx + C]

📋 Méthode : Résolution d'une équation linéaire du 1er ordre

  • Étape 1 : Identifier a(x) et b(x) dans y' + a(x)·y = b(x)
  • Étape 2 : Calculer la primitive de a(x) : A(x) = ∫ a(x) dx
  • Étape 3 : Calculer le facteur intégrant : μ(x) = eA(x)
  • Étape 4 : Calculer ∫ b(x)·μ(x) dx
  • Étape 5 : La solution générale est : y = μ−1 · [∫ b(x)·μ(x) dx + C]

💡 Exemple 1 : Résolution complète

Résoudre : y' + 2y = x avec y(0) = 1

Étape 1 : a(x) = 2,   b(x) = x

Étape 2 : A(x) = ∫ 2 dx = 2x

Étape 3 : μ(x) = e2x

Étape 4 : ∫ b(x)·μ(x) dx = ∫ x·e2x dx

Par intégration par parties : ∫ x·e2x dx = x·e2x2 − ∫e2x2 dx = x·e2x2 − e2x4 + C

Étape 5 : y = e−2x · [x·e2x2 − e2x4 + C]

y = x2 − 14 + C·e−2x

Condition initiale : y(0) = 02 − 14 + C = 1 ⟹ −14 + C = 1 ⟹ C = 54

Solution particulière : y = x2 − 14 + 54·e−2x

💡 Exemple 2 : Équation avec a(x) non constant

Résoudre : y' + 1x·y = x²

Étape 1 : a(x) = 1x,   b(x) = x²

Étape 2 : A(x) = ∫1x dx = ln|x|

Étape 3 : μ(x) = eln|x| = x

Étape 4 : ∫ b(x)·μ(x) dx = ∫ x²·x dx = ∫ x³ dx = x⁴4

Solution générale :

y = 1x · (x⁴4 + C) = x³4 + Cx

Vérification : y' = 3x²4 − Cx²

y' + 1x·y = 3x²4 − Cx² + 1x·(x³4 + Cx) = 3x²4 − Cx² + x²4 + Cx² = x² ✓

4.4. Structure de l'ensemble des solutions

📐 Théorème : Structure des solutions

Pour l'équation linéaire y' + a(x)·y = b(x) :

L'ensemble des solutions est :

y(x) = yp(x) + yh(x)

où :

  • yh(x) est la solution générale de l'équation homogène (y' + a(x)·y = 0)
  • yp(x) est une solution particulière de l'équation complète

C'est le principe de superposition des solutions.

📓 5. Équations différentielles du second ordre à coefficients constants

5.1. Forme générale

Une équation différentielle linéaire du second ordre à coefficients constants est de la forme :

Forme générale d'une équation du 2nd ordre à coefficients constants

ay'' + by' + cy = f(x)

où a, b, c sont des constantes réelles (a ≠ 0) et f(x) est une fonction.

Équation homogène : ay'' + by' + cy = 0

Équation complète : ay'' + by' + cy = f(x)

5.2. Résolution de l'équation homogène

Pour résoudre ay'' + by' + cy = 0, on cherche des solutions de la forme y = erx.

📐 Théorème : Équation caractéristique

En substituant y = erx, y' = r·erx, y'' = r²·erx dans ay'' + by' + cy = 0, on obtient :

ar² + br + c = 0

C'est l'équation caractéristique associée à l'équation différentielle.

La nature des solutions dépend du discriminant Δ = b² − 4ac.

Δ>0

Δ > 0 : Deux racines réelles distinctes r₁ et r₂

Δ=0

Δ = 0 : Une racine réelle double r₀ = −b2a

Δ<0

Δ < 0 : Deux racines complexes conjuguées

5.3. Les trois cas possibles

📐 Théorème : Solutions de l'équation homogène

Cas 1 : Δ > 0 (racines réelles distinctes)

L'équation caractéristique a deux racines réelles r₁ et r₂.

Solution générale : y(x) = A·er₁x + B·er₂x

où A et B sont des constantes arbitraires.

💡 Exemple : Δ > 0

Résoudre : y'' − 3y' + 2y = 0

Équation caractéristique : r² − 3r + 2 = 0

Discriminant : Δ = 9 − 8 = 1 > 0

Racines : r₁ = 3+12 = 2,   r₂ = 3−12 = 1

Solution générale : y = A·e2x + B·ex

📐 Théorème : Solution pour Δ = 0

Cas 2 : Δ = 0 (racine réelle double)

L'équation caractéristique a une racine double r₀ = −b2a.

Solution générale : y(x) = (A + Bx)·er₀x

où A et B sont des constantes arbitraires.

💡 Exemple : Δ = 0

Résoudre : y'' + 4y' + 4y = 0

Équation caractéristique : r² + 4r + 4 = 0

Discriminant : Δ = 16 − 16 = 0

Racine double : r₀ = −42 = −2

Solution générale : y = (A + Bx)·e−2x

📐 Théorème : Solution pour Δ < 0

Cas 3 : Δ < 0 (racines complexes conjuguées)

L'équation caractéristique a deux racines complexes : r₁ = α + iβ et r₂ = α − iβ

où α = −b2a et β = √(4ac − b²)2a

Solution générale : y(x) = eαx·[A·cos(βx) + B·sin(βx)]

où A et B sont des constantes arbitraires.

💡 Exemple : Δ < 0

Résoudre : y'' + 4y = 0

Équation caractéristique : r² + 4 = 0

Discriminant : Δ = 0 − 16 = −16 < 0

Racines complexes : r = ±2i

Donc α = 0 et β = 2

Solution générale : y = A·cos(2x) + B·sin(2x)

C'est l'équation d'un mouvement oscillatoire (ressort, pendule...)

5.4. Solution particulière de l'équation complète

Pour trouver une solution particulière de ay'' + by' + cy = f(x), on peut utiliser la méthode de variation des constantes ou la méthode des coefficients indéterminés.

📋 Méthode : Coefficients indéterminés

Pour certains seconds membres f(x), on cherche yp de la même forme :

  • f(x) = P(x) (polynôme) : chercher yp = Q(x) de même degré
  • f(x) = eαx : chercher yp = A·eαx ou x·A·eαx si α est racine
  • f(x) = cos(βx) ou sin(βx) : chercher yp = A·cos(βx) + B·sin(βx)

💡 Exemple : Second membre polynomial

Résoudre : y'' − 3y' + 2y = 2x + 1

1) Résolution de l'équation homogène : yh = A·e2x + B·ex

2) Recherche d'une solution particulière : f(x) = 2x + 1 (polynôme de degré 1)

Cherchons yp = ax + b

yp' = a,   yp'' = 0

Substitution : 0 − 3a + 2(ax + b) = 2x + 1

2ax + (−3a + 2b) = 2x + 1

⟹ 2a = 2 ⟹ a = 1

⟹ −3(1) + 2b = 1 ⟹ −3 + 2b = 1 ⟹ 2b = 4 ⟹ b = 2

Solution particulière : yp = x + 2

Solution générale : y = A·e2x + B·ex + x + 2

5.5. Conditions initiales pour une équation du second ordre

Pour déterminer complètement la solution, une équation du second ordre nécessite deux conditions initiales :

Conditions initiales

y(x₀) = y₀ (valeur initiale de la fonction)

y'(x₀) = y₁ (valeur initiale de la dérivée)

Ces deux conditions permettent de déterminer les constantes A et B.

💡 Exemple : Problème de Cauchy du second ordre

Résoudre : y'' + 4y = 0 avec y(0) = 2 et y'(0) = 0

Solution générale : y = A·cos(2x) + B·sin(2x)

Condition 1 : y(0) = A·1 + B·0 = 2 ⟹ A = 2

Calcul de y' : y' = −2A·sin(2x) + 2B·cos(2x)

Condition 2 : y'(0) = −2A·0 + 2B·1 = 0 ⟹ B = 0

Solution particulière : y = 2·cos(2x)

📔 6. Problèmes conduisant à une équation différentielle

6.1. Applications en physique

Les équations différentielles sont omniprésentes en physique pour modéliser l'évolution de systèmes dynamiques.

💡 Exemple : Chute libre avec résistance de l'air

Un objet de masse m tombe sous l'effet de la gravité avec une résistance proportionnelle à la vitesse.

Modélisation : mv' + kv = mg

où v(t) est la vitesse, g l'accélération de la pesanteur, et k le coefficient de frottement.

Résolution :

v' + kmv = g

Solution : v(t) = A·e−kt/m + mgk

Condition initiale v(0) = 0 :

0 = A + mgk ⟹ A = −mgk

v(t) = mgk·(1 − e−kt/m)

Cette solution montre que la vitesse tend vers une vitesse limite vlim = mgk.

💡 Exemple : Circuit RL

Un circuit électrique contenant une résistance R et une inductance L est soumis à une tension constante E.

Modélisation : Li' + Ri = E

où i(t) est l'intensité du courant.

Résolution :

i' + RLi = EL

Solution : i(t) = A·e−Rt/L + ER

Condition i(0) = 0 :

0 = A + ER ⟹ A = −ER

i(t) = ER·(1 − e−Rt/L)

L'intensité croît exponentiellement vers la valeur limite Ilim = ER.

💡 Exemple : Oscillations d'un ressort

Un ressort de constante de raideur k, attaché à une masse m, oscille avec un amortissement proportionnel à la vitesse.

Modélisation : mx'' + cx' + kx = 0

où x(t) est l'élongation, et c le coefficient d'amortissement.

Cas non amorti (c = 0) : x'' + kmx = 0

Solution : x(t) = A·cos(ωt) + B·sin(ωt) avec ω = √km

C'est un mouvement harmonique simple de période T = 2πω.

6.2. Applications en biologie

💡 Exemple : Croissance exponentielle

Un population de bactéries double chaque heure. Modéliser l'évolution de la population.

Modélisation : dNdt = kN

où N(t) est le nombre de bactéries et k est le taux de croissance.

Résolution :

dNN = k dt

∫dNN = ∫ k dt

ln|N| = kt + C

N(t) = N₀·ekt

où N₀ est la population initiale.

💡 Exemple : Modèle logistique (croissance limitée)

Une population ne peut pas croître indéfiniment à cause des ressources limitées.

Modélisation : dNdt = kN(1 − NK)

où K est la capacité maximale de l'environnement.

Résolution :

dNN(1 − N/K) = k dt

∫ [1N + 1K−N] dN = k ∫ dt

ln|N| − ln|K−N| = kt + C

lnNK−N = kt + C

NK−N = A·ekt

N(t) = K·A·ekt1 + A·ekt

La population tend vers K (capacité limite) quand t → ∞.

6.3. Applications en économie

💡 Exemple : Intérêts composés continus

Un capital C₀ est placé avec un taux d'intérêt annuel r, capitalisé en continu.

Modélisation : dCdt = rC

Résolution :

C(t) = C₀·ert

Après t années, le capital est C₀·ert.

6.4. Méthode générale de résolution de problèmes appliqués

📋 Méthode : Modéliser et résoudre un problème réel

  • Étape 1 : Identifier les variables : Définir la grandeur qui varie (x, y, t...)
  • Étape 2 : Écrire les hypothèses : Traduire les lois physiques/économiques en équations
  • Étape 3 : Former l'équation différentielle : Obtenir une relation entre la fonction et ses dérivées
  • Étape 4 : Résoudre l'équation : Trouver la solution générale
  • Étape 5 : Appliquer les conditions initiales : Déterminer les constantes
  • Étape 6 : Interpréter les résultats : Donner un sens physique/économique à la solution

⚠️ Conseils pratiques

1. Toujours vérifier les unités dans vos calculs

2. Vérifier que la solution satisfait bien l'équation et les conditions initiales

3. Interpréter le comportement asymptotique (limite quand t → ∞)

4. Vérifier la cohérence des résultats avec le bon sens physique

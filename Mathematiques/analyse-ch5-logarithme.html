📘 1. Définition du logarithme népérien

Motivation — Pourquoi ln ?

La fonction exponentielle exp(x) = eˣ est strictement croissante sur ℝ et prend toutes les valeurs positives. Elle est donc bijective de ℝ vers ]0, +∞[.

Sa bijection réciproque est appelée logarithme népérien, notée ln.

Définition formelle

Définition
Pour tout x > 0 :    ln(x) = y  ⇔  eʸ = x
Propriété fondamentale
e^(ln x) = x    (∀x > 0)
ln(eˣ) = x    (∀x ∈ ℝ)

Le nombre e est la base du logarithme népérien :

Le nombre e
e ≈ 2,718281828...
e = lim(n→∞) (1 + 1/n)ⁿ

Valeurs remarquables

🔢

ln(1) = 0 car e⁰ = 1

🔢

ln(e) = 1 car e¹ = e

🔢

ln(e²) = 2 — ln(eⁿ) = n

🔢

ln(0⁺) = -∞ — lim(x→0⁺) ln(x) = -∞

⚠️ Domaine de définition strict

ln(x) n'existe que pour x > 0

→ ln(0) n'existe pas

→ ln(-3) n'existe pas

→ ln(x) défini sur ]0, +∞[ uniquement

Représentation graphique

La courbe de ln(x) est l'image de la courbe de eˣ par la symétrie d'axe y = x.

📊

Domaine : ]0, +∞[

📊

Image : ℝ (ln prend toutes les valeurs réelles)

📊

Strictement croissante sur ]0, +∞[

📊

Asymptote verticale : axe des ordonnées (x = 0)

📗 2. Propriétés algébriques de ln

Propriétés fondamentales

Produit
ln(a × b) = ln(a) + ln(b)   |   ∀a > 0, ∀b > 0
Quotient
ln(a / b) = ln(a) - ln(b)   |   ∀a > 0, ∀b > 0
Puissance
ln(aⁿ) = n × ln(a)   |   ∀a > 0, ∀n ∈ ℝ
Racine
ln(√a) = (1/2) × ln(a)   |   ln(a^(1/n)) = (1/n) × ln(a)

Autres propriétés utiles

🔑

ln(a) = ln(b) ⇔ a = b (car ln est injective)

🔑

ln(a) > ln(b) ⇔ a > b (car ln est croissante)

🔑

ln(1/a) = -ln(a)

🔑

ln(a^b × a^c) = (b+c) × ln(a)

💡 Exemple 1 — Calcul avec ln

Calculer : A = ln(2) + ln(3) - ln(6)

A = ln(2×3) - ln(6) = ln(6) - ln(6) = 0

💡 Exemple 2 — Simplification

Simplifier : B = ln(e²) + ln(e³) - 2ln(e)

B = 2 + 3 - 2×1 = 3

💡 Exemple 3 — Expression avec ln

Écrire en fonction de ln(2) et ln(3) :

ln(12) = ln(3×4) = ln(3) + ln(4) = ln(3) + 2ln(2)

ln(1/9) = ln(1) - ln(9) = 0 - 2ln(3) = -2ln(3)

ln(√72) = (1/2)ln(72) = (1/2)ln(8×9) = (1/2)(ln(8)+ln(9))

= (1/2)(3ln(2) + 2ln(3)) = (3/2)ln(2) + ln(3)

⚠️ Erreurs fréquentes à éviter

❌ ln(a + b) ≠ ln(a) + ln(b)

❌ ln(a²) ≠ 2ln(a) si on ne sait pas si a > 0

✅ ln(a²) = 2ln|a| (en toute généralité)

❌ ln(a)/ln(b) ≠ log_b(a) en base e, mais c'est bien la définition du log en base b

📙 3. Limites et dérivée de ln

Dérivée de ln(u(x))

Dérivée de ln(x)
(ln x)' = 1/x   |   ∀x > 0
Dérivée de ln(u(x)) — Formule importante
[ln(u(x))]' = u'(x) / u(x)   |   avec u(x) > 0

Limites remarquables

Limites fondamentales
lim(x→0⁺) ln(x) = -∞
lim(x→+∞) ln(x) = +∞
lim(x→0⁺) x·ln(x) = 0
lim(x→0⁺) ln(1+x)/x = 1
lim(x→+∞) ln(x)/x = 0
Croissance comparée
lim(x→0⁺) xᵅ · ln(x) = 0   |   ∀α > 0
lim(x→+∞) ln(x) / xᵅ = 0   |   ∀α > 0
lim(x→+∞) eˣ / xⁿ = +∞   |   ∀n ∈ ℕ

Primitives

Primitives avec ln
∫ (1/x) dx = ln|x| + C
∫ u'(x)/u(x) dx = ln|u(x)| + C

💡 Exemple 1 — Dérivée

f(x) = ln(x² + 1)

f'(x) = (2x) / (x² + 1)

Signe de f'(x) = signe de x

f est décroissante sur ]-∞, 0[ et croissante sur ]0, +∞[

💡 Exemple 2 — Primitive

Calculer ∫ x/(x²+1) dx

On pose u = x² + 1 → du = 2x dx

∫ x/(x²+1) dx = (1/2)∫ du/u = (1/2)ln|u| + C

= (1/2)ln(x²+1) + C

💡 Exemple 3 — Limite avec ln

lim(x→0⁺) (ln x) / (x - 1)

Quand x → 0⁺ : ln x → -∞, x - 1 → -1

= (-∞)/(-1) = +∞

💡 Exemple 4 — Forme indéterminée

lim(x→0⁺) x·ln(x)

Forme : 0 × (-∞)

x·ln(x) = ln(x) / (1/x)

Quand x→0⁺ : num → -∞, denom → +∞

Règle de l'Hôpital : (1/x) / (-1/x²) = -x → 0

Donc lim(x→0⁺) x·ln(x) = 0

📕 4. Équations et inéquations logarithmiques

Équations

Pour résoudre une équation avec ln, on utilise la définition : ln(a) = ln(b) ⇔ a = b > 0

Méthode de résolution
ln(f(x)) = ln(g(x))  ⇔  f(x) = g(x)  et  f(x) > 0  et  g(x) > 0
Forme ln(u) = k
ln(u) = k  ⇔  u = eᵏ   (u > 0 toujours satisfait si k est réel)

💡 Exemple 1 — Équation simple

ln(2x - 1) = 3

⇔ 2x - 1 = e³

⇔ 2x = e³ + 1

⇔ x = (e³ + 1)/2

Vérification : 2x - 1 = e³ > 0 ✓

Solution : x = (e³ + 1)/2

💡 Exemple 2 — Équation avec ln

ln(x + 1) = ln(2x - 3)

⇔ x + 1 = 2x - 3 et x + 1 > 0 et 2x - 3 > 0

⇔ x = 4

Conditions : x > 1 (pour x+1>0) et x > 1,5 (pour 2x-3>0)

⇔ x > 1,5 ✓ pour x = 4

Solution : x = 4

💡 Exemple 3 — Équation se ramenant à un produit

ln(x) + ln(x - 2) = ln(3)

⇔ ln(x(x - 2)) = ln(3)

⇔ x(x - 2) = 3

⇔ x² - 2x - 3 = 0

⇔ (x - 3)(x + 1) = 0

⇔ x = 3 ou x = -1

Conditions : x > 0 et x > 2 → x > 2

Seule x = 3 convient

Solution : x = 3

Inéquations

Pour les inéquations, on utilise la croissance de ln : si a > 0 et b > 0 :

Inéquations
ln(a) < ln(b)  ⇔  a < b
ln(a) > ln(b)  ⇔  a > b
ln(a) < k  ⇔  a < eᵏ
ln(a) > k  ⇔  a > eᵏ

💡 Exemple 4 — Inéquation

ln(2x + 1) ≤ 2

⇔ 2x + 1 ≤ e²   (car ln croissante et ln(x) ≤ k ⇔ x ≤ eᵏ)

⇔ 2x ≤ e² - 1

⇔ x ≤ (e² - 1)/2

Condition : 2x + 1 > 0 → x > -1/2

Solution : ]-1/2, (e² - 1)/2]

💡 Exemple 5 — Inéquation produit

ln(x) + ln(x - 3) > ln(10)

⇔ ln(x(x-3)) > ln(10)

⇔ x(x-3) > 10   (car ln croissante)

⇔ x² - 3x - 10 > 0

⇔ (x - 5)(x + 2) > 0

⇔ x < -2 ou x > 5

Conditions : x > 0 et x > 3 → x > 3

Solution : ]5, +∞[

📒 5. Logarithme décimal et applications

Logarithme décimal (base 10)

Définition
log(x) = log₁₀(x) = ln(x) / ln(10)   |   x > 0

Le logarithme décimal était utilisé avant les calculatrices pour simplifier les calculs (produits → sommes).

📐

log(10) = 1

📐

log(10ⁿ) = n

📐

log(1) = 0

📐

log(a·b) = log(a) + log(b)

Changement de base

Formule générale de changement de base
log_b(a) = ln(a) / ln(b) = 1 / log_a(b)

Applications du logarithme

📊

pH : pH = -log[H⁺] — mesure de l'acidité

📊

Décibel : L = 10·log(P/P₀) — niveau sonore

📊

Échelle de Richter : M = log(A/A₀) — magnitude sismique

📊

Intérêts composés : ln(Capital) croît linéairement avec le temps

💡 Exemple — pH

Si [H⁺] = 10⁻⁷ mol/L → pH = -log(10⁻⁷) = 7 (neutre)

Si [H⁺] = 10⁻³ mol/L → pH = 3 (acide)

Une diminution du pH de 1 correspond à une multiplication de [H⁺] par 10

📓 6. Exercices résolus

📝 Exercice 1 — Dérivation

Calculer la dérivée de f(x) = x·ln(x) - x

Solution :

f'(x) = 1·ln(x) + x·(1/x) - 1 = ln(x) + 1 - 1 = ln(x)

Remarque : une primitive de ln(x) est x·ln(x) - x

📝 Exercice 2 — Limite

Calculer lim(x→0⁺) (ln(1+x))/x

Solution :

lim(x→0⁺) (ln(1+x))/x = lim(x→0⁺) (ln(1+x) - ln(1))/x

= (ln)' à x=0 = 1/(1+0) = 1

Ou par changement : posons u = 1+x → u→1 : lim(u→1) ln(u)/(u-1) = 1

📝 Exercice 3 — Équation

Résoudre : ln(x² - 4) = ln(x + 2) + ln(2)

Solution :

⇔ ln(x² - 4) = ln(2(x + 2))

⇔ x² - 4 = 2(x + 2)

⇔ x² - 2x - 8 = 0

⇔ (x - 4)(x + 2) = 0

⇔ x = 4 ou x = -2

Conditions : x² - 4 > 0 → x < -2 ou x > 2

et x + 2 > 0 → x > -2

→ x = 4 convient, x = -2 ne convient pas (ln(0) n'existe pas)

Solution : x = 4

📝 Exercice 4 — Inéquation

Résoudre : ln²(x) - 3ln(x) + 2 ≤ 0

Solution :

On pose t = ln(x), t ∈ ℝ

t² - 3t + 2 ≤ 0

(t - 1)(t - 2) ≤ 0

⇔ 1 ≤ t ≤ 2

⇔ 1 ≤ ln(x) ≤ 2

⇔ e¹ ≤ x ≤ e²

⇔ e ≤ x ≤ e²

📝 Exercice 5 — Primitive avec ln

Calculer ∫ (2x+1)/(x²+x+1) dx

Solution :

On remarque que le numérateur est la dérivée du dénominateur

∫ (2x+1)/(x²+x+1) dx = ln(x²+x+1) + C

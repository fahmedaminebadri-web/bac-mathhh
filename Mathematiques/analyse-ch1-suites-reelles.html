📘 1. Généralités sur les suites

Une suite numérique est une fonction définie sur ℕ (l'ensemble des entiers naturels) qui associe à chaque entier naturel n un nombre réel un.

On note la suite (un)n∈ℕ ou simplement (un).

🔹 Définitions fondamentales

📌

Suite majorée : ∃ M ∈ ℝ tel que ∀n, un ≤ M

📌

Suite minorée : ∃ m ∈ ℝ tel que ∀n, un ≥ m

📌

Suite bornée : elle est à la fois majorée et minorée

📌

Suite croissante : ∀n, un+1 ≥ un

📌

Suite décroissante : ∀n, un+1 ≤ un

📌

Suite constante : ∀n, un+1 = un

Monotonie
Croissante : un+1 - un ≥ 0   |   Décroissante : un+1 - un ≤ 0

💡 Exemple : Suite bornée

un = (-1)n : la suite prend les valeurs 1 et -1

Elle est bornée : -1 ≤ un ≤ 1 pour tout n

Elle n'est ni croissante ni décroissante (elle alterne)

🔹 Modes de définition d'une suite

1. Forme explicite : un = f(n) où f est une fonction

Exemple
un = 3n + 2   →   u0 = 2, u1 = 5, u2 = 8, u3 = 11...

2. Relation de récurrence : un+1 = f(un) avec u0 connu

Exemple
u0 = 1, un+1 = 2un + 1
  →   u1 = 2(1) + 1 = 3
  →   u2 = 2(3) + 1 = 7
  →   u3 = 2(7) + 1 = 15...

📗 2. Suites arithmétiques

Une suite est arithmétique si la différence entre deux termes consécutifs est constante. Cette constante s'appelle la raison.

Définition
(un) est arithmétique ⟺ ∃ r ∈ ℝ tel que ∀n ∈ ℕ : un+1 = un + r

🔹 Formule du terme général

Terme général
un = u0 + n × r   |   ou    un = up + (n - p) × r

🔹 Somme des termes consécutifs

Somme arithmétique
S = u0 + u1 + ... + un = (n + 1) × (u0 + un) / 2

💡 Exemple 1 : Suite arithmétique

Soit (un) arithmétique avec u0 = 3 et r = 5

un = 3 + 5n

Calculer u10 : u10 = 3 + 5 × 10 = 53

S10 = 11 × (3 + 53) / 2 = 11 × 56 / 2 = 308

💡 Exemple 2 : Trouver la raison

On donne u2 = 7 et u5 = 19

u5 = u2 + 3r ⟹ 19 = 7 + 3r ⟹ r = 4

u0 = u2 - 2r = 7 - 8 = -1

Donc un = -1 + 4n

⚠️ Propriété importante

• Si r > 0 : la suite est strictement croissante

• Si r < 0 : la suite est strictement décroissante

• Si r = 0 : la suite est constante

📙 3. Suites géométriques

Une suite est géométrique si le quotient entre deux termes consécutifs est constant. Cette constante s'appelle la raison.

Définition
(un) est géométrique ⟺ ∃ q ∈ ℝ tel que ∀n ∈ ℕ : un+1 = un × q

🔹 Formule du terme général

Terme général
un = u0 × qn   |   ou    un = up × qn-p

🔹 Somme des termes consécutifs

Somme géométrique
Si q ≠ 1 : S = u0 × (1 - qn+1) / (1 - q)
Si q = 1 : S = (n + 1) × u0

💡 Exemple 1 : Suite géométrique

Soit (un) géométrique avec u0 = 2 et q = 3

un = 2 × 3n

Calculer u4 : u4 = 2 × 34 = 2 × 81 = 162

S4 = 2 × (1 - 35) / (1 - 3) = 2 × (1 - 243) / (-2) = 242

💡 Exemple 2 : Trouver la raison

On donne u2 = 12 et u5 = 96

u5 = u2 × q³ ⟹ 96 = 12 × q³ ⟹ q³ = 8 ⟹ q = 2

u0 = u2 / q² = 12 / 4 = 3

Donc un = 3 × 2n

⚠️ Comportement selon q

• Si q > 1 : la suite est strictement croissante (u0 > 0) ou décroissante (u0 < 0)

• Si 0 < q < 1 : la suite tend vers 0

• Si q = 1 : la suite est constante

• Si q < 0 : la suite alterne de signe

• Si |q| > 1 et u0 ≠ 0 : la suite diverge

📕 4. Limites de suites

La limite d'une suite décrit le comportement des termes quand n devient très grand.

🔹 Définitions

📐 Limite finie (convergence)

limn→+∞ un = ℓ ⟺ ∀ε > 0, ∃N ∈ ℕ : ∀n ≥ N, |un - ℓ| < ε

Autrement dit : un se rapproche arbitrairement de ℓ quand n grandit.

📐 Limite infinie (divergence)

limn→+∞ un = +∞ ⟺ ∀A > 0, ∃N : ∀n ≥ N, un > A

limn→+∞ un = -∞ ⟺ ∀A > 0, ∃N : ∀n ≥ N, un < -A

🔹 Limites des suites usuelles

Limites fondamentales
limn→+∞ n = +∞
limn→+∞ n² = +∞
limn→+∞ an =
0si |a| < 1
1si a = 1
+∞si a > 1

🔹 Opérations sur les limites

➕

Somme : ∞ + ∞ = ∞   |   ∞ + ℓ = ∞

✖️

Produit : (+∞) × (+∞) = +∞   |   (+∞) × ℓ = ±∞

➗

Quotient : ℓ/∞ = 0   |   ℓ/0 = ±∞ (selon le signe)

⚡

Formes indéterminées : ∞ - ∞, 0 × ∞, 0/0, ∞/∞, 0⁰, ∞⁰

💡 Exemple : Lever l'indétermination

Calculer limn→+∞ (n² - 3n + 1)

→ Forme indéterminée : ∞ - ∞

→ Factoriser n² : n²(1 - 3/n + 1/n²)

→ Quand n → ∞ : -3/n → 0 et 1/n² → 0

→ lim = +∞ × 1 = +∞

💡 Exemple : Suite géométrique

un = 5 × (0.7)n

Comme |0.7| < 1, limn→+∞ (0.7)n = 0

Donc limn→+∞ un = 5 × 0 = 0

🔹 Théorèmes importants

📐 Théorème de convergence monotone

• Toute suite croissante et majorée converge.

• Toute suite décroissante et minorée converge.

📐 Théorème des gendarmess

Si un ≤ vn ≤ wn et lim un = lim wn = ℓ

Alors lim vn = ℓ

💡 Application du théorème des gendarmess

Montrer que limn→+∞ sin(n)/n = 0

On a : -1 ≤ sin(n) ≤ 1

Donc : -1/n ≤ sin(n)/n ≤ 1/n

Or lim(-1/n) = lim(1/n) = 0

Par le théorème des gendarmess : lim = 0

📒 5. Raisonner par récurrence

Le raisonnement par récurrence est une méthode de démonstration essentielle pour prouver qu'une propriété dépendant de n est vraie pour tout entier naturel.

🔹 Principe de la récurrence

📐 Les 3 étapes

1. Initialisation : Vérifier que la propriété est vraie pour n = 0 (ou n₀)

2. Hypothèse de récurrence (HR) : Supposer que la propriété est vraie pour un certain rang n

3. Hérédité : Démontrer que si la propriété est vraie au rang n, alors elle est vraie au rang n+1

Schéma logique
P(0) vraie  →  Si P(n) vraie alors P(n+1) vraie  →  Conclusion : P(n) vraie ∀n ∈ ℕ

💡 Exemple : Démontrer une formule

Montrer que : 1 + 2 + 3 + ... + n = n(n+1)/2 pour tout n ∈ ℕ*

1. Initialisation :

Pour n = 1 : 1 = 1(1+1)/2 = 1 ✓

2. Hypothèse de récurrence :

Supposons que 1 + 2 + ... + n = n(n+1)/2

3. Hérédité :

1 + 2 + ... + n + (n+1) = n(n+1)/2 + (n+1)

= n(n+1)/2 + 2(n+1)/2 = (n+2)(n+1)/2

= (n+1)[(n+1)+1]/2 ✓

Conclusion : La formule est vraie pour tout n ∈ ℕ*

💡 Exemple : Suite définie par récurrence

Soit u0 = 1 et un+1 = 2un + 1

Conjecture : un = 2n+1 - 1

1. Initialisation :

u0 = 1 et 20+1 - 1 = 2 - 1 = 1 ✓

2. HR : Supposons un = 2n+1 - 1

3. Hérédité :

un+1 = 2un + 1 = 2(2n+1 - 1) + 1

= 2n+2 - 2 + 1 = 2n+2 - 1 ✓

⚠️ Types de récurrence

• Récurrence simple : P(n) → P(n+1)

• Récurrence forte : P(0) et [P(0) et P(1) et ... et P(n)] → P(n+1)

• Récurrence double : P(n) et P(n+1) → P(n+2)

🔹 Application aux limites

💡 Montrer la convergence d'une suite

Soit u0 = 1 et un+1 = √(2 + un)

Montrons que (un) est croissante et majorée par 2

1. Initialisation : u0 = 1 < 2 ✓

2. HR : Supposons un < 2

3. Hérédité :

un+1 = √(2 + un) < √(2 + 2) = √4 = 2 ✓

Et un+1 - un = √(2 + un) - un > 0 (facile à montrer)

Conclusion : La suite est croissante et majorée par 2, donc elle converge.

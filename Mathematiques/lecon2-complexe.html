📘 1. Introduction et historique des nombres complexes

1.1 Pourquoi avons-nous besoin des nombres complexes ?

Depuis l'Antiquité, les mathématiciens ont progressivement enrichi notre conception du nombre. Cette évolution peut se résumer ainsi :

🔢

ℕ : Les entiers naturels (0, 1, 2, 3...) permettent de compter

➖

ℤ : Les entiers relatifs (...-2, -1, 0, 1, 2...) permettent de soustraire

➗

ℚ : Les rationnels (a/b) permettent de diviser

📏

ℝ : Les réels permettent de mesurer toutes les grandeurs continues (√2, π, e...)

Cependant, l'ensemble ℝ reste insuffisant pour résoudre certaines équations algébriques fondamentales. Considérons l'équation la plus simple :

L'équation impossible dans ℝ
x² + 1 = 0

Cette équation n'admet aucune solution réelle car aucun nombre réel élevé au carré ne peut donner un résultat négatif. Pour résoudre ce problème fondamental, les mathématiciens du XVIe siècle ont introduit progressivement le concept de « nombre imaginaire ».

1.2 Historique

📜

Cardan (1545) : Premier à utiliser des « nombres impossibles » pour résoudre des équations cubiques

📜

Bombelli (1572) : Formalise les règles de calcul avec ces nombres « imaginaires »

📜

Euler (1748) : Introduit le symbole i pour désigner l'unité imaginaire

📜

Gauss (1831) : Donne une interprétation géométrique et nommée « plan complexe »

⚠️ Anecdote historique importante

Le terme « imaginaire » a été donné par Descartes en 1637 de manière péjorative, car il considérait ces nombres comme « impossibles » ou « imaginaires ».

Malgré cela, ces nombres se sont révélés être des outils mathématiques essentiels et fondamentaux !

📗 2. Forme algébrique d'un nombre complexe

2.1 Définition de l'unité imaginaire

On appelle unité imaginaire le nombre noté i qui vérifie la propriété fondamentale suivante :

Propriété fondamentale de i
i² = -1

C'est cette simple propriété qui permet de résoudre l'équation x² + 1 = 0 :

Solution complexe
x² = -1 ⟹ x = ±i

2.2 Définition de l'ensemble ℂ

L'ensemble des nombres complexes, noté ℂ, est l'ensemble de tous les nombres de la forme :

Forme algébrique d'un complexe
z = a + bi où a ∈ ℝ et b ∈ ℝ

Cette écriture est appelée forme algébrique (ou forme canonique) du nombre complexe z.

2.3 Partie réelle et partie imaginaire

📌

a est appelée la partie réelle de z, notée Re(z) ou ℜ(z)

📌

b est appelée la partie imaginaire de z, notée Im(z)

💡 Exemples concrets

• z₁ = 3 + 4i → Re(z₁) = 3 et Im(z₁) = 4

• z₂ = -5 - 2i → Re(z₂) = -5 et Im(z₂) = -2

• z₃ = 7 → Re(z₃) = 7 et Im(z₃) = 0 (c'est un nombre réel)

• z₄ = -3i → Re(z₄) = 0 et Im(z₄) = -3 (c'est un imaginaire pur)

2.4 Cas particuliers importants

🔢

Si b = 0, alors z = a est un nombre réel. On a donc ℝ ⊂ ℂ

🔢

Si a = 0 et b ≠ 0, alors z = bi est un imaginaire pur

🔢

Le nombre 0 = 0 + 0i est à la fois réel et imaginaire pur

🔢

Deux nombres complexes sont égaux si et seulement si ils ont mêmes parties réelles et imaginaires

⛔ Condition d'égalité

Soient z₁ = a + bi et z₂ = c + di.

z₁ = z₂ ⟺ a = c ET b = d

C'est la seule façon d'égaler deux nombres complexes !

📙 3. Opérations sur les nombres complexes

3.1 Addition et soustraction

Soient z₁ = a + bi et z₂ = c + di deux nombres complexes. On définit :

Addition
z₁ + z₂ = (a + c) + (b + d)i
Soustraction
z₁ - z₂ = (a - c) + (b - d)i

L'addition et la soustraction s'effectuent donc en additionnant (ou soustrayant) séparément les parties réelles et les parties imaginaires.

💡 Exemple d'addition

z₁ = 3 + 5i et z₂ = 2 - 3i

z₁ + z₂ = (3 + 2) + (5 - 3)i = 5 + 2i

z₁ - z₂ = (3 - 2) + (5 + 3)i = 1 + 8i

3.2 Multiplication

La multiplication de deux nombres complexes se fait en développant et en utilisant la règle fondamentale i² = -1 :

Multiplication
(a + bi)(c + di) = (ac - bd) + (ad + bc)i

Cette formule se déduit directement du développement algébrique classique en remarquant que (bi)(di) = bd·i² = bd·(-1) = -bd.

💡 Exemple détaillé : Calcul de (3 + 2i)(1 - 5i)

= 3×1 + 3×(-5i) + 2i×1 + 2i×(-5i)

= 3 - 15i + 2i - 10i²

= 3 - 13i - 10×(-1)

= 3 - 13i + 10

= 13 - 13i

💡 Autres exemples de multiplication

• i × i = i² = -1

• 2i × 3i = 6i² = -6

• (1 + i)² = 1 + 2i + i² = 1 + 2i - 1 = 2i

• (1 + i)(1 - i) = 1 - i² = 1 - (-1) = 2

3.3 Division

Pour diviser deux nombres complexes, on multiplie le numérateur et le dénominateur par le conjugué du dénominateur afin d'obtenir un dénominateur réel :

Division
(a + bi) / (c + di) = [(a + bi)(c - di)] / (c² + d²)

Cette technique permet de faire disparaître le « i » du dénominateur car (c + di)(c - di) = c² + d² ∈ ℝ.

💡 Exemple détaillé : Calcul de (2 + 3i) / (1 - 2i)

= (2 + 3i)(1 + 2i) / [(1)² + (-2)²]

= (2 + 4i + 3i + 6i²) / (1 + 4)

= (2 + 7i - 6) / 5

= (-4 + 7i) / 5

= -4/5 + (7/5)i

💡 Vérification du résultat

On peut vérifier : (-4/5 + 7i/5)(1 - 2i)

= (-4/5)(1) + (-4/5)(-2i) + (7i/5)(1) + (7i/5)(-2i)

= -4/5 + 8i/5 + 7i/5 - 14i²/5

= -4/5 + 15i/5 + 14/5

= 10/5 + 3i = 2 + 3i ✓

3.4 Propriétés algébriques de ℂ

➕

Associativité : (z₁ + z₂) + z₃ = z₁ + (z₂ + z₃)

➕

Commutativité : z₁ + z₂ = z₂ + z₁

🔢

Élément neutre : 0 + i0 = 0

🔄

Opposé : z + (-z) = 0

✖️

Associativité : (z₁z₂)z₃ = z₁(z₂z₃)

✖️

Commutativité : z₁z₂ = z₂z₁

🔢

Élément neutre : 1 × z = z

🔄

Inverse : z × (1/z) = 1 (pour z ≠ 0)

📕 4. Conjugué d'un nombre complexe

4.1 Définition

Le conjugué du nombre complexe z = a + bi est le nombre complexe noté z̄ (ou z*) défini par :

Conjugué d'un complexe
z̄ = a - bi

Géométriquement, le conjugué correspond à la symétrie du point M(a, b) par rapport à l'axe des réels dans le plan complexe.

💡 Exemples de conjugués

• Si z = 3 + 4i, alors z̄ = 3 - 4i

• Si z = -5 + 2i, alors z̄ = -5 - 2i

• Si z = 7 (réel), alors z̄ = 7 (le conjugué d'un réel est lui-même)

• Si z = -3i (imaginaire pur), alors z̄ = 3i

4.2 Propriétés fondamentales du conjugué

🔄

Double conjugaison : z̄̄ = z

➕

Conjugué d'une somme : (z₁ + z₂)̄ = z̄₁ + z̄₂

✖️

Conjugué d'un produit : (z₁ × z₂)̄ = z̄₁ × z̄₂

➗

Conjugué d'un quotient : (z₁ / z₂)̄ = z̄₁ / z̄₂

📈

Conjugué d'une puissance : (zⁿ)̄ = (z̄)ⁿ pour tout n ∈ ℤ

📌

Conjugué d'un réel : Si z ∈ ℝ, alors z̄ = z

📌

Conjugué d'un imaginaire pur : Si z = bi (b ∈ ℝ), alors z̄ = -z

➕

Parties réelles et imaginaires : Re(z) = (z + z̄)/2 et Im(z) = (z - z̄)/(2i)

4.3 Propriété essentielle du produit z × z̄

Produit complexe × conjugué
z × z̄ = (a + bi)(a - bi) = a² + b² ∈ ℝ⁺

Cette propriété est fondamentale et permet notamment :

📐

De calculer les modules (comme nous le verrons plus tard)

➗

De rationaliser les dénominateurs dans les divisions

✅

De vérifier qu'un résultat est correct en calculant z × z̄

💡 Application : Vérification d'une division

Vérifions que (1 + i) / (1 - i) = i

Calculons z × z̄ : (1 + i)(1 - i) = 1 - i² = 2 ∈ ℝ⁺ ✓

Multiplions le résultat par le dénominateur : i(1 - i) = i - i² = i + 1 = 1 + i ✓

Donc (1 + i) / (1 - i) = i est correct !

📒 5. Module d'un nombre complexe

5.1 Définition

Le module du nombre complexe z = a + bi est le nombre réel positif noté |z| défini par :

Module d'un complexe
|z| = √(a² + b²)

Géométriquement, le module représente la distance du point M(a, b) à l'origine O du plan complexe. C'est l'équivalent de la valeur absolue pour les nombres réels.

💡 Exemples de calcul de module

• z₁ = 3 + 4i → |z₁| = √(9 + 16) = √25 = 5

• z₂ = -5 + 12i → |z₂| = √(25 + 144) = √169 = 13

• z₃ = 1 - i → |z₃| = √(1 + 1) = √2

• z₄ = 7i → |z₄| = √(0 + 49) = 7

• z₅ = 5 → |z₅| = √(25 + 0) = 5

5.2 Propriétés du module

📌

Positivité : |z| ≥ 0 pour tout z ∈ ℂ, et |z| = 0 si et seulement si z = 0

✖️

Relation module-produit : |z₁ × z₂| = |z₁| × |z₂|

➗

Relation module-quotient : |z₁ / z₂| = |z₁| / |z₂| (avec z₂ ≠ 0)

📈

Relation module-puissance : |zⁿ| = |z|ⁿ pour tout n ∈ ℤ

5.3 Inégalités triangulaires

Première inégalité triangulaire
|z₁ + z₂| ≤ |z₁| + |z₂|
Seconde inégalité triangulaire
||z₁| - |z₂|| ≤ |z₁ + z₂|

Ces inégalités rappellent l'inégalité de la distance dans un triangle : un côté est toujours inférieur ou égal à la somme des deux autres.

5.4 Lien entre module et conjugué

Relation fondamentale module-conjugué
|z|² = z × z̄ = a² + b²

C'est une relation très importante qui permet de calculer facilement le module à partir du conjugué.

💡 Application : Vérification du module

Si z = 1 + i, alors z̄ = 1 - i

z × z̄ = (1 + i)(1 - i) = 1 - i² = 2

Donc |z|² = 2 et |z| = √2 ✓

⚠️ Erreurs fréquentes à éviter

• Ne confondez pas |z|² et z² !

• |z|² = a² + b² (toujours positif)

• z² = (a + bi)² = a² - b² + 2abi (peut être complexe)

• En général, |z₁ + z₂| ≠ |z₁| + |z₂| (c'est une inégalité, pas une égalité)

📘 6. Argument d'un nombre complexe

6.1 Définition

Soit z = a + bi un nombre complexe non nul. On appelle argument de z, noté Arg(z), tout angle θ ∈ ℝ tel que :

Définition de l'argument
cos θ = a/|z| et sin θ = b/|z|

L'argument représente l'angle orienté que fait le vecteur OM avec l'axe des réels positifs, mesuré en radians.

🔄

L'argument est défini à 2π près : si θ est un argument, alors θ + 2kπ (k ∈ ℤ) est aussi un argument

📌

L'argument n'est pas défini pour z = 0

6.2 Arguments remarquables

0️⃣

Arg(1) = 0 (point sur l'axe réel positif)

π️⃣

Arg(-1) = π (point sur l'axe réel négatif)

π/2️⃣

Arg(i) = π/2 (point sur l'axe imaginaire positif)

-π/2️⃣

Arg(-i) = -π/2 ou 3π/2 (axe imaginaire négatif)

6.3 Propriétés de l'argument

✖️

Argument d'un produit : Arg(z₁ × z₂) = Arg(z₁) + Arg(z₂) (mod 2π)

➗

Argument d'un quotient : Arg(z₁ / z₂) = Arg(z₁) - Arg(z₂) (mod 2π)

📈

Argument d'une puissance : Arg(zⁿ) = n·Arg(z) (mod 2π)

🔄

Argument du conjugué : Arg(z̄) = -Arg(z) (mod 2π)

💡 Exemple : Calcul d'argument

Déterminer un argument de z = 1 + i√3

|z| = √(1 + 3) = 2

cos θ = 1/2 et sin θ = √3/2

Donc θ = π/3 (mod 2π)

Arg(1 + i√3) = π/3 + 2kπ, k ∈ ℤ

6.4 Détermination pratique de l'argument

Pour déterminer l'argument de z = a + bi, on suit ces étapes :

1️⃣

Calculer le module r = √(a² + b²)

2️⃣

Calculer cos θ = a/r et sin θ = b/r

3️⃣

Identifier θ en utilisant le cercle trigonométrique

4️⃣

Vérifier que cos²θ + sin²θ = 1

📗 7. Forme trigonométrique

7.1 Définition

Tout nombre complexe non nul z peut s'écrire sous la forme trigonométrique :

Forme trigonométrique
z = r(cos θ + i sin θ)

où r = |z| est le module et θ = Arg(z) est un argument de z.

💡 Exemple fondamental : z = 1 + i

r = √(1² + 1²) = √2

cos θ = 1/√2 et sin θ = 1/√2

θ = π/4

Donc : z = √2(cos(π/4) + i sin(π/4))

7.2 Passage de la forme algébrique à la forme trigonométrique

Pour mettre z = a + bi sous forme trigonométrique :

1️⃣

Calculer le module : r = √(a² + b²)

2️⃣

Déterminer un argument θ vérifiant cos θ = a/r et sin θ = b/r

3️⃣

Écrire : z = r(cos θ + i sin θ)

💡 Exemple : z = -1 + i√3

r = √((-1)² + (√3)²) = √(1 + 3) = √4 = 2

cos θ = -1/2 et sin θ = √3/2

Ces valeurs correspondent à θ = 2π/3 (120°)

donc : z = 2(cos(2π/3) + i sin(2π/3))

💡 Exemple : z = -2 - 2i√3

r = √(4 + 12) = √16 = 4

cos θ = -2/4 = -1/2 et sin θ = -2√3/4 = -√3/2

Ces valeurs correspondent à θ = 4π/3 (240°) ou -2π/3

donc : z = 4(cos(4π/3) + i sin(4π/3))

7.3 Passage de la forme trigonométrique à la forme algébrique

Si z = r(cos θ + i sin θ), alors :

Passage inverse
a = r cos θ et b = r sin θ

donc z = a + bi

💡 Exemple : z = 3(cos(π/6) + i sin(π/6))

cos(π/6) = √3/2 et sin(π/6) = 1/2

a = 3 × √3/2 = 3√3/2

b = 3 × 1/2 = 3/2

donc : z = (3√3/2) + (3/2)i

7.4 Notation exponentielle

En utilisant la formule d'Euler (voir section suivante), on peut écrire :

Notation exponentielle
z = r·e^(iθ)

C'est une notation plus compacte et très utilisée en physique et en ingénierie.

📙 8. Formules d'Euler et Formule de Moivre

8.1 Formules d'Euler

Les formules d'Euler établissent un lien fondamental entre les fonctions exponentielles et trigonométriques :

Formules d'Euler
cos θ = (e^(iθ) + e^(-iθ)) / 2

sin θ = (e^(iθ) - e^(-iθ)) / 2i

Réciproquement, on peut écrire :

Formule d'Euler directe
e^(iθ) = cos θ + i sin θ

C'est la célèbre formule d'Euler, considérée comme l'une des plus belles formules mathématiques. En particulier :

📌

e^(iπ) + 1 = 0 (relation d'Euler, reliant les 5 constantes fondamentales)

📌

e^(iπ/2) = i

📌

e^(-iπ/2) = -i

📌

e^(2iπ) = 1

8.2 Formule de Moivre

La formule de De Moivre permet de calculer les puissances de nombres complexes sous forme trigonométrique :

Formule de Moivre
[cos θ + i sin θ]^n = cos(nθ) + i sin(nθ) pour tout n ∈ ℤ

Cette formule est particulièrement utile pour :

📐

Calculer les puissances de complexes

📐

Démontrer les formules trigonométriques de duplication et de multiplication

📐

Résoudre les équations de la forme zⁿ = a

💡 Exemple : Calcul de (1 + i)^10

1 + i = √2(cos(π/4) + i sin(π/4))

(1 + i)^10 = (√2)^10 × [cos(10×π/4) + i sin(10×π/4)]

= 2^5 × [cos(5π/2) + i sin(5π/2)]

= 32 × [cos(π/2 + 2π) + i sin(π/2 + 2π)]

= 32 × [0 + i×1] = 32i

💡 Exemple : Calcul de (√3 - i)^6

√3 - i a pour module 2 et argument -π/6

(√3 - i)^6 = 2^6 × [cos(-6×π/6) + i sin(-6×π/6)]

= 64 × [cos(-π) + i sin(-π)]

= 64 × [-1 + i×0] = -64

8.3 Applications aux formules trigonométriques

La formule de Moivre permet de démontrer les formules de :

📐

Duplication : cos(2θ) = cos²θ - sin²θ, sin(2θ) = 2sinθ·cosθ

📐

Triplication : cos(3θ) = 4cos³θ - 3cosθ

📐

Linéarisation : exprimer cosⁿθ et sinⁿθ en fonction de cos(kθ) et sin(kθ)

💡 Linéarisation de cos²θ

cos²θ = [(e^(iθ) + e^(-iθ))/2]²

= [e^(2iθ) + 2 + e^(-2iθ)]/4

= [2cos(2θ) + 2]/4

= (1 + cos(2θ))/2

📕 9. Équations dans ℂ

9.1 Équations du premier degré

Les équations de la forme az + b = 0 avec a, b ∈ ℂ et a ≠ 0 ont toujours une solution unique :

Solution d'une équation du 1er degré
z = -b/a

💡 Exemple : 2z + 3 - i = 0

2z = -3 + i

z = (-3 + i)/2 = -3/2 + i/2

9.2 Équations du second degré

Soit l'équation az² + bz + c = 0 avec a, b, c ∈ ℂ et a ≠ 0. Le discriminant est Δ = b² - 4ac.

📌

Si Δ ≠ 0 : deux solutions distinctes
z₁ = (-b + √Δ) / 2a et z₂ = (-b - √Δ) / 2a

📌

Si Δ = 0 : une solution double
z₀ = -b / 2a

💡 Exemple : z² + 2z + 5 = 0

Δ = 4 - 20 = -16

√Δ = √(-16) = 4i

z₁ = (-2 + 4i) / 2 = -1 + 2i

z₂ = (-2 - 4i) / 2 = -1 - 2i

9.3 Cas des coefficients réels

Quand a, b, c sont réels et Δ < 0, les deux solutions sont complexes conjuguées :

Solutions complexes conjuguées
z₁ = (-b + i√|Δ|) / 2a et z₂ = (-b - i√|Δ|) / 2a = z̄₁

C'est une propriété fondamentale : les coefficients réels impliquent des racines conjuguées.

⛔ Théorème fondamental à retenir

Si a, b, c ∈ ℝ et Δ < 0, alors les solutions de ax² + bx + c = 0 sont deux nombres complexes conjugués.

Cette propriété permet de vérifier rapidement ses calculs !

9.4 Équations du type zⁿ = a

Pour résoudre zⁿ = a où a ∈ ℂ, on écrit a sous forme trigonométrique a = r(cos θ + i sin θ), alors :

Racines n-ièmes
z_k = r^(1/n) × [cos((θ + 2kπ)/n) + i sin((θ + 2kπ)/n)]
pour k = 0, 1, 2, ..., n-1

L'équation admet n solutions distinctes, appelées racines n-ièmes de a, réparties régulièrement sur un cercle de rayon r^(1/n).

💡 Exemple : z³ = 8

8 = 8(cos 0 + i sin 0)

r = 8, θ = 0, n = 3

r^(1/3) = 2

z₀ = 2(cos 0 + i sin 0) = 2

z₁ = 2(cos(2π/3) + i sin(2π/3)) = -1 + i√3

z₂ = 2(cos(4π/3) + i sin(4π/3)) = -1 - i√3

💡 Exemple : z⁴ = 16

16 = 16(cos 0 + i sin 0)

r = 16, θ = 0, n = 4

r^(1/4) = 2

z₀ = 2(cos 0 + i sin 0) = 2

z₁ = 2(cos(π/2) + i sin(π/2)) = 2i

z₂ = 2(cos(π) + i sin(π)) = -2

z₃ = 2(cos(3π/2) + i sin(3π/2)) = -2i

💡 Exemple : z² = i

i = cos(π/2) + i sin(π/2)

r = 1, θ = π/2, n = 2

r^(1/2) = 1

z₀ = cos(π/4) + i sin(π/4) = (√2/2) + i(√2/2)

z₁ = cos(5π/4) + i sin(5π/4) = (-√2/2) - i(√2/2)

📒 10. Représentation géométrique des nombres complexes

10.1 Plan complexe (Plan d'Argand-Gauss)

Le plan complexe est un repère orthonormal direct (O, u, v) où chaque point M(x, y) représente le complexe z = x + iy.

📍

O est l'origine du repère

➡️

Axe horizontal (Ox) : appelé axe réel, noté Re(z)

⬆️

Axe vertical (Oy) : appelé axe imaginaire, noté Im(z)

📍

Le point M est appelé l'image de z, et z est l'affixe de M

10.2 Interprétation géométrique des notions fondamentales

📏

Module |z| : distance OM

🔄

Argument Arg(z) : angle orienté (u⃗, OM⃗)

🔄

Conjugué z̄ : symétrique de M par rapport à l'axe réel

📏

|z₁ - z₂| : distance entre les points M₁ et M₂

10.3 Interprétation de la multiplication par i

La multiplication par i correspond à une rotation de 90° (π/2 radians) dans le sens trigonométrique (inverse des aiguilles d'une montre) autour de l'origine.

💡 Exemples de rotation par i

• 1 × i = i (le point (1,0) tourne vers (0,1))

• i × i = -1 (le point (0,1) tourne vers (-1,0))

• z = 2 + 3i → z × i = -3 + 2i

10.4 Interprétation de la multiplication par un complexe

La multiplication par un complexe de module r et d'argument θ correspond à :

🔍

Une homothétie de rapport r (agrandissement si r > 1, réduction si r < 1)

🔄

Une rotation d'angle θ autour de l'origine

C'est ce qu'on appelle une similitude directe de centre O.

📘 11. Transformations dans le plan complexe

11.1 Translation

La translation de vecteur u⃗ d'affixe b correspond à :

Translation
z' = z + b (où b est un complexe)

Tous les points se déplacent de la même manière : le vecteur de translation est le même pour tous.

11.2 Homothétie

L'homothétie de centre O et de rapport k correspond à :

Homothétie
z' = k·z (où k ∈ ℝ⁺)
🔍

Si k > 1 : agrandissement

🔍

Si 0 < k < 1 : réduction

🔍

Si k < 0 : homothétie + rotation de π

11.3 Rotation

La rotation de centre O et d'angle θ correspond à :

Rotation
z' = e^(iθ)·z ou z' = (cos θ + i sin θ)·z

La rotation conserve les distances et les angles.

11.4 Similitude directe

La similitude directe (combinaison de rotation et homothétie) de centre O, de rapport k et d'angle θ :

Similitude directe
z' = k·e^(iθ)·z

La similitude directe la plus générale (de centre Ω) est :

Similitude de centre Ω
z' = a·z + b où a, b ∈ ℂ, a ≠ 0

Le centre Ω a pour affixe ω = b/(1 - a).

💡 Exemple : Rotation de 60° du point z = 1 + i

z' = e^(iπ/3) × (1 + i)

= (cos(π/3) + i sin(π/3)) × (1 + i)

= (1/2 + i√3/2) × (1 + i)

= 1/2 + i/2 + i√3/2 - √3/2

= (1/2 - √3/2) + i(1/2 + √3/2)

📗 12. Lieux géométriques dans le plan complexe

12.1 Cercle

L'ensemble des points M(z) tels que |z - a| = r est un cercle de centre A(a) et de rayon r.

Équation d'un cercle
|z - a| = r

En coordonnées cartésiennes, si a = α + iβ :

Équation cartésienne du cercle
(x - α)² + (y - β)² = r²

💡 Exemple : |z - 2i| = 3

C'est le cercle de centre A(0, 2) et de rayon 3.

En cartésien : x² + (y - 2)² = 9

12.2 Médiatrice

L'ensemble des points M(z) équidistants de A(a) et B(b) est la médiatrice du segment [AB].

Équation de la médiatrice
|z - a| = |z - b|

💡 Exemple : |z + 1| = |z - i|

C'est la médiatrice du segment reliant A(-1, 0) et B(0, 1).

En coordonnées : |z + 1|² = |z - i|²

(x + 1)² + y² = x² + (y - 1)²

x² + 2x + 1 + y² = x² + y² - 2y + 1

2x = -2y ⟹ x = -y

12.3 Droite

L'ensemble des points M(z) tels que Arg(z - a) = θ (ou Arg(z - a) = θ + π) est une droite passant par A(a) et faisant un angle θ avec l'axe réel.

Équation d'une droite (argument constant)
Arg(z - a) = θ

Plus généralement, une droite peut s'écrire sous la forme :

Équation d'une droite
αx + βy + γ = 0 avec α, β, γ ∈ ℝ

💡 Exemple : Re(z) = 2·Im(z)

Si z = x + iy, alors x = 2y

C'est la droite d'équation x - 2y = 0

12.4 Disques et demi-plans

🔴

Disque ouvert : |z - a| < r

🔴

Disque fermé : |z - a| ≤ r

📐

Demi-plan : Re(z) > k ou Im(z) < k

📐

Secteur angulaire : θ₁ < Arg(z - a) < θ₂

12.5 Applications aux triangles

📐

Triangle ABC équilatéral ⟺ z_A + j·z_B + j²·z_C = 0 (où j = e^(2iπ/3))

📐

Triangle ABC rectangle en A ⟺ (z_B - z_A) / (z_C - z_A) est un imaginaire pur

📐

Barycentre G ⟺ z_G = (α·z_A + β·z_B)/(α + β)

📘 1. Vocabulaire des probabilités

1.1. Épreuve aléatoire et univers

Une épreuve aléatoire (ou expérience aléatoire) est une expérience dont on connaît tous les résultats possibles, mais dont on ne peut pas prédire avec certitude quel résultat va se produire. L'ensemble de tous les résultats possibles d'une expérience aléatoire s'appelle l'univers, noté Ω (oméga).

📖 Définition : Univers d'une expérience aléatoire

L'univers Ω est l'ensemble de tous les résultats possibles (ou issues) d'une expérience aléatoire. Chaque résultat individuel est appelé événement élémentaire.

💡 Exemples d'épreuves aléatoires

  • Lancer d'un dé : Ω = {1, 2, 3, 4, 5, 6}
  • Lancer d'une pièce : Ω = {Pile, Face}
  • Tirage d'une carte d'un jeu de 52 cartes : Ω = {52 cartes}
  • Tirage de deux pièces : Ω = {PP, PF, FP, FF}

1.2. Événement

Un événement est un sous-ensemble de l'univers Ω. On dit qu'un événement est réalisé (ou survenu) quand le résultat de l'épreuve lui appartient.

💡 Exemples d'événements

On lance un dé équilibrée à 6 faces. Ω = {1, 2, 3, 4, 5, 6}

  • Événement A : "Obtenir un nombre pair" → A = {2, 4, 6}
  • Événement B : "Obtenir un 6" → B = {6}
  • Événement C : "Obtenir un nombre inférieur à 3" → C = {1, 2}
  • Événement D : "Obtenir un 7" → D = ∅ (événement impossible)

1.3. Types d'événements particuliers

✅

Événement certain : se réalise toujours. C'est l'univers Ω lui-même. Exemple : "Obtenir un nombre entre 1 et 6" en lançant un dé

❌

Événement impossible : ne se réalise jamais. C'est l'ensemble vide ∅. Exemple : "Obtenir un 7" en lançant un dé

🔀

Événements incompatibles (disjoints) : A ∩ B = ∅. Ils ne peuvent pas se produire simultanément. Exemple : A = {1, 3, 5} et B = {2, 4, 6}

↔️

Événement contraire de A : Ā = Ω \ A. Il se réalise quand A ne se réalise pas. Si A = {2, 4, 6}, alors Ā = {1, 3, 5}

🎯

Événement élémentaire : ne contient qu'un seul résultat. Exemple : {6} en lançant un dé

🔗

Événements compatibles : A ∩ B ≠ ∅. Ils peuvent se produire ensemble. Exemple : A = {2, 4, 6} et B = {4, 5, 6}

1.4. Opérations sur les événements

Opérations sur les événements

Réunion : A ∪ B = "A ou B" (au moins un des deux se réalise)

Intersection : A ∩ B = "A et B" (les deux se réalisent simultanément)

Complémentaire : Ā = Ω \ A = "non A"

Différence : A \ B = "A mais pas B"

💡 Opérations sur les événements - Exemple

On lance un dé. A = {2, 4, 6} (pair), B = {4, 5, 6} (≥ 4)

  • A ∪ B = {2, 4, 5, 6} (pair ou ≥ 4)
  • A ∩ B = {4, 6} (pair et ≥ 4)
  • Ā = {1, 3, 5} (impair)
  • A \ B = {2} (pair mais < 4)

📐 Propriété : Lois de Morgan

¬(A ∪ B) = ¬A ∩ ¬B (le contraire de "A ou B" est "non A et non B")

¬(A ∩ B) = ¬A ∪ ¬B (le contraire de "A et B" est "non A ou non B")

📗 2. Probabilités simples

2.1. Définition d'une probabilité

On appelle probabilité une fonction P qui associe à chaque événement un nombre réel de l'intervalle [0, 1], c'est-à-dire une mesure de la chance qu'un événement se réalise.

📖 Définition : Probabilité

Une probabilité P est une fonction définie sur l'ensemble des événements de Ω, à valeurs dans [0, 1], qui vérifie les axiomes suivants :

Axiomes des probabilités (Kolmogorov)

Axiome 1 : P(Ω) = 1 (la probabilité de l'événement certain est 1)

Axiome 2 : P(∅) = 0 (la probabilité de l'événement impossible est 0)

Axiome 3 : Si A ∩ B = ∅ (A et B incompatibles), alors P(A ∪ B) = P(A) + P(B)

2.2. Équiprobabilité

Lorsque tous les événements élémentaires ont la même probabilité, on dit que l'expérience est équilibrée ou uniforme.

Équiprobabilité

Si Ω contient n résultats également probables, alors pour tout événement A :

P(A) = Nombre de cas favorables / Nombre de cas possibles = |A| / |Ω|

2.3. Propriétés immédiates des probabilités

Conséquences directes des axiomes

1. P(Ā) = 1 - P(A)     (propriété du complémentaire)

2. Si A ⊂ B alors P(A) ≤ P(B)     (croissance)

3. Pour tout événement A : 0 ≤ P(A) ≤ 1

4. P(A ∪ B) = P(A) + P(B) - P(A ∩ B)     (formule d'inclusion-exclusion)

📝 Démonstration de P(Ā) = 1 - P(A)

A et Ā sont incompatibles (A ∩ Ā = ∅) et A ∪ Ā = Ω

D'après l'axiome 3 : P(A ∪ Ā) = P(A) + P(Ā)

Or P(A ∪ Ā) = P(Ω) = 1 (axiome 1)

Donc : 1 = P(A) + P(Ā) ⟹ P(Ā) = 1 - P(A)

📝 Démonstration de la formule d'inclusion-exclusion

P(A ∪ B) = P(A \ B) + P(A ∩ B) + P(B \ A)

Or A = (A \ B) ∪ (A ∩ B) et ces deux ensembles sont incompatibles

Donc P(A) = P(A \ B) + P(A ∩ B) ⟹ P(A \ B) = P(A) - P(A ∩ B)

De même, P(B) = P(B \ A) + P(A ∩ B) ⟹ P(B \ A) = P(B) - P(A ∩ B)

En substituant : P(A ∪ B) = [P(A) - P(A ∩ B)] + P(A ∩ B) + [P(B) - P(A ∩ B)]

P(A ∪ B) = P(A) + P(B) - P(A ∩ B) ✓

💡 Exemple : Lancer d'un dé équilibré

On lance un dé équilibré à 6 faces. Ω = {1, 2, 3, 4, 5, 6}, donc |Ω| = 6

Tous les événements élémentaires ont la probabilité 1/6 (équiprobabilité)

  • P({2}) = 1/6 (probabilité d'obtenir un 2)
  • P({pair}) = P({2,4,6}) = 3/6 = 1/2 (probabilité d'obtenir un nombre pair)
  • P({>3}) = P({4,5,6}) = 3/6 = 1/2 (probabilité d'obtenir un nombre supérieur à 3)
  • P({1,2,3,4}) = 4/6 = 2/3 (probabilité d'obtenir un nombre ≤ 4)

💡 Exemple : Urne avec boules

Une urne contient 5 boules rouges, 3 boules bleues et 2 boules vertes (10 boules en tout).

On tire une boule au hasard. Calculer les probabilités suivantes :

  • P(Rouge) = 5/10 = 1/2
  • P(Bleue) = 3/10
  • P(Verte) = 2/10 = 1/5
  • P(Rouge ou Bleue) = (5+3)/10 = 8/10 = 4/5
  • P(non Rouge) = 1 - 5/10 = 5/10 = 1/2

2.4. Cas de deux événements

Formule d'inclusion-exclusion pour deux événements

P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

Cas particulier : Si A et B sont incompatibles (A ∩ B = ∅), alors P(A ∪ B) = P(A) + P(B)

💡 Exemple avec deux événements

Dans une classe de 30 élèves, 18 pratiquent le football, 12 pratiquent le basketball, et 5 pratiquent les deux sports.

On choisit un élève au hasard.

  • P(Football) = 18/30 = 3/5
  • P(Basketball) = 12/30 = 2/5
  • P(les deux) = 5/30 = 1/6
  • P(Football ou Basketball) = 18/30 + 12/30 - 5/30 = 25/30 = 5/6

📙 3. Probabilités conditionnelles

3.1. Définition

La probabilité conditionnelle de A sachant B est la probabilité que l'événement A se réalise sachant que l'événement B s'est déjà réalisé. On la note P_B(A) ou P(A | B).

📖 Définition : Probabilité conditionnelle

Si P(B) > 0, alors la probabilité de A sachant B est :

P(A | B) = P(A ∩ B) / P(B)

De même, si P(A) > 0 :

P(B | A) = P(A ∩ B) / P(A)

Interprétation : P(A | B) représente la probabilité de A dans le "nouvel univers" où l'on sait que B s'est réalisé. On "réduit" l'univers à B.

3.2. Formule des probabilités composées

Probabilités composées

P(A ∩ B) = P(A) × P(B | A) = P(B) × P(A | B)

Cette formule permet de calculer la probabilité que deux événements se réalisent simultanément.

📝 Démonstration

Par définition de la probabilité conditionnelle :

P(A | B) = P(A ∩ B) / P(B) ⟹ P(A ∩ B) = P(B) × P(A | B)

De même : P(B | A) = P(A ∩ B) / P(A) ⟹ P(A ∩ B) = P(A) × P(B | A)

Donc : P(A ∩ B) = P(A) × P(B | A) = P(B) × P(A | B) ✓

💡 Exemple concret : Tirage sans remise

Une urne contient 3 boules rouges (R) et 2 boules bleues (B). On tire 2 boules sans remise.

Question : Quelle est la probabilité de tirer 2 boules rouges ?

Solution :

Soit A = "1ère boule est rouge" et B = "2ème boule est rouge"

On veut P(A ∩ B) = P(A) × P(B | A)

  • P(A) = 3/5 (3 rouges sur 5 boules)
  • P(B | A) = 2/4 = 1/2 (après avoir tiré une rouge, il reste 2 rouges sur 4 boules)
  • P(A ∩ B) = (3/5) × (1/2) = 3/10

💡 Exemple : Probabilité de tirer 2 as dans un jeu de cartes

Un jeu de 52 cartes contient 4 as. On tire 2 cartes sans remise.

P(1er As) = 4/52

P(2ème As | 1er As) = 3/51 (il reste 3 as sur 51 cartes)

P(2 As) = (4/52) × (3/51) = 12/2652 = 1/221 ≈ 0,0045

⚠️ Différence entre avec et sans remise

Avec remise : Les tirages sont indépendants. P(A ∩ B) = P(A) × P(B)

Sans remise : Les tirages ne sont PAS indépendants. Il faut utiliser les probabilités conditionnelles.

3.3. Arbre de probabilités (arbre de décision)

L'arbre de probabilités est un outil visuel très utile pour représenter les issues d'une expérience aléatoire en plusieurs étapes.

📋 Méthode : Construire un arbre de probabilités

  • Étape 1 : Identifier les étapes de l'expérience
  • Étape 2 : Pour chaque nœud, représenter les issues possibles avec leurs probabilités
  • Étape 3 : Chaque branche représente une probabilité conditionnelle
  • Étape 4 : La probabilité d'un chemin est le produit des probabilités des branches (règle du produit)
  • Étape 5 : Pour trouver P(A ∩ B), suivre le chemin correspondant et multiplier

💡 Arbre de probabilités : Exemple du lancer de deux pièces

On lance une pièce 2 fois de suite. Construisons l'arbre :

1er lancer → P (1/2) ou F (1/2)

├── P → 2ème lancer → P (1/2) ou F (1/2)

│  ├── P → PP : probabilité = 1/2 × 1/2 = 1/4

│  └── F → PF : probabilité = 1/2 × 1/2 = 1/4

└── F → 2ème lancer → P (1/2) ou F (1/2)

  ├── P → FP : probabilité = 1/2 × 1/2 = 1/4

  └── F → FF : probabilité = 1/2 × 1/2 = 1/4

P(au moins un Pile) = P(PP) + P(PF) + P(FP) = 1/4 + 1/4 + 1/4 = 3/4

📕 4. Indépendance d'événements

4.1. Définition

Deux événements A et B sont indépendants (ou stochastiquement indépendants) si la réalisation de l'un n'influence pas la probabilité de l'autre.

📖 Définition : Indépendance

A et B sont indépendants si et seulement si :

P(A ∩ B) = P(A) × P(B)

Cette définition est équivalente à :

  • Si P(B) > 0, alors P(A | B) = P(A)
  • Si P(A) > 0, alors P(B | A) = P(B)

En d'autres termes, "savoir que B s'est réalisé" ne change pas la probabilité de A.

4.2. Propriétés de l'indépendance

💡

Propriété 1 : Si A et B sont indépendants, alors Ā et B sont aussi indépendants

💡

Propriété 2 : Si A et B sont indépendants, alors A et B̄ sont aussi indépendants

💡

Propriété 3 : Si A et B sont indépendants, alors Ā et B̄ sont aussi indépendants

💡

Propriété 4 : Si P(A) = 0 ou P(A) = 1, alors A est indépendant de tout événement

⚠️

Attention : Ne pas confondre "incompatibles" et "indépendants" !

❗

A ∩ B = ∅ (incompatibles) ≠ P(A ∩ B) = P(A) × P(B) (indépendants)

💡 Exemple : Indépendance dans le lancer de deux pièces

On lance deux pièces équilibrées distinctes.

Soit A = "la 1ère pièce tombe sur Pile"

Soit B = "la 2ème pièce tombe sur Pile"

P(A) = 1/2, P(B) = 1/2

P(A ∩ B) = P(PP) = 1/4

Or P(A) × P(B) = 1/2 × 1/2 = 1/4

Donc P(A ∩ B) = P(A) × P(B) ⟹ A et B sont indépendants

Le résultat de la 1ère pièce n'influence pas celui de la 2ème.

💡 Exemple : Non-indépendance

Une urne contient 5 boules : 3 rouges (R) et 2 bleues (B). On tire 2 boules sans remise.

Soit A = "1ère boule est rouge"

Soit B = "2ème boule est bleue"

P(A) = 3/5 = 0,6

P(A | B) = P(1ère rouge | 2ème bleue) = ?

Si B est réalisé, il reste 1 bleue + 3 rouges = 4 boules

P(A | B) = 3/4 = 0,75 ≠ 0,6 = P(A)

⟹ A et B ne sont PAS indépendants

Savoir que la 2ème est bleue modifie la probabilité que la 1ère soit rouge.

⚠️ Confusion fréquente à éviter

Événements incompatibles (A ∩ B = ∅) :

Si A se réalise, B ne peut pas se réaliser. Ils s'excluent mutuellement.

Événements indépendants (P(A ∩ B) = P(A) × P(B)) :

La réalisation de l'un n'apporte aucune information sur l'autre.

Conséquence importante : Si A ∩ B = ∅ et P(A) > 0 et P(B) > 0, alors A et B ne sont PAS indépendants !

4.3. Indépendance de plusieurs événements

Trois événements A, B et C sont mutuellement indépendants si :

Indépendance mutuelle de trois événements

P(A ∩ B) = P(A) × P(B)

P(A ∩ C) = P(A) × P(C)

P(B ∩ C) = P(B) × P(C)

P(A ∩ B ∩ C) = P(A) × P(B) × P(C)

Les 4 conditions doivent être satisfaites !

📒 5. Formule des probabilités totales et formule de Bayes

5.1. Partition d'un univers

📖 Définition : Partition

Un ensemble d'événements (B₁, B₂, ..., Bₙ) forme une partition de Ω si :

  • Disjonction : Bᵢ ∩ Bⱼ = ∅ pour tout i ≠ j (ils sont deux à deux incompatibles)
  • Exhaustivité : B₁ ∪ B₂ ∪ ... ∪ Bₙ = Ω (leur réunion est l'univers)

💡 Exemples de partition

Exemple 1 : Pour un lancer de dé, les événements :

A = {1, 2}, B = {3, 4}, C = {5, 6} forment une partition de Ω

Exemple 2 : Pour un tirage de carte :

Cœurs = {13 cartes}, Piques = {13 cartes}, Carreaux = {13 cartes}, Trèfles = {13 cartes}

Ces 4 ensembles forment une partition du jeu de 52 cartes

5.2. Formule des probabilités totales

📐 Théorème : Probabilités totales

Soit (B₁, B₂, ..., Bₙ) une partition de Ω avec P(Bᵢ) > 0 pour tout i.

Alors pour tout événement A :

P(A) = Σ P(Bᵢ) × P(A | Bᵢ)

La somme est étendue à tous les événements de la partition.

📝 Démonstration

Les Bᵢ forment une partition de Ω, donc :

A = A ∩ Ω = A ∩ (B₁ ∪ B₂ ∪ ... ∪ Bₙ) = (A ∩ B₁) ∪ (A ∩ B₂) ∪ ... ∪ (A ∩ Bₙ)

Les (A ∩ Bᵢ) sont deux à deux incompatibles (car les Bᵢ le sont)

Donc : P(A) = Σ P(A ∩ Bᵢ) = Σ P(Bᵢ) × P(A | Bᵢ) (formule des probabilités composées)

✓ Démonstration complète

💡 Exemple : Usine de production

Une usine produit des pièces sur 3 machines :

  • Machine A : produit 50% des pièces, taux de défaut = 2%
  • Machine B : produit 30% des pièces, taux de défaut = 5%
  • Machine C : produit 20% des pièces, taux de défaut = 1%

Question : Quelle est la probabilité qu'une pièce prise au hasard soit défectueuse ?

Solution :

Soit D = "la pièce est défectueuse"

P(D) = P(A) × P(D | A) + P(B) × P(D | B) + P(C) × P(D | C)

P(D) = 0,5 × 0,02 + 0,3 × 0,05 + 0,2 × 0,01

P(D) = 0,01 + 0,015 + 0,002 = 0,027 = 2,7%

5.3. Formule de Bayes

📐 Théorème de Bayes

Soit (B₁, B₂, ..., Bₙ) une partition de Ω avec P(Bᵢ) > 0, et soit A un événement avec P(A) > 0.

Alors pour tout j ∈ {1, 2, ..., n} :

P(Bⱼ | A) = [P(Bⱼ) × P(A | Bⱼ)] / [Σ P(Bᵢ) × P(A | Bᵢ)]

P(Bⱼ | A) = P(Bⱼ) × P(A | Bⱼ) / P(A)

📝 Démonstration

Par définition de la probabilité conditionnelle :

P(Bⱼ | A) = P(Bⱼ ∩ A) / P(A)

Or P(Bⱼ ∩ A) = P(Bⱼ) × P(A | Bⱼ) (probabilités composées)

Et P(A) = Σ P(Bᵢ) × P(A | Bᵢ) (probabilités totales)

Donc : P(Bⱼ | A) = P(Bⱼ) × P(A | Bⱼ) / Σ P(Bᵢ) × P(A | Bᵢ) ✓

💡 Exemple : Application de la formule de Bayes

Reprenons l'exemple de l'usine. On choisit une pièce au hasard et elle est défectueuse.

Question : Quelle est la probabilité qu'elle ait été produite par la machine B ?

Solution :

P(B | D) = [P(B) × P(D | B)] / P(D)

P(B | D) = [0,3 × 0,05] / 0,027 = 0,015 / 0,027 ≈ 0,556 ≈ 55,6%

Malgré que la machine B ne produise que 30% des pièces, elle est responsable de 55,6% des pièces défectueuses car elle a un taux de défaut plus élevé.

⚠️ Interprétation de la formule de Bayes

P(Bⱼ) : probabilité a priori (avant d'observer A)

P(Bⱼ | A) : probabilité a posteriori (après avoir observé A)

La formule de Bayes permet de "mettre à jour" nos connaissances après une observation.

📋 Méthode : Quand utiliser chaque formule

  • Probabilités composées : P(A ∩ B) quand on connaît P(A) et P(B | A) (ou l'inverse)
  • Probabilités totales : P(A) quand on connaît les P(Bᵢ) et les P(A | Bᵢ) pour une partition (Bᵢ)
  • Formule de Bayes : P(Bⱼ | A) quand on connaît P(Bⱼ), P(A | Bⱼ) et P(A) (ou qu'on peut le calculer par probabilités totales)

📓 6. Variable aléatoire et espérance

6.1. Définition d'une variable aléatoire

Une variable aléatoire X est une fonction qui associe à chaque résultat de l'univers Ω un nombre réel. Elle permet de "quantifier" les résultats d'une expérience aléatoire.

📖 Définition : Variable aléatoire

Une variable aléatoire X est une fonction définie sur Ω, à valeurs dans ℝ :

X : Ω → ℝ

Pour chaque ω ∈ Ω, X(ω) est la valeur prise par X quand l'issue ω se réalise.

💡 Exemples de variables aléatoires

Exemple 1 : On lance un dé. X = "le résultat obtenu".

X peut prendre les valeurs 1, 2, 3, 4, 5, 6

Exemple 2 : On lance 3 pièces. X = "le nombre de Pile obtenus".

X peut prendre les valeurs 0, 1, 2, 3

Exemple 3 : Un joueur lance un dé. S'il fait 6, il gagne 5€. Sinon, il perd 1€.

X = "gain du joueur" peut prendre les valeurs 5 (avec probabilité 1/6) et -1 (avec probabilité 5/6)

6.2. Loi de probabilité d'une variable aléatoire

La loi de probabilité (ou distribution) d'une variable aléatoire X est la liste de toutes les valeurs xᵢ que peut prendre X, avec leurs probabilités respectives P(X = xᵢ).

Loi de probabilité

Pour chaque valeur xᵢ prise par X, on a :

P(X = xᵢ) = pᵢ où Σ pᵢ = 1

On peut présenter cette loi sous forme de tableau :

| xᵢ | x₁ | x₂ | ... | xₙ |

| P(X = xᵢ) | p₁ | p₂ | ... | pₙ |

💡 Exemple : Loi de probabilité d'un dé truqué

Un dé est truqué de telle sorte que :

P(1) = P(2) = P(3) = 1/12

P(4) = P(5) = P(6) = 1/4

La loi de X (résultat du lancer) est :

| xᵢ | 1 | 2 | 3 | 4 | 5 | 6 |

| P(X = xᵢ) | 1/12 | 1/12 | 1/12 | 1/4 | 1/4 | 1/4 |

Vérification : 3 × (1/12) + 3 × (1/4) = 3/12 + 9/12 = 12/12 = 1 ✓

6.3. Espérance mathématique

L'espérance mathématique d'une variable aléatoire X représente la valeur moyenne que X prendrait si on répétait l'expérience un très grand nombre de fois. C'est la "moyenne pondérée" des valeurs possibles.

📖 Définition : Espérance

L'espérance de X, notée E(X), est définie par :

E(X) = Σ xᵢ × P(X = xᵢ)

La somme est étendue à toutes les valeurs possibles xᵢ de X.

💡 Exemple : Espérance d'un jeu de dés

Un joueur lance un dé équilibrée. S'il fait 6, il gagne 5€. Sinon, il perd 1€.

X = "gain du joueur"

X peut prendre les valeurs :

  • x₁ = 5 avec probabilité P(X = 5) = 1/6
  • x₂ = -1 avec probabilité P(X = -1) = 5/6

E(X) = 5 × (1/6) + (-1) × (5/6) = (5 - 5)/6 = 0€

Interprétation : En moyenne, le joueur ne gagne ni ne perd. Le jeu est dit "équitable".

Propriétés de l'espérance

1. E(c) = c, où c est une constante

2. E(aX + b) = a × E(X) + b, où a et b sont des constantes

3. E(X + Y) = E(X) + E(Y) (linéarité)

4. Si X et Y sont indépendantes : E(X × Y) = E(X) × E(Y)

6.4. Interprétation concrète de l'espérance

Si on répète n fois l'expérience (avec n très grand), la moyenne des valeurs observées de X se rapproche de E(X).

En notant x̄ₙ = (x₁ + x₂ + ... + xₙ)/n la moyenne observée, on a :

lim(n→∞) x̄ₙ = E(X)

💡 Exemple : Lancers de dés et convergence

Si on lance un dé équilibré 1000 fois et qu'on calcule la moyenne des résultats :

E(X) = (1+2+3+4+5+6)/6 = 21/6 = 3,5

La moyenne observée sera proche de 3,5 (plus n est grand, plus c'est proche).

📔 7. Variance et écart-type

7.1. Définition de la variance

La variance d'une variable aléatoire X mesure la dispersion des valeurs de X autour de l'espérance. Plus la variance est grande, plus les valeurs sont "éloignées" de la moyenne.

📖 Définition : Variance

La variance de X, notée V(X) ou Var(X), est définie par :

V(X) = E[(X - E(X))²]

C'est l'espérance du carré de l'écart entre X et son espérance.

Formule de calcul de la variance

V(X) = Σ (xᵢ - E(X))² × P(X = xᵢ)

Formule simplifiée (plus pratique) :

V(X) = E(X²) - [E(X)]²

où E(X²) = Σ xᵢ² × P(X = xᵢ)

📝 Démonstration de V(X) = E(X²) - [E(X)]²

V(X) = E[(X - E(X))²]

= E[X² - 2X·E(X) + E(X)²]

= E(X²) - 2E(X)·E(X) + E(X)²·E(1) (par linéarité)

= E(X²) - 2E(X)² + E(X)²

= E(X²) - E(X)² ✓

7.2. Définition de l'écart-type

📖 Définition : Écart-type

L'écart-type de X, noté σ(X), est la racine carrée de la variance :

σ(X) = √V(X)

L'écart-type a la même unité que X (contrairement à la variance).

💡 Exemple : Variance et écart-type d'un jeu

Reprenons le jeu : gain de 5€ avec probabilité 1/6, perte de 1€ avec probabilité 5/6.

Calcul de E(X) :

E(X) = 5 × (1/6) + (-1) × (5/6) = 0€

Calcul de E(X²) :

E(X²) = 5² × (1/6) + (-1)² × (5/6) = 25/6 + 5/6 = 30/6 = 5

Calcul de V(X) :

V(X) = E(X²) - [E(X)]² = 5 - 0² = 5

Calcul de σ(X) :

σ(X) = √5 ≈ 2,24 €

💡 Exemple comparatif : Deux jeux

Jeu A : Gagnez 100€ avec probabilité 1/2, perdez 100€ avec probabilité 1/2

E(A) = 0, V(A) = 100² × 1/2 + 100² × 1/2 - 0² = 10 000

σ(A) = 100€

Jeu B : Gagnez 10€ avec probabilité 1/2, perdez 10€ avec probabilité 1/2

E(B) = 0, V(B) = 10² × 1/2 + 10² × 1/2 = 100

σ(B) = 10€

Les deux jeux ont la même espérance (0), mais le Jeu A est plus "risqué" (plus grande variance).

7.3. Propriétés de la variance

Propriétés de la variance

1. V(c) = 0, où c est une constante

2. V(aX + b) = a² × V(X), où a et b sont des constantes

3. V(X + Y) = V(X) + V(Y) + 2 Cov(X, Y)

4. Si X et Y sont indépendantes : V(X + Y) = V(X) + V(Y)

📊

E(X) : valeur moyenne de X (tendance centrale)

📏

V(X) : dispersion des valeurs autour de E(X)

📐

σ(X) : écart-type (même unité que X)

⚠️

Rappel : V(X) est toujours positive ou nulle

📕 8. La loi binomiale

8.1. Épreuve de Bernoulli

Une épreuve de Bernoulli est une expérience aléatoire qui ne possède que deux issues :

  • Succès (S) avec probabilité p
  • Échec (E) avec probabilité 1 - p

💡 Exemples d'épreuves de Bernoulli

  • Lancer une pièce : Pile (succès avec p = 1/2), Face (échec)
  • Tirer une boule rouge d'une urne : Succès avec p = nombre de rouges / total
  • Répondre au hasard à une question : Bonne réponse (succès) ou mauvaise (échec)

8.2. Schéma de Bernoulli

Un schéma de Bernoulli est la répétition de n épreuves de Bernoulli identiques et indépendantes.

📖 Définition : Schéma de Bernoulli

On réalise n fois, de manière indépendante, une épreuve de Bernoulli avec :

  • Probabilité de succès : p
  • Probabilité d'échec : q = 1 - p

La variable X = "nombre de succès" peut prendre les valeurs 0, 1, 2, ..., n

8.3. Loi binomiale

📖 Définition : Loi binomiale

On dit que X suit la loi binomiale de paramètres n et p, et on note :

X ~ B(n, p)

La probabilité d'obtenir exactement k succès est :

P(X = k) = C(n, k) × pᵏ × (1-p)^(n-k)

pour k = 0, 1, 2, ..., n

8.4. Coefficient binomial

C(n, k) est le coefficient binomial ("k parmi n"). Il représente le nombre de façons de choisir k succès parmi n épreuves.

Coefficient binomial

C(n, k) = n! / (k! × (n-k)!)

où n! = n × (n-1) × ... × 2 × 1 (factorielle)

Propriétés :

C(n, 0) = C(n, n) = 1

C(n, 1) = C(n, n-1) = n

C(n, k) = C(n, n-k)

Relation de Pascal : C(n, k) = C(n-1, k-1) + C(n-1, k)

💡 Coefficients binomiaux - Calcul

C(5, 2) = 5! / (2! × 3!) = 120 / (2 × 6) = 120 / 12 = 10

C(5, 3) = 5! / (3! × 2!) = 120 / (6 × 2) = 120 / 12 = 10

C(10, 0) = 1

C(10, 1) = 10

C(10, 5) = 252

8.5. Formule de la loi binomiale

Loi binomiale B(n, p)

P(X = k) = C(n, k) × pᵏ × (1-p)^(n-k)

pour k = 0, 1, ..., n

Démonstration intuitive :

Pour obtenir exactement k succès parmi n épreuves :

① Choisir les k épreuves qui seront des succès → C(n, k) possibilités

② Probabilité que ces k épreuves soient des succès → pᵏ

③ Probabilité que les (n-k) autres soient des échecs → (1-p)^(n-k)

Par la règle du produit : C(n, k) × pᵏ × (1-p)^(n-k)

💡 Exemple : Lancers de pièces

On lance 5 fois une pièce équilibrée. X = "nombre de Pile". X ~ B(5, 0,5)

Question 1 : P(X = 3) ?

P(X = 3) = C(5, 3) × (0,5)³ × (0,5)² = 10 × 1/8 × 1/4 = 10/32 = 5/16 = 0,3125

Question 2 : P(X = 0) ?

P(X = 0) = C(5, 0) × (0,5)⁰ × (0,5)⁵ = 1 × 1 × 1/32 = 1/32 ≈ 0,031

Question 3 : P(X = 5) ?

P(X = 5) = C(5, 5) × (0,5)⁵ × (0,5)⁰ = 1 × 1/32 × 1 = 1/32 ≈ 0,031

💡 Exemple : Questions de quiz

Un QCM contient 10 questions. Chaque question a 4 réponses dont une seule est correcte.

Un élève répond au hasard à toutes les questions. X = "nombre de bonnes réponses".

X ~ B(10, 1/4) car p = 1/4 pour chaque question

Question : P(X ≥ 5) ?

P(X ≥ 5) = 1 - P(X ≤ 4) = 1 - [P(0) + P(1) + P(2) + P(3) + P(4)]

P(X = k) = C(10, k) × (1/4)ᵏ × (3/4)^(10-k)

P(X = 0) = (3/4)^10 ≈ 0,056

P(X = 1) = 10 × (1/4) × (3/4)^9 ≈ 0,188

P(X = 2) = 45 × (1/4)² × (3/4)^8 ≈ 0,282

P(X = 3) = 120 × (1/4)³ × (3/4)^7 ≈ 0,250

P(X = 4) = 210 × (1/4)⁴ × (3/4)^6 ≈ 0,146

P(X ≥ 5) = 1 - (0,056 + 0,188 + 0,282 + 0,250 + 0,146) ≈ 0,078

8.6. Espérance et variance de la loi binomiale

Paramètres de la loi binomiale

Si X ~ B(n, p), alors :

E(X) = n × p     (espérance)

V(X) = n × p × (1-p)     (variance)

σ(X) = √(n × p × (1-p))     (écart-type)

💡 Exemple : Application des formules

On lance 5 fois une pièce équilibrée. X ~ B(5, 0,5)

E(X) = 5 × 0,5 = 2,5 (en moyenne 2,5 piles sur 5 lancers)

V(X) = 5 × 0,5 × 0,5 = 1,25

σ(X) = √1,25 ≈ 1,12

💡 Exemple : Quiz QCM

10 questions, 4 réponses par question, réponses au hasard. X ~ B(10, 1/4)

E(X) = 10 × 1/4 = 2,5 (en moyenne 2,5 bonnes réponses)

V(X) = 10 × 1/4 × 3/4 = 30/16 = 1,875

σ(X) = √1,875 ≈ 1,37

8.7. Conditions d'application de la loi binomiale

⚠️ Conditions d'application (impératif !)

La loi binomiale s'applique uniquement si les 3 conditions suivantes sont satisfaites :

① Épreuve à deux issues : L'expérience doit avoir exactement 2 issues (succès / échec)

② Indépendance : Les épreuves doivent être indépendantes les unes des autres

③ Paramètre constant : La probabilité de succès p doit être la même pour toutes les épreuves

✅

Répétitions indépendantes : Lancer plusieurs fois une pièce ou un dé (résultats indépendants)

❌

Tirages sans remise : Ne suit PAS une loi binomiale en général (sauf si la population est très grande)

⚠️

Répétitions dépendantes : Ne suit PAS une loi binomiale

💡

Approximation : Si n ≥ 30 et np ≥ 5 et n(1-p) ≥ 5, on peut approcher B(n, p) par une loi normale N(np, np(1-p))

8.8. Représentation graphique de la loi binomiale

La loi binomiale peut être représentée par un diagramme en bâtons. La forme dépend des paramètres n et p :

  • Si p = 0,5 : distribution symétrique
  • Si p < 0,5 : distribution asymétrique vers la droite
  • Si p > 0,5 : distribution asymétrique vers la gauche
  • Si n augmente : distribution se rapproche d'une courbe en cloche (loi normale)

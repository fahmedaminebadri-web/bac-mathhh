📘 1. Dérivabilité et nombre dérivé

La dérivation permet d'étudier le taux de variation instantané d'une fonction, c'est-à-dire sa vitesse de changement en chaque point.

🔹 Nombre dérivé en un point

📐 Définition

Soit f une fonction définie sur un intervalle I et a ∈ I.

f est dérivable en a si et seulement si la limite suivante existe (et est finie) :

f'(a) = limh→0 [f(a + h) - f(a)]/h

Cette limite s'appelle le nombre dérivé de f en a.

Notation alternative
f'(a) = limx→a [f(x) - f(a)]/(x - a)

🔹 Interprétation géométrique

📐 Tangente à une courbe

Si f est dérivable en a, alors la courbe Cf admet une tangente au point A(a, f(a)).

Équation de la tangente :

y = f'(a)(x - a) + f(a)

💡 Exemple : Tangente à une parabole

f(x) = x² au point a = 3

f'(x) = 2x donc f'(3) = 6

f(3) = 9

Équation de la tangente :

y = 6(x - 3) + 9 = 6x - 18 + 9

y = 6x - 9

🔹 Dérivabilité et continuité

📐 Théorème fondamental

Si f est dérivable en a, alors f est continue en a.

La réciproque est fausse ! Une fonction peut être continue sans être dérivable.

🔹 Dérivabilité à gauche et à droite

Dérivées latérales
f'+(a) = limh→0⁺ [f(a + h) - f(a)]/h
f'-(a) = limh→0⁻ [f(a + h) - f(a)]/h

📐 Condition de dérivabilité

f est dérivable en a  ⟺  f'+(a) = f'-(a) (et finie)

💡 Exemple : |x| en x = 0

f(x) = |x| est continue en 0 (f(0) = 0)

Mais :

f'+(0) = limh→0⁺ h/h = 1

f'-(0) = limh→0⁻ (-h)/h = -1

f'+(0) ≠ f'-(0) → f n'est pas dérivable en 0

La courbe admet un "point anguleux" en x = 0.

⚠️ Cas de non-dérivabilité

• Point anguleux : f'+(a) ≠ f'-(a) (ex: |x|)

• Tangente verticale : f'(a) = ±∞ (ex: √x en 0)

• Point de rebroussement : f'+ = f'- = ±∞ (ex: x1/3)

📗 2. Formules de dérivation

Ces formules sont les outils de base pour dériver n'importe quelle fonction.

🔹 Opérations algébriques

Fonction Dérivée
(f + g)' f' + g'
(f - g)' f' - g'
(k × f)' k × f' (k constante)
(f × g)' f'g + fg' (produit)
(f/g)' (f'g - fg')/g² (quotient)

💡 Exemple : Dérivée d'un produit

f(x) = x² × sin(x)

f'(x) = (x²)' × sin(x) + x² × (sin(x))'

f'(x) = 2x × sin(x) + x² × cos(x)

💡 Exemple : Dérivée d'un quotient

f(x) = x/(x + 1)

f'(x) = [(1)(x+1) - (x)(1)]/(x+1)²

f'(x) = (x + 1 - x)/(x+1)²

f'(x) = 1/(x+1)²

🔹 Dérivée d'une fonction composée

📐 Formule de composition (chaînage)

Si g est dérivable en u et f est dérivable en x, alors :

(f ∘ g)'(x) = f'(g(x)) × g'(x)

En notation plus simple : d(f(g(x)))/dx = f'(g(x)) × g'(x)

Cas particulier important
(uⁿ)' = n × u' × un-1   (n ∈ ℤ)

💡 Exemple : Fonction composée

f(x) = (3x² + 2)⁵

On pose u = 3x² + 2, donc u' = 6x

f'(x) = 5 × u' × u⁴

f'(x) = 5 × 6x × (3x² + 2)⁴

f'(x) = 30x(3x² + 2)⁴

💡 Exemple : sin(2x + 1)

f(x) = sin(2x + 1)

Composée : f(x) = g(h(x)) avec h(x) = 2x + 1, g(u) = sin(u)

h'(x) = 2, g'(u) = cos(u)

f'(x) = cos(2x + 1) × 2 = 2cos(2x + 1)

💡 Exemple : √(x² + 1)

f(x) = √(x² + 1) = (x² + 1)1/2

u = x² + 1, u' = 2x

f'(x) = (1/2) × u' × u-1/2

f'(x) = (1/2) × 2x × (x² + 1)-1/2

f'(x) = x/√(x² + 1)

🔹 Dérivée de l'inverse

Inverse d'une fonction
(1/f)' = -f'/f²   (si f ≠ 0)

💡 Exemple : Dérivée de x³ + 2√x - 5/x²

f(x) = x³ + 2x1/2 - 5x⁻²

f'(x) = 3x² + 2 × (1/2)x-1/2 - 5 × (-2)x⁻³

f'(x) = 3x² + 1/√x + 10/x³

💡 Exemple : Dérivée de ex²

f(x) = ex²

Composée : f'(x) = ex² × (x²)'

f'(x) = ex² × 2x

f'(x) = 2x × ex²

💡 Exemple : Dérivée de ln(3x + 2)

f(x) = ln(3x + 2)

Composée : f'(x) = (1/(3x + 2)) × (3x + 2)'

f'(x) = (1/(3x + 2)) × 3

f'(x) = 3/(3x + 2)

⚠️ Erreurs fréquentes

• (eˣ)' = eˣ (pas eˣ⁻¹ !)

• (ln(x))' = 1/x (pas 1/ln(x) !)

• (1/f)' = -f'/f² (pas f'/f !)

• Ne pas oublier le "× u'" dans les composées !

📕 4. Dérivées successives

On peut dériver une fonction plusieurs fois de suite pour obtenir les dérivées successives.

🔹 Définition

Dérivées successives
f' = dérivée première
f'' = (f')' = dérivée seconde
f''' = (f'')' = dérivée troisième
f(n) = dérivée n-ième

💡 Exemple : Polynôme

f(x) = 5x⁴ - 3x³ + 2x² - x + 7

f'(x) = 20x³ - 9x² + 4x - 1

f''(x) = 60x² - 18x + 4

f'''(x) = 120x - 18

f⁽⁴⁾(x) = 120

f⁽⁵⁾(x) = 0 (car dérivée d'ordre > degré d'un polynôme)

🔹 Formule de Leibniz

📐 Formule de Leibniz

Pour les dérivées successives d'un produit :

(uv)(n) = Σk=0n Cnk × u(k) × v(n-k)

💡 Exemple : Application de Leibniz

Calculer f⁽²⁾(x) où f(x) = x × eˣ

(uv)'' = u''v + 2u'v' + uv''

u = x, v = eˣ

u' = 1, u'' = 0

v' = eˣ, v'' = eˣ

f''(x) = 0 × eˣ + 2 × 1 × eˣ + x × eˣ

f''(x) = (2 + x)eˣ

🔹 Classe d'une fonction

📚

f ∈ C⁰ : f est continue

📚

f ∈ C¹ : f est dérivable avec f' continue

📚

f ∈ Cⁿ : f admet des dérivées continues jusqu'à l'ordre n

📚

f ∈ C∞ : f est indéfiniment dérivable (ex: eˣ, sin, cos)

📒 5. Applications de la dérivation

La dérivation a de nombreuses applications en analyse et en géométrie.

🔹 Sens de variation d'une fonction

📐 Théorème fondamental

Soit f dérivable sur un intervalle I :

• f est croissante sur I  ⟺  ∀x ∈ I, f'(x) ≥ 0

• f est décroissante sur I  ⟺  ∀x ∈ I, f'(x) ≤ 0

• f est constante sur I  ⟺  ∀x ∈ I, f'(x) = 0

⚠️ Attention

Si f'(x) > 0 en tout point → f est strictement croissante

Mais f peut être strictement croissante avec f'(x) = 0 en certains points (ex: x³)

💡 Exemple : Étudier les variations

f(x) = 2x³ - 3x² - 12x + 5

f'(x) = 6x² - 6x - 12 = 6(x² - x - 2)

f'(x) = 6(x - 2)(x + 1)

Tableau de signes :

x-∞-12+∞
f'(x)++0-0++
f(x)↗↗max↘min↗↗

🔹 Extremums locaux

📐 Condition nécessaire (Fermat)

Si f admet un extremum local en a et si f est dérivable en a, alors :

f'(a) = 0

📐 Condition suffisante

Si f'(a) = 0 et si f' change de signe en a :

• f' passe de + à - → maximum local

• f' passe de - à + → minimum local

💡 Exemple : Trouver les extremums

f(x) = x³ - 3x

f'(x) = 3x² - 3 = 3(x² - 1)

f'(x) = 0 ⟺ x = -1 ou x = 1

• En x = -1 : f' passe de + à - → maximum (f(-1) = 2)

• En x = 1 : f' passe de - à + → minimum (f(1) = -2)

🔹 Dérivée et convexité

📐 Convexité

• f est convexe sur I ⟺ f'' ≥ 0 sur I

• f est concave sur I ⟺ f'' ≤ 0 sur I

📐 Point d'inflexion

Un point d'inflexion est un point où la convexité change.

f admet un point d'inflexion en a ⟺ f''(a) = 0 et f'' change de signe en a

🔹 Formules de Taylor

📐 Formule de Taylor-Young à l'ordre n

Si f ∈ Cⁿ au voisinage de a :

f(x) = f(a) + f'(a)(x-a) + f''(a)/2!(x-a)² + ... + f⁽ⁿ⁾(a)/n!(x-a)ⁿ + o((x-a)ⁿ)

💡 Exemple : Taylor-Young de eˣ en 0

eˣ = 1 + x + x²/2! + x³/3! + ... + xⁿ/n! + o(xⁿ)

À l'ordre 3 : eˣ ≈ 1 + x + x²/2 + x³/6

💡 Exemple : Taylor-Young de sin(x) en 0

sin(x) = x - x³/3! + x⁵/5! - x⁷/7! + ...

À l'ordre 5 : sin(x) ≈ x - x³/6 + x⁵/120

🔹 Application : Équation de la normale

📐 Normale à une courbe

La normale en A(a, f(a)) est perpendiculaire à la tangente :

(x - a) + f'(a)(y - f(a)) = 0

Ou encore : f'(a)(x - a) + (y - f(a)) = 0

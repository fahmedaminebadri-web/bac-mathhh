📘 1. Primitives d'une fonction

Définition

Soit f une fonction définie sur un intervalle I. Une primitive de f sur I est une fonction F définie sur I telle que :

Définition de primitive
F est une primitive de f sur I  ⇔  F'(x) = f(x)   ∀x ∈ I

Propriété fondamentale

Théorème

Si F est une primitive de f sur I, alors toutes les primitives de f sur I sont de la forme :

F(x) + C    où C est une constante réelle arbitraire

Notation : L'ensemble de toutes les primitives de f est : {F + C | C ∈ ℝ}

💡 Exemples fondamentaux

• f(x) = 3x² → Primitive : F(x) = x³ + C ✓ (F' = 3x²)

• f(x) = cos(x) → Primitive : F(x) = sin(x) + C ✓

• f(x) = eˣ → Primitive : F(x) = eˣ + C ✓ (eˣ est sa propre primitive)

• f(x) = 1/x → Primitive : F(x) = ln|x| + C ✓

⚠️ Primitives vs dérivées

La dérivation est une opération linéaire et facile.

L'intégration (recherche de primitives) est plus difficile et ne suit pas de méthode systématique dans le cas général.

Il est essentiel de connaître par cœur les primitives des fonctions usuelles.

Primitives des fonctions usuelles

Table de primitives — Fonctions élémentaires
∫ xⁿ dx = xⁿ⁺¹/(n+1) + C    pour n ≠ −1
∫ 1/x dx = ln|x| + C
∫ eˣ dx = eˣ + C
∫ aˣ dx = aˣ/ln(a) + C    (a > 0, a ≠ 1)
∫ sin(x) dx = −cos(x) + C
∫ cos(x) dx = sin(x) + C
∫ 1/cos²(x) dx = tan(x) + C
∫ 1/sin²(x) dx = −cot(x) + C
∫ 1/√x dx = 2√x + C

Primitives de fonctions composées

Formules avec u(x)
Si u est une fonction dérivable sur I :

∫ u'(x)·uⁿ(x) dx = uⁿ⁺¹(x)/(n+1) + C    pour n ≠ −1
∫ u'(x)/u(x) dx = ln|u(x)| + C
∫ u'(x)·e^(u(x)) dx = e^(u(x)) + C
∫ u'(x)·sin(u(x)) dx = −cos(u(x)) + C
∫ u'(x)·cos(u(x)) dx = sin(u(x)) + C

💡 Exemple 1 — Primitive simple

Primitive de f(x) = 3x²

F(x) = x³ + C ✓

Vérification : (x³ + C)' = 3x²

💡 Exemple 2 — Composition : ∫ 2x·e^(x²) dx

On reconnaît : u'(x) = 2x, u(x) = x²

∫ u'(x)·e^(u(x)) dx = e^(u(x)) + C

= e^(x²) + C

Vérification : (e^(x²))' = e^(x²)·2x = 2x·e^(x²) ✓

💡 Exemple 3 — Composition : ∫ (3x² + 1/x + 2eˣ) dx

= ∫ 3x² dx + ∫ 1/x dx + ∫ 2eˣ dx

= x³ + ln|x| + 2eˣ + C

💡 Exemple 4 — Composition : ∫ x·(x²+1)⁵ dx

u = x²+1 → u' = 2x → x dx = du/2

∫ x·u⁵ dx = ∫ (du/2)·u⁵ = (1/2)·∫ u⁵ du

= (1/2)·u⁶/6 + C = u⁶/12 + C

= (x²+1)⁶/12 + C

💡 Exemple 5 — Tangente : ∫ tan(x) dx

∫ tan(x) dx = ∫ sin(x)/cos(x) dx

cos(x) = u → −sin(x)dx = du → sin(x)dx = −du

= ∫ (−du)/u = −ln|u| + C = −ln|cos(x)| + C

= ln|1/cos(x)| + C = ln|tan(x)| + C/2... en fait : ∫ tan(x) = −ln|cos(x)| + C ✓

📗 2. L'intégrale définie

Approche géométrique — Introduction

Si f est positive sur [a, b], l'intégrale ∫ₐᵇ f(x)dx représente l'aire de la région limitée par :

📈

La courbe de f

↔

L'axe des abscisses (Ox)

↕

Les verticales x = a et x = b

Définition formelle

Intégrale définie
Soit f une fonction admettant une primitive F sur [a, b]. On appelle intégrale définie de a à b le nombre :

∫ₐᵇ f(x)dx = [F(x)]ₐᵇ = F(b) − F(a)

Le résultat est un nombre réel (pas une fonction).

Notation : [F(x)]ₐᵇ signifie F(b) − F(a).

⚠️ Ordre des bornes

∫ₐᵇ f(x)dx = F(b) − F(a)

∫ᵇₐ f(x)dx = F(a) − F(b) = −∫ₐᵇ f(x)dx

Quand on inverse les bornes, l'intégrale change de signe.

🔄

Changement de signe : ∫ₐᵇ = −∫ᵇₐ

📍

Borne supérieure = inférieure : ∫ₐₐ = 0

Signe de l'intégrale

📈

Si f ≥ 0 sur [a, b] (f positive) → ∫ₐᵇ f(x)dx ≥ 0

📉

Si f ≤ 0 sur [a, b] (f négative) → ∫ₐᵇ f(x)dx ≤ 0

📊

Si f change de signe → l'intégrale = somme des aires algébriques

💡 Exemple : ∫₀² x² dx

F(x) = x³/3

∫₀² x² dx = F(2) − F(0) = 8/3 − 0 = 8/3

Interprétation : aire sous la parabole y = x² entre 0 et 2 = 8/3 u.a.

💡 Exemple avec signe négatif : ∫₀² (−x²) dx

F(x) = −x³/3

∫₀² (−x²) dx = F(2) − F(0) = −8/3 − 0 = −8/3

Interprétation : l'aire (algébrique) est négative car la courbe est sous l'axe.

📙 3. Propriétés de l'intégrale

Propriétés de linéarité

Théorème 1 — Linéarité

∫ₐᵇ (f + g)(x)dx = ∫ₐᵇ f(x)dx + ∫ₐᵇ g(x)dx

∫ₐᵇ λ·f(x)dx = λ · ∫ₐᵇ f(x)dx   (λ ∈ ℝ)

Relation de Chasles

Théorème 2 — Relation de Chasles

Pour tous a, b, c ∈ ℝ :

∫ₐᵇ f(x)dx + ∫ᵇᶜ f(x)dx = ∫ₐᶜ f(x)dx

Peu importe l'ordre de a, b, c (les bornes peuvent être dans n'importe quel ordre).

💡 Exemple de Chasles

∫₀² x² dx + ∫₂⁵ x² dx = ∫₀⁵ x² dx

8/3 + [x³/3]₂⁵ = 8/3 + (125/3 − 8/3) = 125/3 ✓

Positivité et comparaison

Théorème 3 — Positivité

Si f ≥ g sur [a, b] avec a < b, alors :

∫ₐᵇ f(x)dx ≥ ∫ₐᵇ g(x)dx

En particulier : f ≥ 0 → ∫ₐᵇ f ≥ 0

Théorème 4 — Inégalité triangulaire

Pour tout a < b :

|∫ₐᵇ f(x)dx| ≤ ∫ₐᵇ |f(x)| dx

Valeur moyenne d'une fonction

Théorème de la moyenne
Si f est continue sur [a, b], alors il existe c ∈ [a, b] tel que :

∫ₐᵇ f(x)dx = (b − a) · f(c)

La valeur moyenne de f sur [a, b] est :
m = (1/(b−a)) · ∫ₐᵇ f(x)dx

💡 Exemple — Valeur moyenne

Valeur moyenne de f(x) = x² sur [0, 2] :

moyenne = (1/2) · ∫₀² x² dx = (1/2) · (8/3) = 4/3

Il existe c ∈ [0, 2] tel que f(c) = 4/3

c² = 4/3 → c = 2/√3 ≈ 1,155 ∈ [0, 2] ✓

📕 4. Formule fondamentale de l'analyse (Théorème de Barrow)

Théorème de Barrow

Théorème fondamental (Barrow, 1630–1704)
Si F est une primitive de f sur [a, b], alors :

∫ₐᵇ f(x)dx = F(b) − F(a)

Ce théorème établit le lien fondamental entre dérivation et intégration : ce sont des opérations inverses.

Fonction intégrale

Théorème fondamental de l'analyse
Si f est continue sur [a, b], la fonction G définie par :

G(x) = ∫ₐˣ f(t)dt

est dérivable sur [a, b] et :
G'(x) = f(x)    pour tout x ∈ [a, b]

G est une primitive de f. La dérivée de l'intégrale redonne la fonction.

💡 Théorème fondamental : d/dx [∫₀ˣ t² dt]

= d/dx [x³/3] = x² ✓

→ La dérivée de l'intégrale redonne la fonction de départ !

💡 Calcul d'intégrale par Barrow

∫₁ᵉ (1/x) dx

= [ln|x|]₁ᵉ = ln(e) − ln(1) = 1 − 0 = 1

💡 Intégrale avec paramètre

Calculer d/dx [∫₀ˣ e^(t²) dt]

Par le théorème fondamental : = e^(x²) ✓

Démonstration du théorème de Barrow

Soit F primitive de f. On veut montrer ∫ₐᵇ f = F(b) − F(a).

Considérer G(x) = ∫ₐˣ f(t)dt. Par le théorème fondamental, G'(x) = f(x).

Donc G est aussi une primitive de f.

Deux primitives diffèrent d'une constante : G(x) = F(x) + C.

G(a) = ∫ₐᵃ f = 0 = F(a) + C → C = −F(a)

G(b) = ∫ₐᵇ f = F(b) − F(a) ∎

📒 5. Calcul d'aires

Aire entre une courbe et l'axe des abscisses

Aire algébrique
Si f ≥ 0 sur [a, b] : A = ∫ₐᵇ f(x)dx
Si f ≤ 0 sur [a, b] : A = −∫ₐᵇ f(x)dx
Si f change de signe : on somme les aires positives et négatives sur chaque intervalle

Aire entre deux courbes

Théorème — Aire entre f et g
Si f ≥ g sur [a, b], l'aire de la région comprise entre les courbes est :

A = ∫ₐᵇ (f(x) − g(x)) dx

💡 Exemple 1 — Aire sous une courbe

Aire entre y = x², x = 0, x = 1 et l'axe Ox

f(x) = x² ≥ 0 sur [0, 1]

A = ∫₀¹ x² dx = [x³/3]₀¹ = 1/3 − 0 = 1/3 u.a.

💡 Exemple 2 — Aire entre deux courbes

Aire entre y = x et y = x² entre x = 0 et x = 1

Sur [0, 1] : x ≥ x²

A = ∫₀¹ (x − x²) dx

= [x²/2 − x³/3]₀¹ = (1/2 − 1/3) − 0 = (3−2)/6 = 1/6 u.a.

💡 Exemple 3 — sin(x) entre 0 et 2π

sin(x) ≥ 0 sur [0, π] : A₁ = ∫₀ᵠ sin(x)dx = [−cos(x)]₀ᵠ = −(−1) + 1 = 2

sin(x) ≤ 0 sur [π, 2π] : A₂ = −∫ᵠ²ᵠ sin(x)dx = −[−cos(x)]ᵠ²ᵠ = −(−1 − 1) = 2

Aire totale = 2 + 2 = 4 u.a.

⚠️ Aire = valeur absolue

L'aire géométrique est toujours positive :

Aire = ∫ₐᵇ |f(x)| dx

Méthode :

① Trouver les zéros de f sur [a, b]

② Séparer l'intégrale sur chaque sous-intervalle

③ Prendre la valeur absolue de chaque morceau et additionner

💡 Exemple : Aire de |x² − 1| sur [−2, 2]

x² − 1 = 0 → x = −1 et x = 1

• Sur [−2, −1] : x² ≥ 1 → |x²−1| = x²−1

• Sur [−1, 1] : x² ≤ 1 → |x²−1| = 1−x²

• Sur [1, 2] : x² ≥ 1 → |x²−1| = x²−1

A = ∫₋₂₋₁ (x²−1)dx + ∫₋₁¹ (1−x²)dx + ∫₁² (x²−1)dx

= [x³/3−x]₋₂₋₁ + [x−x³/3]₋₁¹ + [x³/3−x]₁²

= [(−1/3+1) − (−8/3+2)] + [(1−1/3) − (−1+1/3)] + [(8/3−2) − (1/3−1)]

= (2/3+2/3) + (2/3+2/3) + (2/3+2/3) = 6 × (2/3) = 4 u.a.

📓 6. Techniques d'intégration

Intégration par parties (IPP)

Formule d'intégration par parties
∫ u·v' dx = u·v − ∫ u'·v dx

Version définie :
∫ₐᵇ u·v' dx = [u·v]ₐᵇ − ∫ₐᵇ u'·v dx

Le choix de u et dv est crucial. Règle mnémotechnique : LIATE

Logarithmic, Inverse trig, Algebraic, Trigonometric, Exponential

→ On choisit u dans cet ordre de priorité.

💡 Exemple 1 — IPP classique : ∫ x·eˣ dx

Selon LIATE : x (algébrique) → u = x ; eˣ (exponentielle) → dv = eˣdx

du = dx ; v = eˣ

= x·eˣ − ∫ 1·eˣ dx

= x·eˣ − eˣ + C

= eˣ(x − 1) + C

💡 Exemple 2 — IPP itérée : ∫ x²·eˣ dx

IPP 1 : u = x², dv = eˣdx → du = 2x dx, v = eˣ

= x²eˣ − ∫ 2x·eˣ dx = x²eˣ − 2∫ x·eˣ dx

IPP 2 : ∫ x·eˣ dx = x·eˣ − eˣ (par IPP) = eˣ(x−1)

Résultat : x²eˣ − 2eˣ(x−1) = eˣ(x² − 2x + 2) + C

💡 Exemple 3 — IPP sur ln : ∫ ln(x) dx

∫ ln(x) dx = ∫ 1·ln(x) dx

u = ln(x), dv = dx → du = (1/x)dx, v = x

= x·ln(x) − ∫ x·(1/x) dx

= x·ln(x) − ∫ 1 dx

= x·ln(x) − x + C

💡 Exemple 4 — IPP sur arctan : ∫ arctan(x) dx

∫ arctan(x) dx = ∫ 1·arctan(x) dx

u = arctan(x), dv = dx → du = dx/(1+x²), v = x

= x·arctan(x) − ∫ x/(1+x²) dx

= x·arctan(x) − (1/2)ln(1+x²) + C

= x·arctan(x) − (1/2)ln(1+x²) + C

Changement de variable

Changement de variable
Pour calculer ∫ f(x) dx :

Poser u = g(x) → du = g'(x)dx
Réécrire l'intégrale en fonction de u.
Calculer ∫ f(u) du.
Remplacer u par g(x) à la fin.

💡 Exemple 1 — Changement simple : ∫ x·cos(x²) dx

Poser u = x² → du = 2x dx → x dx = du/2

= ∫ cos(u) · (du/2) = (1/2)·∫ cos(u) du

= (1/2)·sin(u) + C

= (1/2)·sin(x²) + C

💡 Exemple 2 — Racine : ∫ dx/(1+√x)

Poser u = √x → x = u² → dx = 2u du

= ∫ 2u/(1+u) du = 2∫ (u+1−1)/(u+1) du

= 2∫ (1 − 1/(u+1)) du

= 2(u − ln|u+1|) + C

= 2√x − 2ln(√x+1) + C

💡 Exemple 3 — Tangente : ∫ dx/(1+x²)

C'est une primitive classique : = arctan(x) + C

Ou par changement : u = arctan(x)...

📔 7. Primitives des fonctions usuelles composées — Résumé

Formules de primitives composées
∫ (ax+b)ⁿ dx = (ax+b)ⁿ⁺¹/[(n+1)a] + C    (n ≠ −1)
∫ dx/(ax+b) = (1/a)ln|ax+b| + C
∫ e^(ax+b) dx = (1/a)e^(ax+b) + C
∫ sin(ax+b) dx = −(1/a)cos(ax+b) + C
∫ cos(ax+b) dx = (1/a)sin(ax+b) + C
∫ dx/cos²(ax+b) = (1/a)tan(ax+b) + C
∫ dx/√(ax+b) = (2/a)√(ax+b) + C

💡 Récapitulatif d'exemples

• ∫ (3x+5)⁴ dx = (3x+5)⁵/15 + C

• ∫ dx/(2x−3) = (1/2)ln|2x−3| + C

• ∫ e^(−x) dx = −e^(−x) + C

• ∫ sin(2x+π/4) dx = −(1/2)cos(2x+π/4) + C

• ∫ dx/√(5x−2) = (2/5)√(5x−2) + C

Primitives rationnelles

📐

Si numérateur = dérivée du dénominateur → ln|dénominateur| + C

📐

Décomposer en éléments simples si degré(num) ≥ degré(déno)

💡 Exemple : ∫ (2x+1)/(x²+x+1) dx

On remarque : numérateur = dérivée de (x²+x+1)

= ln|x²+x+1| + C ✓

💡 Exemple : ∫ dx/(x²+4)

= (1/2)arctan(x/2) + C

En effet : ∫ dx/(x²+a²) = (1/a)arctan(x/a) + C

📕 8. Exercices résolus

📝 Exercice 1 — Calcul d'intégrale simple

Calculer ∫₀¹ (3x² + 2x + 1) dx

Solution :

= [x³ + x² + x]₀¹

= (1 + 1 + 1) − (0 + 0 + 0)

= 3

📝 Exercice 2 — Aire sous une courbe

Calculer l'aire entre y = √x, x = 1, x = 4 et l'axe Ox

Solution :

A = ∫₁⁴ √x dx = ∫₁⁴ x^(1/2) dx

= [x^(3/2) / (3/2)]₁⁴

= (2/3)·[4^(3/2) − 1^(3/2)]

= (2/3)·(8 − 1) = (2/3)·7 = 14/3 u.a.

📝 Exercice 3 — Intégration par parties

Calculer ∫₀¹ x·e^(−x) dx

Solution :

IPP : u = x, dv = e^(−x)dx → du = dx, v = −e^(−x)

= [−x·e^(−x)]₀¹ − ∫₀¹ (−e^(−x)) dx

= [−x·e^(−x)]₀¹ + [e^(−x)]₀¹

= (−1·e^(−1) − 0) + (e^(−1) − 1)

= −1/e + 1/e − 1 = −1

📝 Exercice 4 — Aire entre deux courbes

Aire entre y = 2x et y = x²

Solution :

Intersection : 2x = x² → x(x−2) = 0 → x = 0 et x = 2

Sur [0, 2] : 2x ≥ x²

A = ∫₀² (2x − x²) dx

= [x² − x³/3]₀²

= (4 − 8/3) − (0 − 0)

= 4/3

Aire = 4/3 u.a.

📝 Exercice 5 — Primitive avec ln

Calculer ∫ (x+2)/(x+1) dx

Solution :

(x+2)/(x+1) = (x+1+1)/(x+1) = 1 + 1/(x+1)

= ∫ 1 dx + ∫ 1/(x+1) dx

= x + ln|x+1| + C

📝 Exercice 6 — Théorème de la moyenne

Calculer la valeur moyenne de f(x) = 1/x sur [1, e]

Solution :

moyenne = (1/(e−1)) · ∫₁ᵉ (1/x) dx

= (1/(e−1)) · [ln|x|]₁ᵉ

= (1/(e−1)) · (1 − 0)

= 1/(e−1)

📝 Exercice 7 — Primitive de tan²(x)

Calculer ∫ tan²(x) dx

Solution :

tan²(x) = sec²(x) − 1

= ∫ sec²(x) dx − ∫ 1 dx

= tan(x) − x + C

📝 Exercice 8 — Changement de variable

Calculer ∫₀¹ x·√(1+x²) dx

Solution :

Poser u = 1+x² → du = 2x dx → x dx = du/2

Quand x=0 : u=1 ; quand x=1 : u=2

= ∫₁² √u · (du/2) = (1/2)∫₁² u^(1/2) du

= (1/2)·[u^(3/2)/(3/2)]₁²

= (1/3)·(2^(3/2) − 1)

= (2√2 − 1)/3

📘 1. Fonctions injectives et bijectives

Pour qu'une fonction admette une réciproque, elle doit être bijective. Comprenons d'abord injective et surjective.

🔹 Fonction injective

📐 Définition

f est injective sur I si :

∀x₁, x₂ ∈ I : x₁ ≠ x₂ ⟹ f(x₁) ≠ f(x₂)

Deux antécédents différents ont toujours des images différentes.

Test de la droite horizontale
Une fonction est injective ⟺ toute droite horizontale coupe son graphe au plus une fois

💡 Exemple : Injection

f(x) = 3x + 2 sur ℝ

Si 3x₁ + 2 = 3x₂ + 2 ⟹ 3x₁ = 3x₂ ⟹ x₁ = x₂

Donc f est injective sur ℝ ✓

💡 Contre-exemple : Non injective

f(x) = x² sur ℝ

f(2) = f(-2) = 4

Mais 2 ≠ -2

Donc f n'est pas injective sur ℝ ✗

Mais f(x) = x² est injective sur [0, +∞[ ou sur ]-∞, 0]

🔹 Fonction surjective

📐 Définition

f est surjective de I vers J si :

∀y ∈ J, ∃x ∈ I : f(x) = y

Tout élément de l'ensemble d'arrivée a au moins un antécédent.

🔹 Fonction bijective

📐 Définition

f est bijective si elle est à la fois :

• Injective : chaque image a un antécédent unique

• Surjective : tout élément de l'arrivée a un antécédent

📐

Bijection : chaque x a une image unique y, et chaque y a un antécédent unique x

📐

Condition pour admettre une réciproque : f doit être bijective

💡 Exemple : Bijection

f(x) = 2x + 1 de ℝ vers ℝ

• Injective : montré précédemment ✓

• Surjective : pour tout y ∈ ℝ, x = (y-1)/2 ∈ ℝ ✓

→ f est bijective sur ℝ

🔹 Restriction d'une fonction

📐 Principe

Si f n'est pas bijective sur son domaine, on peut restreindre son domaine pour la rendre bijective.

Exemple : f(x) = sin(x)

• Non injective sur ℝ

• Mais injective sur [-π/2, π/2] → c'est la restriction principale

📗 2. Définition de la fonction réciproque

La fonction réciproque permet de "remonter" de l'image vers l'antécédent.

🔹 Définition formelle

📐 Fonction réciproque

Si f est bijective de I vers J, alors la fonction réciproque f⁻¹ est définie par :

f⁻¹(y) = x   tel que   f(x) = y

f⁻¹ "inverse" l'effet de f : elle renvoie l'antécédent.

Notation
f⁻¹ se lit "f réciproque" ou "f inverse"
Attention : f⁻¹ ≠ 1/f   |   (1/f) se note (f)⁻¹ ou 1/f

🔹 Propriétés fondamentales

Identités réciproques
f⁻¹(f(x)) = x   pour tout x ∈ I   (réciproque de la réciproque)

f(f⁻¹(y)) = y   pour tout y ∈ J

💡 Exemple : Réciproque de x → 3x + 2

f(x) = 3x + 2, bijective de ℝ vers ℝ

y = 3x + 2 ⟺ x = (y - 2)/3

Donc f⁻¹(y) = (y - 2)/3

En variable x : f⁻¹(x) = (x - 2)/3

Vérification :

f⁻¹(f(x)) = ((3x + 2) - 2)/3 = 3x/3 = x ✓

💡 Exemple : Réciproque de x → x³

f(x) = x³ est bijective de ℝ vers ℝ

y = x³ ⟺ x = ∛y

f⁻¹(x) = ∛x

Vérification : f⁻¹(f(x)) = ∛(x³) = x ✓

🔹 Représentation graphique

📐 Symétrie

Les courbes de f et f⁻¹ sont symétriques par rapport à la droite y = x

Cette propriété permet de tracer f⁻¹ à partir de f.

📈

Point (a, b) sur Cf ⟺ point (b, a) sur Cf⁻¹

📉

Intersection avec y = x : points invariants

🔄

Monotonie : f et f⁻¹ ont la même monotonie

💡 Tracer la réciproque

Si on a la courbe de f(x) = 2x + 1

Pour tracer f⁻¹ :

1. Tracer la droite y = x

2. Symétriser la courbe de f par rapport à cette droite

3. La courbe symétrique est celle de f⁻¹(x) = (x - 1)/2

📙 3. Propriétés de la fonction réciproque

La fonction réciproque hérite et inverse certaines propriétés de la fonction originale.

🔹 Continuité

📐 Continuité de la réciproque

Si f est continue et bijective sur I, alors f⁻¹ est continue sur J = f(I).

🔹 Dérivabilité

📐 Dérivée de la réciproque

Si f est dérivable sur I, f⁻¹ existe, et f'(f⁻¹(y)) ≠ 0

(f⁻¹)'(y) = 1 / f'(f⁻¹(y))

Formule équivalente
(f⁻¹)'(x) = 1 / f'(f⁻¹(x))

💡 Exemple : Dérivée de ∛x

f(x) = x³, donc f'(x) = 3x²

f⁻¹(x) = ∛x

(f⁻¹)'(x) = 1 / f'(∛x)

= 1 / [3(∛x)²]

= 1 / (3 × x2/3)

(∛x)' = 1/(3∛(x²))

💡 Exemple : Dérivée de ln(x)

On sait que exp et ln sont réciproques : exp(ln(x)) = x

ln'(x) = 1 / exp'(ln(x))

= 1 / exp(ln(x))

= 1 / x

ln'(x) = 1/x ✓

🔹 Monotonie

📐 Monotonie de la réciproque

• Si f est strictement croissante, alors f⁻¹ est strictement croissante

• Si f est strictement décroissante, alors f⁻¹ est strictement décroissante

🔹 Limites

📐 Limites de la réciproque

Si limx→a f(x) = b, alors limy→b f⁻¹(y) = a

Les limites se "renversent" avec la réciproque.

💡 Exemple : Limite avec ln

limx→+∞ ln(x) = +∞

Donc limy→+∞ exp(y) = +∞ ✓

limx→0⁺ ln(x) = -∞

Donc limy→-∞ exp(y) = 0 ✓

🔹 Sommes et produits

⚠️ Propriété importante

En général : (f + g)⁻¹ ≠ f⁻¹ + g⁻¹

La réciproque d'une somme n'est pas la somme des réciproques.

De même pour le produit : (f × g)⁻¹ ≠ f⁻¹ × g⁻¹

🔹 Composée avec la réciproque

Composition avec la réciproque
f⁻¹ ∘ f = IdI   |   f ∘ f⁻¹ = IdJ

📕 4. Fonctions trigonométriques inverses

Les fonctions trigonométriques ne sont pas bijectives sur ℝ, donc on restreint leur domaine.

🔹 Fonction arcsin (arc sinus)

📐 Définition

sin : [-π/2, π/2] → [-1, 1] est bijective

arcsin : [-1, 1] → [-π/2, π/2]

arcsin(y) = x ⟺ sin(x) = y

Propriétés de arcsin
arcsin(-x) = -arcsin(x)   (impaire)
-π/2 ≤ arcsin(x) ≤ π/2
sin(arcsin(x)) = x   |   arcsin(sin(x)) = x (sur [-π/2, π/2])

🔹 Fonction arccos (arc cosinus)

📐 Définition

cos : [0, π] → [-1, 1] est bijective

arccos : [-1, 1] → [0, π]

arccos(y) = x ⟺ cos(x) = y

Propriétés de arccos
arccos(-x) = π - arccos(x)
0 ≤ arccos(x) ≤ π
arccos(x) + arcsin(x) = π/2

🔹 Dérivées

FonctionDérivée
arcsin(x)1/√(1 - x²)
arccos(x)-1/√(1 - x²)
arctan(x)1/(1 + x²)

💡 Dérivation de arcsin

y = arcsin(x) ⟺ sin(y) = x

Dérivons : cos(y) × y' = 1

y' = 1/cos(y) = 1/√(1 - sin²(y))

y' = 1/√(1 - x²)

arcsin'(x) = 1/√(1 - x²)

🔹 Fonction arctan (arc tangente)

📐 Définition

tan : ]-π/2, π/2[ → ℝ est bijective

arctan : ℝ → ]-π/2, π/2[

Propriétés de arctan
arctan(-x) = -arctan(x)   (impaire)
limx→+∞ arctan(x) = π/2
limx→-∞ arctan(x) = -π/2
arctan(x) + arctan(1/x) = π/2   (pour x > 0)

💡 Exemple : Calcul avec arctan

Calculer arctan(1) + arctan(2) + arctan(3)

En utilisant la formule arctan(a) + arctan(b) :

arctan(1) + arctan(2) = arctan((1+2)/(1-2)) + adjustment

= arctan(3) + π (car dénominateur négatif)

arctan(1) + arctan(2) + arctan(3) = arctan(3) + π + arctan(3)

= π

📒 5. Fonctions hyperboliques et leurs réciproques

Les fonctions hyperboliques sont définies à partir de l'exponentielle.

🔹 Fonctions hyperboliques directes

Définitions
sh(x) = (eˣ - e⁻ˣ)/2   (sinus hyperbolique)

ch(x) = (eˣ + e⁻ˣ)/2   (cosinus hyperbolique)

th(x) = sh(x)/ch(x)   (tangente hyperbolique)

🔹 Propriétés

📐

ch²(x) - sh²(x) = 1 (identité hyperbolique)

📐

sh est impaire, ch est paire

📐

sh'(x) = ch(x) et ch'(x) = sh(x)

📐

th'(x) = 1 - th²(x)

🔹 Fonctions hyperboliques réciproques

📐 Arguments hyperboliques

argsinh(y) = ln(y + √(y² + 1))

argcosh(y) = ln(y + √(y² - 1))   (pour y ≥ 1)

argtanh(y) = (1/2) × ln((1+y)/(1-y))   (pour |y| < 1)

💡 Exemple : argsinh

y = argsinh(x) ⟺ sh(y) = x

(eʸ - e⁻ʸ)/2 = x

Multiplions par 2eʸ : e²ʸ - 2xeʸ - 1 = 0

Posons t = eʸ : t² - 2xt - 1 = 0

t = x ± √(x² + 1)

Comme t = eʸ > 0 : t = x + √(x² + 1)

argsinh(x) = ln(x + √(x² + 1))

🔹 Dérivées

FonctionDérivée
argsinh(x)1/√(x² + 1)
argcosh(x)1/√(x² - 1)
argtanh(x)1/(1 - x²)

💡 Exemple : Dérivée de argsinh

argsinh(x) = ln(x + √(x² + 1))

Dérivons :

= 1/(x + √(x²+1)) × (1 + x/√(x²+1))

= 1/(x + √(x²+1)) × (√(x²+1) + x)/√(x²+1)

= 1/√(x² + 1)

argsinh'(x) = 1/√(x² + 1)

🔹 Résumé : Liens entre fonctions

🔄

exp ↔ ln   |   exp(x) = y ⟺ ln(y) = x

🔄

sin ↔ arcsin   |   sin(x) = y ⟺ arcsin(y) = x (sur [-π/2, π/2])

🔄

sh ↔ argsinh   |   sh(x) = y ⟺ argsinh(y) = x

🔄

xⁿ ↔ x1/n   |   xⁿ = y ⟺ x = ∛y

⚠️ Attention aux domaines

• sin est bijective sur [-π/2, π/2] → arcsin prend ses valeurs dans [-π/2, π/2]

• ch est bijective sur [0, +∞[ → argcosh prend ses valeurs dans [0, +∞[

• argcosh n'existe que pour x ≥ 1 (car ch(x) ≥ 1)

📘 1. Définition de la fonction puissance

Idée générale

La fonction puissance est une fonction de la forme f(x) = xᵅ où α est un exposant réel. Selon la nature de α (entier positif, entier négatif, rationnel, irrationnel), le domaine de définition et les propriétés changent.

Définition

Fonction puissance
f(x) = xᵅ   |   α ∈ ℝ

Rappel : xᵅ = e^(α·ln x)   |   pour x > 0 uniquement si α est non-entier

Lien avec ln et exp

Formule de passage
xᵅ = e^(α · ln x)   |   ∀x > 0, ∀α ∈ ℝ

Cette formule est la clé : elle permet de définir xᵅ pour tout α réel et x > 0.

Domaines selon α

📐

α ∈ ℕ : Df = ℝ (polynôme)

📐

α ∈ ℤ⁻ : Df = ℝ* (x⁻ⁿ = 1/xⁿ)

📐

α = p/q (rationnel) : Df dépend de q

📐

α ∉ ℚ : Df = ]0, +∞[ (défini par ln)

📗 2. Propriétés algébriques

Propriétés valables pour tout α, β et x > 0

Produit
xᵅ · xᵝ = xᵅ⁺ᵝ
Quotient
xᵅ / xᵝ = xᵅ⁻ᵝ
Puissance de puissance
(xᵅ)ᵝ = xᵅ·ᵝ
Propriété commutative
(x·y)ᵅ = xᵅ · yᵅ   |   (x/y)ᵅ = xᵅ / yᵅ

💡 Exemples

x³ · x⁴ = x⁷

(x²)⁵ = x¹⁰

(2x)³ = 8x³

(x/y)⁻² = x⁻² / y⁻² = y² / x²

⚠️ Erreurs fréquentes

❌ xᵅ · yᵝ ≠ (xy)ᵅ⁺ᵝ (les exposants doivent être égaux)

❌ (xᵅ)ᵝ ≠ xᵅ^ᵝ (c'est xᵅ·ᵝ)

❌ x⁰ ≠ 0 (x⁰ = 1 pour tout x ≠ 0)

✅ (−2)⁴ = 16 mais (−2)³ = −8 (la parité compte pour les racines)

📙 3. Fonction xⁿ pour n ∈ ℕ

Fonctions xⁿ — Polynômes

📐

n pair : f(x) = xⁿ est paire, positive, admet un minimum en 0

📐

n impair : f(x) = xⁿ est impaire, strictement croissante, bijective ℝ→ℝ

Dérivée de xⁿ

Formule fondamentale
(xⁿ)' = n·xⁿ⁻¹   |   ∀n ∈ ℕ, ∀x ∈ ℝ

Primitive de xⁿ

Primitive
∫ xⁿ dx = xⁿ⁺¹/(n+1) + C   |   ∀n ≠ -1

Fonctions x¹/², x¹/³, etc. — Racines

Quand l'exposant est 1/n, on parle de racine n-ième.

Racine n-ième
x^(1/n) = ⁿ√x   |   (ⁿ√x)ⁿ = x

Domaine :
• Si n est impair : D = ℝ
• Si n est pair : D = [0, +∞[
Propriétés des racines
ⁿ√a · ⁿ√b = ⁿ√(ab)
ⁿ√a / ⁿ√b = ⁿ√(a/b)
(ⁿ√a)ᵐ = ᵐⁿ√(aᵐ) = aᵐ/ⁿ

💡 Exemple 1

√4 = 2, √9 = 3

∛8 = 2, ∛(-8) = -2

√2 · √8 = √16 = 4

∛27 · ∛1000 = ∛(27000) = 30

💡 Exemple 2 — Simplification

√12 = √(4×3) = 2√3

∛54 = ∛(27×2) = 3∛2

√50 + √8 - √18 = 5√2 + 2√2 - 3√2 = 4√2

Dérivée de x^(1/n)

Dérivée d'une racine
(x^(1/n))' = (1/n) · x^(1/n - 1) = x^((1-n)/n) / n

💡 Exemple

f(x) = √x = x^(1/2)

f'(x) = (1/2) · x^(-1/2) = 1/(2√x)

Dérivable sur ]0, +∞[ uniquement (car √x n'a pas de dérivée en 0)

📕 4. Fonction xᵅ pour α réel (via ln)

Définition pour α ∈ ℝ

Pour tout α réel et x > 0, on définit :

Définition par ln et exp
xᵅ = e^(α · ln x)   |   ∀x > 0, ∀α ∈ ℝ

Fonctions xᵅ selon α

📊

α > 0 : xᵅ est définie en x = 0 si α ≥ 1, et croît

📊

α < 0 : asymptote verticale en x = 0, décroissante

📊

α = 0 : f(x) = 1 (constante)

📊

α = 1/n : f(x) = ⁿ√x

Dérivée de xᵅ

Formule universelle
(xᵅ)' = α · xᵅ⁻¹   |   ∀α ∈ ℝ, ∀x > 0

Cette formule est la même que pour n entier !

💡 Exemple 1

f(x) = x^(3/2)

f'(x) = (3/2) · x^(1/2) = (3/2)√x

Domaine : x ≥ 0 (car √x)

💡 Exemple 2

f(x) = x^(-2) = 1/x²

f'(x) = (-2) · x^(-3) = -2/x³

f'(x) < 0 pour tout x ≠ 0 → strictement décroissante sur ]0,+∞[ et ]-∞,0[

💡 Exemple 3 — Primitive

∫ x^(2/3) dx

= x^(5/3) / (5/3) + C = (3/5)·x^(5/3) + C

📒 5. Limites de xᵅ

Limites fondamentales

Limites en +∞
lim(x→+∞) xᵅ = +∞   |   si α > 0
lim(x→+∞) xᵅ = 0   |   si α < 0
lim(x→+∞) xᵅ = 1   |   si α = 0
Limites en 0⁺
lim(x→0⁺) xᵅ = +∞   |   si α < 0
lim(x→0⁺) xᵅ = 0   |   si α > 0
lim(x→0⁺) x⁰ = 1

Croissance comparée des puissances

Hiérarchie quand x → +∞
xᵅ grandit d'autant plus vite que α est grand

Si α > β :   lim(x→+∞) xᵅ / xᵝ = +∞
Si α < β :   lim(x→+∞) xᵅ / xᵝ = 0

💡 Exemples de limites

lim(x→+∞) x^(1/2) / x = lim x^(-1/2) = 0 ✓

lim(x→0⁺) x³ = 0

lim(x→0⁺) x^(-2) = +∞

lim(x→+∞) √x / x = 0

Comparaison ln, xᵅ, eˣ

📊

ln(x) croît plus lentement que toute puissance positive xᵅ

📊

eˣ croît plus vite que toute puissance xᵅ

📊

ln(x) << xᵅ << eˣ   (pour x → +∞)

📓 6. Exercices résolus

📝 Exercice 1 — Simplification

Simplifier : (x³·√x) / x^(1/2)

Solution :

x³ · x^(1/2) / x^(1/2) = x³ · x^(1/2 - 1/2) = x³ · x⁰ = x³

📝 Exercice 2 — Dérivation

f(x) = x²·√x = x^(5/2)

Solution :

f'(x) = (5/2) · x^(3/2) = (5/2)·x·√x

📝 Exercice 3 — Primitive

∫ x^(3/2) dx

Solution :

= x^(5/2) / (5/2) + C = (2/5)·x^(5/2) + C

📝 Exercice 4 — Racines

Simplifier : √(12x⁶)

Solution :

√(12) · √(x⁶) = 2√3 · |x³|

Si x ≥ 0 : = 2√3 · x³

Si x < 0 : = 2√3 · (-x)³ = -2√3·x³

📝 Exercice 5 — Limite

lim(x→+∞) x^(1/3) / √x

Solution :

= x^(1/3) / x^(1/2) = x^(1/3 - 1/2) = x^(-1/6)

Quand x→+∞, x^(-1/6) → 0

lim = 0

📝 Exercice 6 — Équation avec racine

∛x = 2

Solution :

On cube : x = 2³ = 8

Solution : x = 8

📘 1. Fonctions polynômes

Définition

Une fonction polynôme est une fonction de la forme :

Forme générale d'un polynôme
P(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀
où aₙ, aₙ₋₁, ..., a₀ ∈ ℝ et aₙ ≠ 0

Le nombre n est appelé le degré du polynôme. Le coefficient aₙ est le coefficient dominant.

Fonctions usuelles par degré

📐

Degré 0 : P(x) = c (constante) — droite horizontale

📐

Degré 1 : P(x) = ax + b — fonction affine (droite)

📐

Degré 2 : P(x) = ax² + bx + c — fonction quadratique (parabole)

📐

Degré 3 : P(x) = ax³ + bx² + cx + d — fonction cubique

Factorisation des polynômes

Tout polynôme de degré n peut s'écrire sous forme factorisée :

Factorisation
P(x) = aₙ × (x - r₁)(x - r₂)...(x - rₙ)
où r₁, r₂, ..., rₙ sont les racines du polynôme

⚠️ Racines et discriminant (degré 2)

Pour ax² + bx + c = 0 :

• Δ = b² - 4ac

• Si Δ > 0 : deux racines réelles distinctes

• Si Δ = 0 : une racine double (x = -b/2a)

• Si Δ < 0 : deux racines complexes conjuguées

Opérations sur les polynômes

• Addition : on additionne les coefficients des termes de même degré

• Multiplication : on développe en appliquant la distributivité

• Division euclidienne : pour A et B polynômes, ∃!(Q, R) tels que A = BQ + R avec deg(R) < deg(B)

💡 Exemple 1 — Factorisation

P(x) = x³ - 4x² - 7x + 10

On cherche une racine évidente : P(1) = 1 - 4 - 7 + 10 = 0

Donc (x - 1) est un facteur. Division euclidienne :

P(x) = (x - 1)(x² - 3x - 10) = (x - 1)(x - 5)(x + 2)

Racines : 1, 5, -2

💡 Exemple 2 — Division euclidienne

Diviser x³ + 2x² - 5x + 3 par x² - 1 :

x³ + 2x² - 5x + 3 = (x + 2)(x² - 1) + (-3x + 5)

Donc Q(x) = x + 2 et R(x) = -3x + 5

Vérification : (x+2)(x²-1) + (-3x+5) = x³ + 2x² - x - 2 - 3x + 5 = x³ + 2x² - 5x + 3 ✓

📗 2. Fonctions rationnelles

Définition

Une fonction rationnelle est le quotient de deux polynômes :

Forme générale
f(x) = P(x) / Q(x)   |   avec Q(x) ≠ 0

Domaine de définition : Df = ℝ \ {x | Q(x) = 0}

Asymptotes

Les fonctions rationnelles peuvent admettre différents types d'asymptotes :

📉

Asymptote verticale : en x = a si Q(a) = 0 et P(a) ≠ 0

📊

Asymptote horizontale : si deg(P) < deg(Q) → y = 0 ; si deg(P) = deg(Q) → y = aₙ/bₙ

📈

Asymptote oblique : si deg(P) = deg(Q) + 1, on effectue la division euclidienne

Décomposition en éléments simples

Pour étudier et intégrer une fonction rationnelle, on la décompose en éléments simples.

Cas 1 — Pôles simples
f(x) = (2x + 1) / ((x - 1)(x + 2))
= A/(x - 1) + B/(x + 2)
On trouve A = 3/2 et B = 1/2
Cas 2 — Pôles multiples
f(x) = (x²) / ((x - 1)²(x + 2))
= A/(x - 1) + B/(x - 1)² + C/(x + 2)

💡 Exemple — Asymptote oblique

f(x) = (x² + 3x + 1) / (x + 1)

Division euclidienne : x² + 3x + 1 = (x + 1)(x + 2) - 1

donc f(x) = x + 2 - 1/(x + 1)

Asymptote oblique : y = x + 2

La courbe se rapproche de la droite y = x + 2 quand |x| → ∞

💡 Exemple — Domaine et asymptotes

f(x) = (x² - 4) / (x² - x - 2)

Dénominateur : x² - x - 2 = (x - 2)(x + 1)

Domaine : Df = ℝ \ {-1, 2}

Asymptotes verticales : x = -1 et x = 2

Asymptote horizontale : y = 1 (car deg num = deg dén)

Simplification : f(x) = (x - 2)(x + 2) / ((x - 2)(x + 1)) = (x + 2)/(x + 1) pour x ≠ 2

⚠️ Trou (lacune)

Si P(a) = 0 et Q(a) = 0, il y a simplification possible :

Ex : f(x) = (x² - 4)/(x - 2) = (x - 2)(x + 2)/(x - 2) = x + 2 pour x ≠ 2

Il y a un trou au point (2, 4) qui n'appartient pas à la courbe.

📙 3. Fonctions trigonométriques

Les fonctions trigonométriques usuelles

En terminale math tunisienne, on studie trois fonctions trigonométriques principales :

📐

sin(x) : fonction impaire, période 2π, bornée entre -1 et 1

📐

cos(x) : fonction paire, période 2π, bornée entre -1 et 1

📐

tan(x) = sin(x)/cos(x), période π, asymptotes verticales en x = π/2 + kπ

Propriétés fondamentales

Identité fondamentale
sin²(x) + cos²(x) = 1   |   ∀x ∈ ℝ
Formules d'addition
sin(a + b) = sin(a)cos(b) + cos(a)sin(b)
sin(a - b) = sin(a)cos(b) - cos(a)sin(b)
cos(a + b) = cos(a)cos(b) - sin(a)sin(b)
cos(a - b) = cos(a)cos(b) + sin(a)sin(b)
Formules de duplication
sin(2x) = 2sin(x)cos(x)
cos(2x) = cos²(x) - sin²(x) = 2cos²(x) - 1 = 1 - 2sin²(x)
tan(2x) = 2tan(x) / (1 - tan²(x))
Formules de Carnot (passage produit → somme)
cos(a)cos(b) = [cos(a-b) + cos(a+b)] / 2
sin(a)sin(b) = [cos(a-b) - cos(a+b)] / 2
sin(a)cos(b) = [sin(a+b) + sin(a-b)] / 2

Dérivées des fonctions trigonométriques

Dérivées
(sin x)' = cos x
(cos x)' = -sin x
(tan x)' = 1 + tan²(x) = 1/cos²(x)

Limites remarquables

Limites importantes
lim(x→0) sin(x)/x = 1
lim(x→0) tan(x)/x = 1
lim(x→0) (1 - cos(x))/x² = 1/2
lim(x→∞) sin(x)/x = 0

💡 Exemple 1 — Dérivation trigonométrique

f(x) = sin³(2x + 1)

f'(x) = 3sin²(2x+1) × cos(2x+1) × 2

f'(x) = 6 sin²(2x+1) cos(2x+1)

💡 Exemple 2 — Calcul de limite

lim(x→0) sin(3x)/tan(2x)

= lim(x→0) sin(3x)/3x × 2x/tan(2x) × 3x/(2x)

= 1 × 1 × 3/2 = 3/2

💡 Exemple 3 — Linéarisation

Exprimer cos³(x) en fonction de cos(3x) :

cos³(x) = (1/4)(3cos(x) + cos(3x))

Cette formule est utile pour l'intégration.

📕 4. Étude complète de fonctions

Méthode d'étude d'une fonction

Pour étudier une fonction f, on suit toujours le plan suivant :

📝

1. Déterminer le domaine de définition Df

📝

2. Étudier la parité (f(-x)) et la périodicité

📝

3. Calculer les limites aux bornes du domaine

📝

4. Calculer f'(x) et étudier son signe (tableau de variation)

📝

5. Chercher les asymptotes (verticales, horizontales, obliques)

📝

6. Tracer la courbe représentative

💡 Exemple complet — f(x) = (x² - 3x + 2)/(x - 1)

1. Domaine : x - 1 ≠ 0 → Df = ℝ \ {1}

2. Simplification : (x-1)(x-2)/(x-1) = x - 2 pour x ≠ 1

Il y a un trou en (1, -1) et une droite y = x - 2 pour x ≠ 1

3. Limites :

• lim(x→1⁺) f(x) = -1+ (par simplification)

• lim(x→±∞) f(x) = ±∞ (droite oblique)

4. Asymptote oblique : y = x - 2 (droite)

5. Tracé : tracer la droite y = x - 2 avec un trou en x = 1

💡 Exemple — Fonction trigonométrique

f(x) = sin(x) + (1/2)sin(2x)

Période : T = 2π (PPCM de 2π et π)

f'(x) = cos(x) + cos(2x)

f'(x) = 0 → cos(x) + 2cos²(x) - 1 = 0

→ cos(x) = 1/2 ou cos(x) = -1

→ x = ±π/3 + 2kπ ou x = π + 2kπ

Tableau de variation sur [0, 2π] :

x : 0 → π/3 → π → 5π/3 → 2π

f' : + → 0 → - → 0 → +

f : 0 → f(π/3)=3√3/4 ≈ 1,3 → 0 → f(5π/3)=-3√3/4 → 0

📒 5. Exercices résolus

📝 Exercice 1 — Polynôme

Soit P(x) = 2x³ - 7x² + 2x + 3

a) Montrer que 1 est racine de P

b) Factoriser P(x)

c) Résoudre P(x) = 0

Solution :

a) P(1) = 2 - 7 + 2 + 3 = 0 ✓

b) Division euclidienne par (x - 1) :

P(x) = (x - 1)(2x² - 5x - 3)

Δ = 25 + 24 = 49 > 0 → 2x² - 5x - 3 = (x - 3)(2x + 1)

c) P(x) = (x - 1)(x - 3)(2x + 1) = 0

→ Solutions : x = 1, x = 3, x = -1/2

📝 Exercice 2 — Fonction rationnelle

Soit f(x) = (x² + x - 2)/(x² + 3x + 2)

a) Déterminer Df

b) Déterminer les asymptotes

c) Écrire la fonction sous forme simplifiée

Solution :

a) x² + 3x + 2 = (x+1)(x+2) = 0 → Df = ℝ \ {-1, -2}

b) AV : x = -1 et x = -2

AH : y = 1 (car deg num = deg dén = 2, rapport coefficients = 1)

c) x² + x - 2 = (x+2)(x-1)

f(x) = (x+2)(x-1)/(x+2)(x+1) = (x-1)/(x+1) pour x ≠ -2

Trou en x = -2 : f(-2) = (-2-1)/(-2+1) = -3/-1 = 3

Trou au point (-2, 3)

📝 Exercice 3 — Trigonométrie

Simplifier : A = (sin(3x) - 3sin(x)) / (cos(3x) - cos(x))

Solution :

sin(3x) = 3sin(x) - 4sin³(x) → sin(3x) - 3sin(x) = -4sin³(x)

cos(3x) = 4cos³(x) - 3cos(x) → cos(3x) - cos(x) = 4cos³(x) - 4cos(x)

= 4cos(x)(cos²(x) - 1) = -4cos(x)sin²(x)

A = (-4sin³(x))/(-4cos(x)sin²(x)) = tan(x)

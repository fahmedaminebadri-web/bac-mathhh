📘 1. Définition de la fonction exponentielle

Rappel — Pourquoi exp ?

La fonction logarithme népérien ln(x) est définie sur ]0, +∞[ et est strictement croissante. Elle admet une fonction réciproque appelée fonction exponentielle, notée exp(x) ou eˣ.

Définition formelle

Définition
exp(x) = eˣ est l'unique fonction f telle que :
f' = f   |   f(0) = 1

Relation fondamentale :
exp(ln x) = x   |   ln(eˣ) = x

Propriétés clés de eˣ

📐

Définition : ∀x ∈ ℝ, eˣ > 0 (toujours positive)

📐

Courbe : passe par (0, 1) — y = 1 à x = 0

📐

Croissance : strictement croissante sur ℝ

📐

Dérivée : (eˣ)' = eˣ (propriété extraordinaire !)

Représentation graphique

La courbe de eˣ est l'image de la courbe de ln(x) par la symétrie d'axe y = x.

📊

Domaine : ℝ

📊

Image : ]0, +∞[

📊

Asymptote horizontale : y = 0 quand x → -∞

📊

En x = 0 : e⁰ = 1

Valeurs remarquables

Valeurs importantes
e⁰ = 1
e¹ = e ≈ 2,718
e² ≈ 7,389
e⁻¹ = 1/e ≈ 0,368
e⁻² ≈ 0,135

📗 2. Propriétés algébriques de exp

Propriétés fondamentales

Produit
eᵃ × eᵇ = eᵃ⁺ᵇ   |   ∀a, ∀b ∈ ℝ
Quotient
eᵃ / eᵇ = eᵃ⁻ᵇ
Puissance
(eᵃ)ᵇ = eᵃᵇ   |   e⁻ᵃ = 1/eᵃ
Racine
eᵃ/² = √(eᵃ)   |   e^(a/2) = √(eᵃ)

Relation entre ln et exp

🔗

ln(eˣ) = x — la composition ln∘exp "s'annule"

🔗

e^(ln x) = x — pour x > 0

🔗

e^a = e^b ⇔ a = b

🔗

e^a > e^b ⇔ a > b (car exp est croissante)

💡 Exemple 1 — Calcul avec exp

Simplifier : A = e² × e⁵ / e³

A = e²⁺⁵⁻³ = e⁴

Remarque : e⁴ = exp(4) et ne se calcule pas en valeur exacte simplement

Valeur approchée : e⁴ ≈ 54,6

💡 Exemple 2 — Simplification

Simplifier : B = (eˣ)² × e⁻ˣ / e³ˣ

B = e²ˣ × e⁻ˣ / e³ˣ = e²ˣ⁻ˣ⁻³ˣ = e⁻²ˣ

💡 Exemple 3 — Utilisation de ln

Résoudre eˣ = 3

⇔ ln(eˣ) = ln(3)

⇔ x = ln(3)

⇔ x = ln(3) ≈ 1,099

⚠️ Pièges à éviter

❌ eˣ⁺ʸ ≠ eˣ + eʸ (c'est eˣ · eʸ !)

❌ eˣ² ≠ (eˣ)² (le premier = e^(x²), le second = e^(2x))

✅ eˣ est TOUJOURS positive

✅ Pour passer de ln à exp : e^(ln x) = x

📙 3. Limites et dérivée de exp

Dérivée

Dérivée de eˣ
(eˣ)' = eˣ   |   ∀x ∈ ℝ
Dérivée de e^(u(x)) — Formule essentielle
[e^(u(x))]' = u'(x) · e^(u(x))

Limites remarquables

Limites fondamentales
lim(x→-∞) eˣ = 0
lim(x→+∞) eˣ = +∞
lim(x→0) (eˣ - 1)/x = 1
lim(x→+∞) eˣ / xⁿ = +∞   (∀n ∈ ℕ)
lim(x→+∞) xⁿ / eˣ = 0
Croissance comparée (fondamental !)
lim(x→+∞) eˣ / xⁿ = +∞   |   n'importe quelle puissance perd face à l'exponentielle
lim(x→-∞) xⁿ · eˣ = 0

Formes indéterminées

FI à maîtriser
lim(x→0) (eˣ - 1)/x = 1
lim(x→+∞) x/eˣ = 0
lim(x→0) eˣ - 1 ~ x
lim(x→+∞) eᵃˣ / xᵇ = +∞   (pour a > 0, b > 0)

💡 Exemple 1 — Dérivée

f(x) = e^(2x + 1)

f'(x) = 2 · e^(2x + 1)

Signe : toujours positif → f strictement croissante

💡 Exemple 2 — Dérivée composée

f(x) = e^(x²)

f'(x) = 2x · e^(x²)

f'(x) > 0 si x > 0, f'(x) < 0 si x < 0

f est décroissante sur ]-∞, 0[ et croissante sur ]0, +∞[

💡 Exemple 3 — Limite

lim(x→+∞) (eˣ) / (x¹⁰)

Par croissance comparée : lim = +∞

L'exponentielle l'emporte toujours sur n'importe quelle puissance

💡 Exemple 4 — Primitive

∫ e^(3x) dx

∫ e^(3x) dx = (1/3) · e^(3x) + C

Règle : ∫ e^(ax+b) dx = (1/a) · e^(ax+b) + C

💡 Exemple 5 — Limite avec FI

lim(x→0) (e^(2x) - 1) / (eˣ - 1)

Quand x→0 : num → 0, denom → 0

Règle de l'Hôpital : num' = 2e^(2x), denom' = eˣ

lim = lim(x→0) 2e^(2x)/eˣ = 2e⁰/e⁰ = 2

📕 4. Équations et inéquations exponentielles

Équations

Principe fondamental : eᵃ = eᵇ ⇔ a = b

Types d'équations
e^(u(x)) = e^(v(x))  ⇔  u(x) = v(x)
e^(u(x)) = k             ⇔  u(x) = ln(k)   (k > 0)

💡 Exemple 1 — Équation simple

e^(3x) = e²

⇔ 3x = 2

⇔ x = 2/3

Solution : x = 2/3

💡 Exemple 2 — eˣ = k

eˣ = 5

⇔ x = ln(5)

Solution : x = ln(5) ≈ 1,609

💡 Exemple 3 — Équation avec substitution

e^(2x) - 3eˣ + 2 = 0

On pose X = eˣ (X > 0)

X² - 3X + 2 = 0

(X - 1)(X - 2) = 0

X = 1 ou X = 2

eˣ = 1 → x = 0

eˣ = 2 → x = ln(2)

Solutions : x = 0 et x = ln(2)

💡 Exemple 4 — Équation avec ln

e^(x+1) = 3

⇔ x + 1 = ln(3)

⇔ x = ln(3) - 1

Solution : x = ln(3) - 1

Inéquations

Principe : eᵃ > eᵇ ⇔ a > b (car exp est strictement croissante)

Inéquations
e^(u(x)) > e^(v(x))  ⇔  u(x) > v(x)
e^(u(x)) > k             ⇔  u(x) > ln(k)   (k > 0)
e^(u(x)) < k             ⇔  u(x) < ln(k)   (k > 0)

💡 Exemple 5 — Inéquation

e^(2x) > 3

⇔ 2x > ln(3)

⇔ x > ln(3)/2

Solution : ]ln(3)/2, +∞[

💡 Exemple 6 — Inéquation quadratique

e^(2x) - 3eˣ + 2 < 0

On pose X = eˣ, X > 0

X² - 3X + 2 < 0

(X - 1)(X - 2) < 0

⇔ 1 < X < 2

⇔ 1 < eˣ < 2

⇔ ln(1) < x < ln(2)

⇔ 0 < x < ln(2)

📒 5. Fonction eᵘ⁽ˣ⁾ et applications

Fonction e^(u(x))

La forme la plus générale de l'exponentielle qu'on étudie est f(x) = e^(u(x)) où u(x) est une fonction.

Étude de f(x) = e^(u(x))
Dérivée : f'(x) = u'(x) · e^(u(x))
Signe : même signe que u'(x) (car e^(u) > 0 toujours)
Tableau : les variations de u se reflètent dans f

💡 Exemple — Étude complète

f(x) = e^(x² - 2x)

• Domaine : ℝ

• u(x) = x² - 2x, u'(x) = 2x - 2

• f'(x) = (2x - 2) · e^(x² - 2x)

• Signe : f'(x) > 0 ⇔ x > 1

• Décroissante sur ]-∞, 1[, croissante sur ]1, +∞[

• Minimum en x = 1 : f(1) = e^(1 - 2) = e^(-1) = 1/e

• Asymptote horizontale : y = 0 en -∞

Applications concrètes

📈

Croissance / décroissance exponentielle : y = e^(kt)

📈

Intérêts composés : C(t) = C₀·e^(rt) — capital croissant continuellement

📈

Désintégration radioactive : N(t) = N₀·e^(-λt)

📈

Physique : charge d'un condensateur RC, réponse d'un circuit RL

Équation différentielle fondamentale

Équation différentielle y' = y
Les solutions de y' = y sont f(x) = C·eˣ (C ∈ ℝ)
C'est l'équation fondamentale de la croissance exponentielle

📓 6. Exercices résolus

📝 Exercice 1 — Dérivation

Calculer la dérivée de f(x) = eˣ · (x² + 1)

Solution :

f'(x) = eˣ · (x² + 1) + eˣ · 2x

= eˣ · (x² + 2x + 1)

= eˣ · (x + 1)²

📝 Exercice 2 — Limite

lim(x→+∞) (eˣ - 1) / x²

Solution :

Par croissance comparée : eˣ domine x²

lim = +∞ / +∞ (FI)

Règle de l'Hôpital appliquée 2 fois :

lim = lim eˣ / (2x) = lim eˣ / 2 = +∞

📝 Exercice 3 — Équation

eˣ + e⁻ˣ = 3

Solution :

On pose t = eˣ, t > 0

t + 1/t = 3

t² + 1 = 3t

t² - 3t + 1 = 0

Δ = 9 - 4 = 5

t = (3 ± √5)/2

Les deux sont > 0

x = ln((3 ± √5)/2)

Solutions : x = ln((3+√5)/2) et x = ln((3-√5)/2)

📝 Exercice 4 — Primitive

∫ (eˣ - 1) / eˣ dx

Solution :

(eˣ - 1) / eˣ = 1 - e⁻ˣ

∫ (1 - e⁻ˣ) dx = x + e⁻ˣ + C

📝 Exercice 5 — Inéquation avec ln et exp

e^(2x) ≤ 4

Solution :

⇔ 2x ≤ ln(4) = 2ln(2)

⇔ x ≤ ln(2)

Solution : ]-∞, ln(2)]

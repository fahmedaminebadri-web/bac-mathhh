2. pH des solutions d'acides et bases forts

🧪 Acide fort AH

Condition : Cₐ > 10⁻⁶ mol·L⁻¹, pH ≤ 6

Réaction :

AH + H₂O → A⁻ + H₃O⁺

Taux d'avancement : τf = 1 (totale)

[H₃O⁺] = Cₐ
pH = −log Cₐ

🧪 Base forte B

Condition : Cᵦ > 10⁻⁶ mol·L⁻¹, pH ≥ 8

Réaction :

B + H₂O → BH⁺ + OH⁻

Taux d'avancement : τf = 1 (totale)

[OH⁻] = Cᵦ
pH = pKe + log Cᵦ

📊 Tableau d'avancement (acide fort)

Final n₀ − xf − xf xf

3. pH des solutions d'acides et bases faibles

🧪 Acide faible AH

Condition : τf < 0,05 (faiblement dissocié)

Réaction (état d'équilibre) :

AH + H₂O ⇌ A⁻ + H₃O⁺

Constante d'acidité

Ka = [A⁻][H₃O⁺] / [AH]

pKa = −log Ka

Expression du pH :

pH = ½(pKa − log Cₐ)

Taux d'avancement final :

τf = √(Ka/Cₐ)

🧪 Base faible B

Condition : τf < 0,05 (faiblement dissociée)

Réaction (état d'équilibre) :

B + H₂O ⇌ BH⁺ + OH⁻

Constante de basicité

Kb = [BH⁺][OH⁻] / [B]

Relation importante :

Ka × Kb = Ke = 10⁻¹⁴

Expression du pH :

pH = ½(pKe + pKa + log Cᵦ)

4. Effet de la dilution sur le pH

📐 Relation de dilution

Si on dilue n fois (n > 1) :

C₁V₁ = C₂V₂ ⟹ n = C₁/C₂

🔴 Dilution d'un acide fort

pH₂ = pH₁ + log n

Le pH augmente de log n

τf reste = 1 (inchangé)

Exemple : dilution 10 fois → pH augmente de 1

🔴 Dilution d'un acide faible

pH₂ = pH₁ + ½ log n

Le pH augmente de ½ log n

τf augmente (la dissociation augmente)

Exemple : dilution 100 fois → pH augmente de 1

🔵 Dilution d'une base forte

pH₂ = pH₁ − log n

Le pH diminue de log n

τf reste = 1 (inchangé)

🔵 Dilution d'une base faible

pH₂ = pH₁ − ½ log n

Le pH diminue de ½ log n

τf augmente (la dissociation augmente)

⚠️ Cas spécial

La dilution d'une solution neutre (pH = 7) n'a aucun effet sur le pH.

  • Équivalence : état où nₐcide = nᵦₐse
  • Relation à l'équivalence : CₐVₐ = CᵦVᵦₑ

6. Solutions tampons

📌 Définition

Une solution tampon est un mélange d'un acide faible et de sa base conjuguée (ou d'une base faible et de son acide conjugué) qui résiste aux variations de pH lors de l'ajout modéré d'acide ou de base.

Propriétés
  • pH stable malgré l'ajout d'acide ou de base modéré
  • pH quasi inchangé par dilution modérée
  • pH calculable par la relation de Henderson-Hasselbalch :

pH = pKa + log([base]/[acide])

Exemples de solutions tampons
  • Mélange d'acide acétique (CH₃COOH) et d'acétate (CH₃COO⁻)
  • Mélange de NH₄⁺ et NH₃ (tampon ammoniacal)
  • Le sang humain (tampon carbonate)

7. Indicateurs colorés de pH

📌 Définition

Un indicateur coloré est un couple acide/base dont les formes acide et basique ont des couleurs différentes.

Zone de virage

La zone de virage est l'intervalle de pH où l'indicateur prend une couleur intermédiaire.

La teinte sensible correspond à pH = pKind (quand [HInd] = [Ind⁻])

Tableau des indicateurs courants
Indicateur Couleur acide Zone de virage Couleur basique
Hélianthine Rouge 3,1 - 4,4 Jaune
Bleu de bromothymol Jaune 6,0 - 7,6 Bleu
Phénolphtaléine Incolore 8,2 - 10,0 Rose
💡 Choix de l'indicateur

On choisit un indicateur tel que le pH à l'équivalence appartienne à sa zone de virage.

  • Dosage acide fort + base forte : pH = 7 → BBT
  • Dosage acide faible + base forte : pH > 7 → Phénolphtaléine
  • Dosage base faible + acide fort : pH < 7 → Hélianthine

🔑 Concepts clés à retenir

  • pH = −log[H₃O⁺] et [H₃O⁺] = 10⁻ᵖᴴ
  • Ke = 10⁻¹⁴ = [H₃O⁺][OH⁻]
  • Acide fort : pH = −log Cₐ
  • Base forte : pH = pKe + log Cᵦ
  • Acide faible : pH = ½(pKa − log Cₐ)
  • Dilution : pH acide fort + log n
  • Équivalence : CₐVₐ = CᵦVᵦₑ
  • Demi-équivalence : pH = pKa
  • Tampon : pH = pKa + log([base]/[acide])

📝 Exercices corrigés

Exercice 1 : pH d'un acide fort

Énoncé : Calculer le pH d'une solution d'acide chlorhydrique de concentration C = 0,01 mol·L⁻¹.

📖 Correction

Solution :

HCl est un acide fort → dissociation totale

[H₃O⁺] = C = 0,01 = 10⁻² mol·L⁻¹

pH = −log(10⁻²) = 2

Exercice 2 : pH d'une base faible

Énoncé : Calculer le pH d'une solution d'ammoniac NH₃ 0,1 mol·L⁻¹ (pKa = 9,25).

📖 Correction

Solution :

NH₃ est une base faible

Kb = Ke/Ka = 10⁻¹⁴ / 10⁻⁹·²⁵ = 10⁻⁴·⁷⁵

pH = ½(pKe + pKa + log C)

pH = ½(14 + 9,25 + (−1))

pH = ½(22,25) = 11,1

Exercice 3 : Dosage

Énoncé : On dose 25 mL d'une solution d'acide acétique 0,1 mol·L⁻¹ par NaOH 0,1 mol·L⁻¹. Calculer le volume à l'équivalence.

📖 Correction

Solution :

À l'équivalence : CₐVₐ = CᵦVᵦₑ

0,1 × 25 = 0,1 × Vᵦₑ

Vᵦₑ = 25 mL

📌 Notes importantes

  • Un acide fort a un pH plus bas qu'un acide faible de même concentration
  • La dilution augmente toujours le pH d'un acide (et diminue celui d'une base)
  • À la demi-équivalence d'un dosage acide faible/base forte : pH = pKa
  • Une solution tampon est efficace quand [base]/[acide] est entre 0,1 et 10
  • L'indicateur coloré doit avoir sa zone de virage qui contient le pH à l'équivalence

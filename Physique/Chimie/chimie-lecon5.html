📖 Contenu détaillé

1. Définition et structure des amides

📌 Définition

Un amide est un composé organique oxygéné et azoté dont la molécule renferme le groupe fonctionnel :

—C—N—
║
O
📊 Formule et propriétés
  • Formule brute : CₙH₂ₙ₊₁NO
  • Masse molaire : M = 14n + 31 g·mol⁻¹
🧪 Les trois types d'amides
Amide N-non substitué
R—C—NH₂
║
O

Exemple : Méthanamide HCONH₂

Amide N-monosubstitué
R—C—NH—R'
║
O

Exemple : N-méthyléthanamide

Amide N,N-disubstitué
R—C—NR'R''
║
O

Exemple : N,N-diméthylméthanamide

2. Nomenclature des amides

📝 Règles de nomenclature
  1. Identifier la chaîne carbonée la plus longue contenant obligatoirement le groupe amide
  2. Identifier le (ou les) substituant(s) alkyle(s) sur l'atome d'azote
  3. Le nom dérive de l'acide carboxylique correspondant en remplaçant « -ïque » par « -amide »
Exemples
Structure Nom
CH₃—CH₂—CONH₂ Propanamide
CH₃—CH(NH₂)—CH₃ N'est pas un amide !
CH₃—CH₂—CH₂—CONH—CH₃ N-méthylbutanamide
HCON(CH₃)₂ N,N-diméthylméthanamide

3. Préparation des amides

A. Déshydratation d'un carboxylate d'ammonium

À partir de :

R—COO⁻ + NH₄⁺ → R—C—NH₂ + H₂O
║
O

Caractéristiques :

  • Réaction lente
  • Amorcée par chauffage (240°C, 30 min)
  • Réaction limitée (état d'équilibre)
B. Réaction d'un chlorure d'acyle avec l'ammoniac ou une amine

Avec NH₃ en excès → amide N-non substitué :

R—COCl + 2 NH₃ → R—CONH₂ + NH₄Cl

Avec une amine primaire (R'NH₂) en excès → amide N-monosubstitué :

R—COCl + 2 R'NH₂ → R—CONHR' + R'NH₃⁺Cl⁻

Avec une amine secondaire (R₂NH) → amide N,N-disubstitué :

R—COCl + R₂NH → R—CONR₂ + HCl

⚠️ Important

Si l'ammoniac n'est pas en excès, la réaction de chlorure d'acyle avec l'ammoniac s'arrête à la 1ère étape :

R—COCl + NH₃ → R—CONH₂·HCl (chlorhydrate d'amide)

4. Hydrolyse des amides

⚗️ Définition

L'hydrolyse d'un amide est sa réaction avec l'eau. Elle se produit en présence d'un catalyseur (H₃O⁺ ou OH⁻).

1er cas : En milieu acide (H₃O⁺)

Amide N-non substitué :

R—CONH₂ + H₂O + H₃O⁺ → R—COOH + NH₄⁺

Amide N-monosubstitué :

R—CONHR' + H₂O + H₃O⁺ → R—COOH + R'NH₃⁺

Amide N,N-disubstitué :

R—CONR'₂ + H₂O + H₃O⁺ → R—COOH + R'₂NH₂⁺

2ème cas : En milieu basique (OH⁻)

Amide N-non substitué :

R—CONH₂ + NaOH → R—COO⁻Na⁺ + NH₃

Amide N-monosubstitué :

R—CONHR' + NaOH → R—COO⁻Na⁺ + R'NH₂

Amide N,N-disubstitué :

R—CONR'₂ + NaOH → R—COO⁻Na⁺ + R'₂NH

5. Dérivés d'acides carboxyliques

📌 Définition

Lorsqu'on remplace le groupe hydroxyle —OH d'un acide carboxylique RCO—OH par un autre groupe X, on obtient un dérivé de l'acide carboxylique RCO—X.

Tableau des dérivés
Dérivé Groupe X Formule Exemple
Chlorure d'acyle —Cl R—COCl CH₃COCl (chlorure d'éthanoyle)
Anhydride d'acide —O—CO—R' R—CO—O—CO—R' (CH₃CO)₂O (anhydride éthanoïque)
Ester —OR' R—COOR' CH₃COOCH₂CH₃ (éthanoate d'éthyle)
Amide —NH₂, —NHR, —NR₂ R—CONH₂, R—CONHR, R—CONR₂ CH₃CONH₂ (éthanamide)

6. Réactions de passage entre les dérivés

                    NH₃              Amine I (R'NH₂)       Amine II (R₂NH)
                     ↓                    ↓                    ↓
                    ═══════════════════════════════════════════════
                    ║                                            ║
    Chlorure       →  Amide N,N-     →  Amide N-          →  Amide N-
    d'acyle            disubstitué         monosubstitué         non substitué
    (R-CCl)       ←  (R-CONR'₂)     ←  (R-CONHR')        ←  (R-CONH₂)
                    ║                                            ║
                    ═══════════════════════════════════════════════
                     ↓                    ↓                    ↓
    Alcool         ←  Ester            ←  Acide              →  Anhydride
    (R'-OH)            (R-COOR')           carboxylique           d'acide
                                            (R-COOH)          (R-CO-O-CO-R)
Réactions principales
🔄 Chlorure d'acyle + Alcool → Ester

R—COCl + R'—OH → R—COOR' + HCl

Caractéristiques : Rapide, totale, exothermique

🔄 Chlorure d'acyle + Ammoniac/Amine → Amide

R—COCl + 2 NH₃ → R—CONH₂ + NH₄Cl

🔄 Chlorure d'acyle + Acide carboxylique → Anhydride

R—COCl + R'—COOH → (RCO)₂O + HCl

🔄 Anhydride + Alcool → Ester + Acide

(RCO)₂O + R'—OH → R—COOR' + R—COOH

Avantage : Plus doux que la réaction avec chlorure d'acyle

🔄 Anhydride + Ammoniac → Amide + Acide

(RCO)₂O + 2 NH₃ → R—CONH₂ + R—COO⁻NH₄⁺

🔑 Concepts clés à retenir

  • Groupe amide : —CONH₂, —CONHR, ou —CONR₂
  • Chlorure d'acyle : le dérivé le plus réactif
  • Anhydride d'acide : formed from 2 carboxylic acids
  • Esters et amides : derived from carboxylic acids by substitution
  • Hydroxyde et H₃O⁺ : catalyseurs d'hydrolyse
  • Réactivité décroissante : chlorure d'acyle > anhydride > ester > amide

📝 Exercices corrigés

Exercice 1 : Nomenclature des amides

Énoncé : Nommer les composés suivants :

  1. CH₃—CH₂—CH₂—CONH₂
  2. CH₃—CONH—CH₃
  3. H—CO—N(CH₃)₂
📖 Correction

Solutions :

  1. Butanamide (4 carbones, amide non substitué)
  2. N-méthyléthanamide (amide monosubstitué par CH₃)
  3. N,N-diméthylméthanamide (amide disubstitué par 2 CH₃)

Exercice 2 : Préparation d'un amide

Énoncé : Écrire l'équation de réaction entre le chlorure d'éthanoyle et l'ammoniac en excès.

📖 Correction

Solution :

CH₃—COCl + 2 NH₃ → CH₃—CONH₂ + NH₄Cl

En excès d'ammoniac, on obtient l'éthanamide (N-non substitué) et du chlorure d'ammonium.

Exercice 3 : Hydrolyse basique

Énoncé : L'hydrolyse basique du propanamide par la soude donne quels produits ?

📖 Correction

Solution :

CH₃CH₂CONH₂ + NaOH → CH₃CH₂COO⁻Na⁺ + NH₃

Produits formés :

  • Propanoate de sodium (R—COO⁻Na⁺)
  • Ammoniac (NH₃)

📌 Notes importantes

  • Les amides sont des solides cristallins (sauf formamide)
  • Les amides ont des températures d'ébullition élevées grâce aux liaisons H
  • La réaction de déshydration des carboxylates est limitée (équilibre)
  • En milieu basique, l'hydrolyse des amides est totale (formation de carboxylate)
  • L'ordre de réactivité des dérivés : R—COCl > (RCO)₂O > R—COOR' > R—CONH₂
